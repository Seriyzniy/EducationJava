# 5. Подгрузка данных ПОСЛЕ ЗАВЕРШЕНИЕ СЕССИИ

Для того чтобы подгрузить данные ВНЕ СЕССИИ, необходимо ЗАНОВО СОЗДАТЬ сессию.
Однако, после создания новой сессии, объекты предыдущей сессии не будут находиться в `Persistent context`, и соответственно НЕ БУДЕТ отслеживаться `Hibernate`'ом.

Для того чтобы подцепить существующий объект в контекст, используется метод `merge()`
```java
try {
	session.beginTransaction();
	
	Client client = session.get(Client.class,1);
	System.out.println("Получили товар");
	
	session.getTransaction().commit();
	
	//End of first SESSION
	session = sessionFactory.getCurrentSession();
	session.beginTransaction();
	
	client = (Client)session.merge(client);
	
	session.getTransaction().commit();
```
