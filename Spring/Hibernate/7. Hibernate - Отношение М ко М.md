![[Pasted image 20241215132301.png]]

> [!NOTE] `@JoinTable`
> Данная ==аннотация помечает АССОЦИАТИВНУЮ таблицу==, а именно `Actor_Movie`
> Можно использовать в любой из таблиц, т.к отношение М ко М
> 
> В данной аннотации указывается:
> `name` - имя ассоциативной (связывающей) таблицы
> `joinColumns` - указываем СТОЛБЕЦ в АССОЦИАТИВНОЙ таблице, который принадлежит ТЕКУЩЕЙ СУЩНОСТИ (КЛАССУ)
> `inverseJoinColumns` -  указываем СТОЛБЕЦ АССОЦИАТИВНОЙ таблицы, который принадлежит ПРОТИВОПОЛОЖНОЙ СТОРОНЕ в отношении, т.е таблице `Movie`

# 2. Создание классов сущностей

Начнем с класса `Actor`
```java
package ru.myProj.Hibernate.HibernateApp.model;

@Entity
@Table(name = "Actors")
public class Actor {
	@Id
	@Column(name = "actor_id")
	@GeneratedValue(strategy = GenerationType.IDENTITY)
	private int id;
	
	@Column(name = "actor_name")
	private String name;
	
	@Column(name = "actor_age")
	private int age;
	
	public Actor() {};
	public Actor(String name, int age) {
		this.name = name;
		this.age = age;
	}
	public int getId() {
		return id;
	}
	public int getAge() {
		return age;
	}
	public String getName() {
		return name;
	}
	public void setName(String name) {
		this.name = name;
	}
	public void setAge(int age) {
		this.age = age;
	}
	@Override
	public String toString() {
		return "Actor " + name + ", " + age;
	}
}
```

Теперь опишем класс `Movie`
```java
@Entity
@Table(name = "Movies")
public class Movie {
	@Id
	@Column(name = "movie_id")
	@GeneratedValue(strategy = GenerationType.IDENTITY)
	private int id;
	
	@Column(name="movie_name")
	private String name;
	
	@Column(name="movie_year_of_out")
	private int year_of_out;
	
	public Movie() {}
	public Movie(String name, int year) {
		this.name = name;
		this.year_of_out = year;
	}
	public int getId() {
		return id;
	}
	public String getName() {
		return name;
	}
	public int getYear_of_out() {
		return year_of_out;
	}
	public void setName(String name) {
		this.name = name;
	}
	public void setYear_of_out(int year_of_out) {
		this.year_of_out = year_of_out;
	}
	@Override
	public String toString() {
		return "Movie " + name + ", " + year_of_out;
	}
}
```

# 4. Работа с сущностями
Создадим два актера и один фильм

```java
public class App {
	public static void main(String[] args) {
		Configuration myConfigHiber = new Configuration();
		myConfigHiber.addAnnotatedClass(Actor.class);
		myConfigHiber.addAnnotatedClass(Movie.class);
	
		SessionFactory mySessionFact = myConfigHiber.buildSessionFactory();
		Session mySession = mySessionFact.getCurrentSession();
		try {
			mySession.beginTransaction();
		
			Movie movie = new Movie("Criminal chtivo,", 1994);
			Actor actor1 = new Actor("Harvey Keitel", 81);
			Actor actor2 = new Actor("Samuel L. Jachson", 91);
		
			movie.setActors(new ArrayList<>(List.of(actor1, actor2)));
			actor1.setMovies(new ArrayList<>(Collections.singletonList(movie)));
			actor2.setMovies(new ArrayList<>(Collections.singletonList(movie)));
			
			mySession.save(movie);
			mySession.save(actor1);
			mySession.save(actor2);
		
			mySession.getTransaction().commit();
		}finally {
			mySessionFact.close();
		}
	}
}
```
В результате в БД добавятся записи с новыми актерами и новым фильмом, а также добавятся записи в ассоциативную таблицу

# 5. Получение актеров по фильму
```java
try {
	mySession.beginTransaction();
	
	Movie movie = mySession.get(Movie.class, 1);
	for(Actor a : movie.getActors()) {
		System.out.println(a);
	}
	
	mySession.getTransaction().commit();
}finally {
	mySessionFact.close();
}
```
В результате получим всех актеров этого фильма

# 6. Добавление нового фильма для одного актера

```java
try {
	mySession.beginTransaction();
	
	Movie movie = new Movie("Titanit", 1992);
	Actor actor = mySession.get(Actor.class, 1);
	
	movie.setActors(new ArrayList<>(Collections.singletonList(actor)));
	actor.getMovies().add(movie);
	
	mySession.save(movie);
	
	mySession.getTransaction().commit();
}finally {
	mySessionFact.close();
}
```
