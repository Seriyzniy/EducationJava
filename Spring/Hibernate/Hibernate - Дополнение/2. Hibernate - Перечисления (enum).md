Чтобы сохранить перечисления необходимо пометить поле с типом перечисления аннотацией `@Enumerated`

В качестве параметра этой аннотации можно указать КАК ОН БУДЕТ СОХРАНЯТЬ перечисление

- ORDINAL - сохраняет значение перечисления как число (используется по умолчанию)
- STRING - сохраняет строковое значение

Создадим перечисление НАСТРОЕНИЕ и создадим поле в классе Person с этим типом данных
```java
public enum Mood {
	HAPPY,SAD,ANGRY
}
```

Теперь в классе Person создадим поле и для него set/get методы
```java
@Enumerated(EnumType.ORDINAL)
public Mood getMood() {
	return mood;
}
```

Добавим колонку в ТАБЛИЦУ для хранения настроения
Здесь укажем ТИП ДАННЫХ `INT` так как выбрали тип сохранения как `ORDINAL`
```sql
ALTER TABLE Persons ADD COLUMN mood INT;
```

Теперь в СЕРВИСЕ будем назначать настроение человеку
```java
@Transactional
public void save(Person person) {
	person.setCreateAt(new Date());
	person.setMood(Mood.HAPPY);     //Настроение
	personRepository.save(person);
}
```
