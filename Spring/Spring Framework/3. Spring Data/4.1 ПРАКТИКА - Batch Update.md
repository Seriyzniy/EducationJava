Реализуем на практике два метода вставки записей в БД
1) С использованием Batch Update (1000 записей)
2) Множественная вставка  (1000 записей)

А затем сравним эффективность двух методов
# 1. Реализация контроллера `BatchController`

Создадим контроллер, который будет отправлять пользователю страницу с двумя ссылками:
1) Выполняет Batch Update
2) Выполняет Multiple Update

```java
@Controller
@RequestMapping("/test-update")
public class BatchController {
	private final PersonDAO personDAO;
	
	@Autowired
	public BatchController(PersonDAO personDAO) {
		this.personDAO = personDAO;
	}
	
	@GetMapping
	public String welcomePage() {
		return "batch/welcome";
	}
	@GetMapping("/batch")
	public String batchUpdate() {
		personDAO.testBatchUpdate();
		return "redirect:/people";
	}
	
	@GetMapping("/multi")
	public String multiUpdate() {
		personDAO.testMultipleUpdate();
		return "redirect:/people";
	}
}
```

# 2. Реализация методов `testBatchUpdate()` и `testMultipleUpdate()` в PersonDAO

```java
public void testMultipleUpdate() {
	List<Person> list = create1000People();
	
	long before = System.currentTimeMillis();
	for(Person p : list) {
		myJdbcTemplate.update("INSERT INTO persons VALUES(?, ?, ?, ?)", p.getId(), p.getName(), p.getAge(), p.getEmail());
	}
	long after = System.currentTimeMillis();
	
	System.out.println("\n Time eclipsed = " + (after - before));
}

public List<Person> create1000People(){
	List<Person> list = new ArrayList<Person>();
	for(int i = 0; i <1000; i++) {
		list.add(new Person(i, "TestUser" + i, 20, "test"+i+"mail.com" ));
	}
	return list;
}
```

Теперь реализуем `testBatchUpdate()`
```java
public void testBatchUpdate() {
	List<Person> list = create1000People();
	
	long before = System.currentTimeMillis();
	myJdbcTemplate.batchUpdate("INSERT INTO persons VALUES(?, ?, ?, ?)", new BatchPreparedStatementSetter() {
		@Override
		public void setValues(PreparedStatement ps, int i) throws SQLException {
			ps.setInt(1, list.get(i).getId());
			ps.setString(2, list.get(i).getName());
			ps.setInt(3, list.get(i).getAge());
			ps.setString(4, list.get(i).getEmail());
		}
		@Override
		public int getBatchSize() {
			// TODO Auto-generated method stub
			return list.size();
		}
	});
	long after = System.currentTimeMillis();
	System.out.println("\n Time BATCH eclipsed = " + (after - before));
}
```

Здесь используется специальный метод класса `JdbcTemplate` `batchUpdate`, который принимает в качестве параметров:
1) SQL запрос
2) Объект класса `BatchPreparedStatementSetter()`
		У этого класса необходимо реализовать два метода
		1) Получение КОЛИЧЕСТВА ЗАПИСЕЙ
		2) Добавление данных в SQL запрос
