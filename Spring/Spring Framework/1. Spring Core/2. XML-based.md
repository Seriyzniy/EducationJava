### XML-Based Configuration (Maven)
 
Для реализации XML-based необходимо создать файл XML в папке src/main/resources. 
Именно его будет читать IoC Container для внедрения зависимостей.

```xml
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
		xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
		xmlns:context="http://www.springframework.org/schema/context"
		xsi:schemaLocation="http://www.springframework.org/schema/beans
		http://www.springframework.org/schema/beans/spring-beans.xsd
		http://www.springframework.org/schema/context
		http://www.springframework.org/schema/context/spring-context.xsd">
		
	<bean id="musicBean"
		class="ru.tudyspring.myproject.ClassicMusic">
	</bean>
</beans>
```

==Bean== - обычные Java классы, созданные при помощи Spring

Будем использовать изученные принципы IoC и DI (XML). Для этого дополним ранее созданный XML файл (applicationcontext.xml):

```xml
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
		xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
		xmlns:context="http://www.springframework.org/schema/context"
		xsi:schemaLocation="http://www.springframework.org/schema/beans
		http://www.springframework.org/schema/beans/spring-beans.xsd
		http://www.springframework.org/schema/context
		http://www.springframework.org/schema/context/spring-context.xsd">
		
	<bean id="musicBean"
		class="ru.tudyspring.myproject.ClassicMusic">
	</bean>
	
	<bean id="musicPlayer"
		class="ru.studyspring.myproject.MusicPlayer">
		<constructor-arg ref="musicBean"/>
	</bean>
</beans>
```

Здесь мы добавили новый Bean класса MusicPlayer. И указали параметр, который ему необходимо передать при создании, а именно другой Bean musicBean.
```xml
	<bean id="musicBean"
		class="ru.tudyspring.myproject.ClassicMusic">
	</bean>
	
	<bean id="musicPlayer"
		class="ru.studyspring.myproject.MusicPlayer">
		<constructor-arg ref="musicBean"/>
	</bean>
```

Теперь необходимо создать IoC контейнер и предоставить ему информацию о Bean Definitions в виде XML файла:
```java
package ru.studyspring.myproject;

import org.springframework.context.support.ClassPathXmlApplicationContext;

public class ApplicationRunner {
	public static void main(String[] args) {
		ClassPathXmlApplicationContext appContext = new ClassPathXmlApplicationContext("applicationcontext.xml");
	
		MusicPlayer musicPlayer = appContext.getBean("musicPlayer", MusicPlayer.class);
		musicPlayer.PlaingMusic();
		
		appContext.close();
	
	}

}
```

Здесь с помощью класса ClassPathXmlApplicationContext мы создаем IoC контейнер и указываем ему где лежат BeanDefinitions, необходимые для создания бинов.
