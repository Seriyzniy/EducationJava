# Структура приложения MVC

Приложение MVC состоит из:
- Обычных классов Java (контроллеры, модели и т.п)
	 Очень часто используются аннотации, применяемые к классам и дающие дополнительные возможности
	
- Набор HTML-страниц
	 Часто используют Java-Script код который "оживляет" html представления и CSS для стилизации
	
- Spring конфигурация (XML, Annotation, Java-код)

# DispatcherServlet
Ниже приведена схема того, как работает Spring Web приложение
![[Pasted image 20241026161220.png]]

==DispatcherServlet - специальный контроллер==

Он является ВХОДНОЙ точкой Spring MVC приложения, и он УЖЕ РЕАЛИЗОВАН за нас.

### Порядок работы системы MVC:
- Пользователь отправил запрос HTTP
	
- Сервер принимает запрос и отправляет на Spring MVC приложение 
	
- Запрос попадает в DispatcherServlet
	
- ==DispatcherServlet отправляет запрос КОНКРЕТНОМУ контроллеру, ответственному за данный запрос==
	
- Контроллер обращается к модели данных
	
- Модель данных выбирает информацию, в соответствии с запросом, из БД
	
- Модель данных передается контроллеру
	
- Контроллер отправляет ответ

# Controller

==Класс Java==, реализуемый программистом и предназначен ==для обработки определенного запроса==

Такой класс создается обычно с аннотацией `@Controller`

> [!NOTE]
> 1 Контроллер - 1 Запрос

Что делает Controller?
- Обрабатывает запросы от пользователя
- Обменивается данными с моделью
- Показывает пользователю правильное представление
- Переадресовывает пользователя на другие страницы


# Model

==Класс Java==, реализуемый программистом и предназначен ==для работы с данными==

Что делает Model?
- Хранит в себе данные
- Взаимодействует с БД для получения данных
- Отдает данные контроллеру

# View

==HTML код,== для отображения ==страницы в браузере== пользователя

Что делает View?
- Получает данные от контроллера и отображает их в браузере
- Для динамического отображения данных используются шаблонизаторы (T==hymleaf==, Freemarker, Velocity)
	- Пользователи заходят на одну и ту же страницу HTML, но информация для каждого пользователя индивидуальная (ВК, Instagram)
