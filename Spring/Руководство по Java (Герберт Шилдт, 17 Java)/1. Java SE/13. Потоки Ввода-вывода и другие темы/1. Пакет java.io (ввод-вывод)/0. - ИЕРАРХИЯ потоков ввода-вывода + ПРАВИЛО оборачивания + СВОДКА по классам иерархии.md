
> [!NOTE] ПРАВИЛО ОБОРАЧИВАНИЯ
> - Что хотим сделать/добавить? (Внешний класс, `то, КУДА передаем параметр`) 
> - На чем хотим сделать/добавить? (Внутренний класс, `то, ЧТО передаем в параметр`)
> 
> Пример
> Хотим добавить буферизацию перед записью в файл. 
> - Что хотим? - Буферизацию
> - На что? - Перед запись файл => поток файловой записи
> ```java
> BufferedOutputStream(new FileOutputStream(...))
> ```

# 3. Сводка по всем классам

> [!NOTE] 🛑Потоки из `java.io` `НЕ ПОТОКОБЕЗОПАСНЫ`-нужна СИНХРОНИЗАЦИЯ🛑
> Только `StringWriter` потокобезопасен через `StringBuffer`
> 

| Класс                       | ==Нужна== ли внешняя ==синхронизация== | Напарник (Input/Output)   | Для чего нужен                                                                           |
| --------------------------- | -------------------------------------- | ------------------------- | ---------------------------------------------------------------------------------------- |
|                             | --- ==Input== ---                      |                           |                                                                                          |
| **ByteArray-InputStream**   | Обычно нет                             | ByteArray-OutputStream    | Чтение байтов `ИЗ МАССИВА` в памяти                                                      |
| **File-<br>InputStream**    | ✅ Если из разных потоков               | File-OutputStream         | Чтение байтов из файла                                                                   |
| **Filter-<br>InputStream**  | ✅                                      | Filter-OutputStream       | Базовый класс-декоратор для InputStream                                                  |
| **Buffered-InputStream**    | ✅                                      | Buffered-OutputStream     | Чтение байтов с буферизацией (ускоряет доступ)                                           |
| **Data-<br>InputStream**    | ✅                                      | Data-OutputStream         | Чтение `примитивов` (int, double, boolean) в бинарной форме                              |
| **Pushback-InputStream**    | ✅                                      | —                         | Чтение байтов с `возможностью «отката»` назад (unread)                                   |
| **Sequence-InputStream**    | ✅                                      | —                         | Последовательное `ОБЪЕДИНЕНИЕ НЕСКОЛЬКИХ ` `InputStream`                                 |
| **Object<br>-InputStream**  | ✅                                      | Object-OutputStream       | Чтение `объектов` (десериализация)                                                       |
| **Piped-<br>InputStream**   | ✅ (вокруг пары)                        | Piped-OutputStream        | Межпоточный обмен байтами через буфер                                                    |
|                             | ---- ==Output== ----                   |                           |                                                                                          |
| ByteArray-OutputStream      | ✅                                      | **ByteArray-InputStream** | `ЗАПИСЬ` байтов `в МАССИВ`                                                               |
| **Buffered-OutputStream**   | ✅                                      | Buffered-InputStream      | Запись байтов с буферизацией                                                             |
| **File-<br>OutputStream**   | ✅                                      | File-InputStream          | Запись байтов в файл                                                                     |
| **Filter-<br>OutputStream** | ✅                                      | Filter-InputStream        | Базовый декоратор для OutputStream                                                       |
| **PrintStream**             | ✅                                      | —                         | Удобный ==вывод текстовых представлений== примитивов и объектов (==работает с байтами==) |
| **Data-<br>OutputStream**   | ✅                                      | Data-InputStream          | Запись примитивов в бинарной форме                                                       |
| **Object-<br>OutputStream** | ✅                                      | Object-InputStream        | Запись объектов (сериализация)                                                           |
| **Piped-<br>OutputStream**  | ✅ (вокруг пары)                        | Piped-InputStream         | Межпоточный вывод байтов                                                                 |
|                             |                                        |                           |                                                                                          |
|                             | --- ==Reader== ---                     |                           |                                                                                          |
| **Buffered-<br>Reader**     | ✅                                      | Buffered-Writer           | Чтение текста с буферизацией (по строкам или символам)                                   |
| **LineNumber-<br>Reader**   | ✅                                      | —                         | Чтение текста с отслеживанием номера строки                                              |
| **CharArray-<br>Reader**    | Обычно нет                             | CharArray-Writer          | Чтение символов из массива                                                               |
| **InputStream-<br>Reader**  | ✅                                      | OutputStream-Writer       | ==Мост: байты → символы== (с учетом кодировки)                                           |
| **File-<br>Reader**         | ✅                                      | FileWriter                | Упрощенный InputStreamReader для файлов                                                  |
| **Filter-<br>Reader**       | ✅                                      | FilterWriter              | Базовый декоратор для Reader                                                             |
| **Pushback-<br>Reader**     | ✅                                      | —                         | Чтение текста с возможностью «отката» символов                                           |
| **Piped-<br>Reader**        | ✅ (вокруг пары)                        | PipedWriter               | Межпоточный обмен символами                                                              |
| **String-<br>Reader**       | Обычно нет                             | StringWriter              | Чтение символов из строки                                                                |
|                             | --- ==Writer== ---                     |                           |                                                                                          |
| **Buffered-<br>Writer**     | ✅                                      | BufferedReader            | Запись текста с буферизацией                                                             |
| **CharArray-<br>Writer**    | Обычно нет                             | CharArrayReader           | Запись текста в массив символов в памяти                                                 |

| **Print-<br>Writer**        | ✅                                      | —                         | Удобный вывод текстовых данных (работает с символами)                                    |
| **String-<br>Writer**       | Обычно нет                             | StringReader              | Запись текста в строку (в памяти)                                                        |
