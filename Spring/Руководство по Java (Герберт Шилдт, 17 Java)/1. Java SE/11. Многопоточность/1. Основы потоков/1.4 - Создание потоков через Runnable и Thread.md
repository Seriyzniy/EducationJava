# 📕 1. Главный поток 
При запуске программы на Java, немедленно начинает выполняться один поток  `главный поток` программы

Этим `главным потоком` можно управлять при помощи объекта `Thread`. 

Для этого понадобится получить ссылку на него, вызвав метод `currentThread()`, который является `public static` членом класса `Thread`
```java
// Возвращает ссылку на поток, из которого был 
//вызван метод
static Thread currentThread(); 

// Прерывает целевой поток (
//устанавливает **флаг прерывания** у целевого потока (interrupted status = `true`)
public void interrupt() { ... }      

// Проверяет статус прерывания 
public boolean isInterrupted() { ... }  

// Возвращает и сбрасывает флаг текущего потока (единственный способ сброса)
public static boolean interrupted() { ... } 
}                               
```

# 📕 2. Создание потока
Многопоточная система Java построена а основе класса `Thread`, его методах и дополняющем интерфейсе `Runnable`

> [!NOTE] 🛑 Создание потока 🛑
> Для создания нового потока, программа: 
> - Либо расширяет класс `Thread`
> - Либо реализует интерфейс `Runnable`
>

1. Расширение `Thread`
```java
class MyThread extends Thread {
    public void run() {
        System.out.println("Привет из потока");
    }
}
```
> [!NOTE] 🛑 Метод `run` класса `Thread` выбрасывает исключение `InterruptedException`, а `run` в интерфейсе `Runnable` НЕ ВЫБРАСЫВАЕТ ничего 🛑
> 

2. Реализация `Runnable`
```java
class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Привет из потока");
    }
}
```

> [!NOTE] ☣==Оба способа== подразумевают ==наличие метода== `run()`
>

## 📙2.1 Реализация `Runnable`
Создание потока с помощью интерфейса `Runnable` — это **один из классических способов многопоточности в Java**. 
Он используется для **разделения задачи (что делать) и механизма (как выполнять)**. Давай подробно разберём:

> [!NOTE] 🛑Главный метод `run()`🛑
> Внутри этого метода помещается код, который будет выполняться в отдельном потоке
> 
> Метод определяет новую точку входа для другого потока

```java
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("Привет из потока " + Thread.currentThread().getName());
    }
}

public class MyMain {
    public static void main(String[] args) {
        Runnable task = new MyRunnable();           // 1. Создаем задачу
        Thread thread = new Thread(task);           // 2. 🛑Оборачиваем её в поток
        thread.start();                             // 3. Запускаем поток 
    }
}
```


> [!NOTE] 🛑 Под капотом 🛑
> После создания класса, реализующего интерфейс `Runnable`, внутри него создается объект типа `Thread`
> 
> При этом используется конструктор:
> ```java
> Thread(Runnable threadOb, String threadName)
> ```
> `threadOb` - ==экземпляр класса, который реализовал== `Runnable` 
> (можно `this` если `run()` в текущем классе)

> [!NOTE] 🛑 Запуск потока 🛑
> После реализации метода `run()`, поток НЕ ЗАПУСТИТСЯ
> 
> 🔑Для запуска необходимо вызвать метод `start()`

### Пример с Лямбда-выражением
```java
public class LambdaThread {
    public static void main(String[] args) {
        Thread t = new Thread(() -> {
				System.out.println("Lambda-поток");
			});
        t.start();
    }
}
```
### Использование `Runnable`:
- **Чище с точки зрения дизайна** (разделение обязанностей),
    
- ==Позволяет использовать задачу в других API== (например, `ExecutorService`),
    
- Позволяет **наследоваться от другого класса** (ведь Java поддерживает только один `extends`).

## 📙 2.2 Расширение класса`Thread`
Создание потока путём **наследования от класса `Thread`** — это второй базовый способ многопоточности в Java. 
Он проще с точки зрения написания кода, но **менее гибкий**, чем использование `Runnable`.

☣`Thread` под капотом реализует интерфейс `Runnable`

```java
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("Поток: " + Thread.currentThread().getName());
    }
}

public class MyMain {
    public static void main(String[] args) {
        MyThread t1 = new MyThread();  // создаём объект потока
        t1.start();                    // запускаем поток
    }
}
```

### 🧪 Java 8+: пример через анонимный класс
```java
public class Demo {
    public static void main(String[] args) {
        Thread t = new Thread() {
            public void run() {
                System.out.println("Поток через анонимный класс");
            }
        };
        t.start();
    }
}
```
