
> [!NOTE] **TimeUnit** — это перечисление (enum) в пакете `java.util.concurrent`, 
> Которое предоставляет удобные методы для преобразования и работы с единицами времени.
> 

Доступные единицы измерения:
- `NANOSECONDS` (наносекунды)
    
- `MICROSECONDS` (микросекунды)
    
- `MILLISECONDS` (миллисекунды)
    
- `SECONDS` (секунды)
    
- `MINUTES` (минуты)
    
- `HOURS` (часы)
    
- `DAYS` (дни)

---
#### Преобразование единиц времени
```java
// Конвертация между единицами
long seconds = TimeUnit.MINUTES.toSeconds(5); // 300 секунд
long millis = TimeUnit.SECONDS.toMillis(30);  // 30000 миллисекунд
long hours = TimeUnit.DAYS.toHours(2);        // 48 часов
```

#### Использование с методами ожидания
```java
// Более читаемая альтернатива Thread.sleep()
TimeUnit.SECONDS.sleep(5);        // Спать 5 секунд
TimeUnit.MILLISECONDS.sleep(500); // Спать 500 миллисекунд

// С примитивами синхронизации
Lock lock = new ReentrantLock();
lock.tryLock(2, TimeUnit.MINUTES); // Ждать блокировку до 2 минут
```
