
> [!NOTE] `Predicate<T>`
> Функция, которая проверяет условие для объекта: `T → boolean`.

## Интерфейс
```java
@FunctionalInterface
public interface Predicate<T> {
    💡 boolean test(T t);

    default Predicate<T> and(Predicate<? super T> other);
    default Predicate<T> or(Predicate<? super T> other);
    default Predicate<T> negate();
    static <T> Predicate<T> isEqual(Object targetRef);
}
```

## 💡 Особенности 💡
- Базовый интерфейс для фильтрации в `Stream.filter()`.
    
- Поддерживает комбинирование условий (`and`, `or`, `negate`).

## ✅ Best Practices ✅
- Используйте ссылки на методы (`Objects::isNull`).
    
- Для читаемости объединяйте условия с `and` / `or`.

## Пример
```java
Predicate<String> notEmpty = s -> !s.isEmpty();
Predicate<String> longStr = s -> s.length() > 5;

System.out.println(notEmpty.and(longStr).test("HelloWorld")); 

// Вывод: 
true
```
