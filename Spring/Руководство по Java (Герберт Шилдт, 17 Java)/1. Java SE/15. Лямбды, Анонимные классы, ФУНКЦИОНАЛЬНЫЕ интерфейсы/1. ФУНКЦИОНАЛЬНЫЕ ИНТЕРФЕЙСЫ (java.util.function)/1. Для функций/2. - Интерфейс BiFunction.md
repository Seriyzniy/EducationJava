
> [!NOTE] `BiFunction<T, U, R>` 
> Он описывает функцию, принимающую 
> - Два аргумента (`T` и `U`) 
> - Возвращающую результат (`R`).  

Используется в **Stream API**, лямбдах, методах высшего порядка (`map`, `reduce`, `compute` у `Map`, и т.д.).

## Абстрактный метод `R apply(T t, U u);`
```java
// Применяет функцию к двум аргументам
R apply(T t, U u);
```
## Интерфейс
```java
@FunctionalInterface
public interface BiFunction<T, U, R> {

    // Применяет функцию к двум аргументам
    R apply(T t, U u);

    // Возвращает новую функцию, которая сначала применяет текущую,
    // а потом применяет указанную функцию к результату
    default <V> BiFunction<T, U, V> andThen(Function<? super R, ? extends V> after) {
        Objects.requireNonNull(after);
        return (T t, U u) -> after.apply(apply(t, u));
    }
}
```

## Пример
```java
import java.util.function.BiFunction;

public class Main {
    public static void main(String[] args) {
        // Сложение двух чисел
        BiFunction<Integer, Integer, Integer> sum = (a, b) -> a + b;
        System.out.println(sum.apply(2, 3)); // 5

        // Конкатенация строки и числа
        BiFunction<String, Integer, String> concat = (s, n) -> s + n;
        System.out.println(concat.apply("Age: ", 25)); // Age: 25

        // Цепочка: сначала сложение, потом перевод в строку
        BiFunction<Integer, Integer, Integer> add = (x, y) -> x + y;
        String result = add.andThen(Object::toString).apply(4, 6);
        System.out.println(result); // "10"
    }
}
```
