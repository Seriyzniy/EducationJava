
> [!NOTE] `Supplier<T>`
> Функция без аргументов, возвращающая значение.

## Интерфейс
```java
@FunctionalInterface
public interface Supplier<T> {
    T get();
}
```

## 💡 Особенности 💡
- Используется для ленивой генерации значений.
    
## ✅ Best Practices ✅
- Удобен ==для фабрик== (`Supplier<Stream<T>>`, `Supplier<Connection>`).

## Пример
```java
Supplier<Double> random = Math::random;

System.out.println(random.get()); // случайное число
```
