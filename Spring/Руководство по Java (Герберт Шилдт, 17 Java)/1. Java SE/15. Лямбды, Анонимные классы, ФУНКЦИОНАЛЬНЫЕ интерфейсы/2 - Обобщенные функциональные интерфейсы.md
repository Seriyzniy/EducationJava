
> [!NOTE] 1 метод - несколько реализаций
> Можно использовать один метод, для разных реализаций, в зависимости от необходимого типа аргумента

Лямбда-выражения могут работать с обобщенными интерфейсами. 

В таком случае ==целевой тип лямбда-выражения== частично ==определяется аргументом ==или аргументами типов, ==указанными ПРИ ОБЪЯВЛЕНИИ ССЫЛКИ на функциональный интерфейс.

```java
interface SomeFunc<T> {
    T func(T t);
}

class GenericFunctionalInterfaceDemo {
    public static void main(String[] args) {
        // Для строк - реверс строки
        // 🟢 Здесь str - String
        SomeFunc<String> reverse = (str) -> {
            String result = "";
            for(int i = str.length()-1; i >= 0; i--)
                result += str.charAt(i);
            return result;
        };
        
        // Для целых чисел - факториал
        // 🟢 Здесь n - Integer
        SomeFunc<Integer> factorial = (n) -> {
            int result = 1;
            for(int i = 1; i <= n; i++)
                result = i * result;
            return result;
        };
    }
}
```
