
Создадим такую структуру
```java
class Student {
	private String name;
	private Integer age;

	public Student(String name, Integer age) {
		super();
		this.name = name;
		this.age = age;
	}

	public String getName() {
		return name;
	}

	public void setName(String name) {
		this.name = name;
	}

	public Integer getAge() {
		return age;
	}

	public void setAge(Integer age) {
		this.age = age;
	}
}

class Team {
	private String type;
	private List<Student> students;

	public Team(String type, List<Student> students) {
		super();
		this.type = type;
		this.students = students;
	}

	public String getType() {
		return type;
	}

	public void setType(String type) {
		this.type = type;
	}

	public List<Student> getStudents() {
		return students;
	}

	public void setStudents(List<Student> students) {
		this.students = students;
	}
}

public class MyMain {
	public static void main(String[] args) throws InterruptedException {
		List<Student> basketballStudents = List.of(
				new Student("Bob", 18), 
				new Student("Ted", 17),
				new Student("Zeka", 19));

		List<Student> footballStudent = List.of(
				new Student("Alan", 19), 
				new Student("Anne", 21),
				new Student("Davis", 21));

		
		Team basketballTeam = new Team(
			"bastetball", basketballStudents);
		
		Team footballTeam = new Team(
			"football", footballStudent);
		
		List<Team> teams = List.of(basketballTeam, footballTeam);
	}
}
```


> [!NOTE] `teams` - имеет тип `List<Team>`
> 

# 1. stream()
```JAVA
Stream<Team> l = teams.stream(); 		
```

`stream()` - "==распаковывает==" коллекцию, и представляет экземпляр для итерации

# 2. map()
```java
Stream<List<Student>> l = teams 
	.stream()
	.map(t -> t.getStudents());
```

`map()` - как бы "==запаковывает==" то, что было получено внутри, в объект `Stream`

# 3. collector()
```java
List<List<Student>> l = teams 
	.stream() // Stream<Team>
	.map(t -> t.getStudents())  	// Stream<List<Student>>
	.collect(Collectors.toList()); // List<List<Student>>
```
`collector` - с использованием `Collectors` изменяет тип с `Stream<T>` на `List<T>`
_Как бы заменяет Stream на List_

# Попытка преобразования без flatMap
## Спойлер - используй flatMap
```java
List<Student> list = teams 
	.stream()
	.flatMap(t -> t.getStudents().stream())
	.toList();
```

### Преобразования в flatMap:
```java
teams.stream()
    .flatMap(t -> t.getStudents().stream())
```
- Для каждой команды вызывается `t.getStudents().stream()` → поток студентов этой команды
    
- **flatMap** берёт все эти потоки и **«сплющивает»**(==объединяет==) их в один большой поток:
```java
Stream[Bob, Ted, Zeka, Alan, Anne, Davis]
```

### Преобразования в map:
Если тоже самое сделать в map:
```java
teams.stream()
    .map(t -> t.getStudents().stream())
```
❗❗Каждый `Team` превращается в `Stream<Student>`❗❗ (см что делает stream())

Результат на выходе: `Stream<Stream<Student>>` — поток потоков.

То есть структура становится:
```java
Stream[
   Stream[Bob, Ted, Zeka],
   Stream[Alan, Anne, Davis]
]
```

# Преобразования
```java
List<List<Student>> list = teams 
	.stream() // Stream<Team>
	.map(t -> t.getStudents()) // Stream<List<Student>>
	.map(l -> l.stream() // List<Student> -> ListStudent => Stream<List<Student>>
			.map(s -> s) // Student -> Student => Stream<Student>
			.toList() // Stream<Student> => List<Student>
	).toList(); // Stream<List<Student>> -> List<<List<Student>>
```
