
> [!NOTE] ⭐ДВА метода для поиска символа/подстроки⭐
> Класс `String` предлагает `два метода`, которые позволяют искать в строке указанный символ или подстроку: 
> - `indexOf()` - ищет ==первое вхождение== символа или подстроки;  
> - `lastIndexOf()` - ищет ==последнее вхождение== символа или подстроки.
> 
> Указанные два метода имеют несколько перегруженных версий. 
> 
> ✅Во всех случаях они `ВОЗВРАЩАЮТ ИНДЕКС, по которому был найден` символ или подстрока, либо -1, если символ или подстрока не обнаружены.

```java
// =============================================================================
// ✅ МЕТОДЫ indexOf() - поиск ПЕРВОГО вхождения✅
// =============================================================================

// 📗 ПОИСК СИМВОЛА 📗
// Поиск первого вхождения символа ch в строке
// Возвращает индекс первого найденного символа или -1 если не найден
int indexOf(int ch)

// Поиск первого вхождения символа ch в строке, НАЧИНАЯ С УКАЗАННОГО ИНДЕКСА  startIndex
// Возвращает индекс первого найденного символа или -1 если не найден
int indexOf(int ch, int startIndex)


// 📗 ПОИСК СТРОКИ 📗
// Поиск первого вхождения подстроки str в строке
// Возвращает индекс первого символа найденной подстроки или -1 если не найден
int indexOf(String str)

// Поиск первого вхождения подстроки str в строке, НАЧИНАЯ С УКАЗАННОГО ИНДЕКСА startIndex
// Возвращает индекс первого символа найденной подстроки или -1 если не найден
int indexOf(String str, int startIndex)



// =============================================================================
// ✅ МЕТОДЫ lastIndexOf() - поиск ПОСЛЕДНЕГО вхождения ✅
// =============================================================================

// 📗 ПОИСК СИМВОЛА 📗
// Поиск последнего вхождения символа ch в строке
// Возвращает индекс последнего найденного символа или -1 если не найден
int lastIndexOf(int ch)

// Поиск последнего вхождения символа ch в строке, НАЧИНАЯ С УКАЗАННОГО ИНДЕКСА  startIndex
// Поиск выполняется от startIndex до начала строки (индекс 0)
// Возвращает индекс последнего найденного символа или -1 если не найден
int lastIndexOf(int ch, int startIndex)


// 📗 ПОИСК СТРОКИ 📗
// Поиск последнего вхождения подстроки str в строке
// Возвращает индекс первого символа последней найденной подстроки или -1 если не найден
int lastIndexOf(String str)

// Поиск последнего вхождения подстроки str в строке, НАЧИНАЯ С УКАЗАННОГО ИНДЕКСА  startIndex
// Поиск выполняется от startIndex до начала строки (индекс 0)
// Возвращает индекс первого символа последней найденной подстроки или -1 если не найден
int lastIndexOf(String str, int startIndex)
```
