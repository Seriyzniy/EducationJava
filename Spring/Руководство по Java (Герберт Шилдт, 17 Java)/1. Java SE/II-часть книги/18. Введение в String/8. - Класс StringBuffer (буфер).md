# 📕 1. Конструкторы
По умолчанию `StringBuffer` ==РЕЗЕРВИРУЕТ 16 СИМВОЛОВ==
```java
StringBuffer() // емкость 16
StringBuffer(int size) // указанная емкость
StringBuffer(String str) // начальное содержимое + 16

StringBuffer(CharSequence chars)
```

# 📕 2. Основные методы

## 📙 1. length() и capacity()
Текущую длину `StringBuffer` можно узнать посредством метода `length()` , а общую выделенную емкость - с помощью метода `capacity()`
```java
int length()
int capacity()
```

```java
StringBuffer sb = new StringBuffer("Hello");
sb.length(); // 5
sb.capacity(); // 21 (5 + 16)
```

## 📙 2. ensureCapacity()
При желании предварительно выделить место для определенного количества символов после конструирования объекта `StringBuffer` можете установить размер буфера с помощью метода `ensureCapacity()`
```java
void ensureCapacity(int minCapacity)
```

> [!NOTE] `Гарантирует минимальную емкость`
> 

## 📙 3. setLength()
Для установки длины строки в объекте `StringBuffer` используйте метод
```java
void setLength(int len)
```
- Устанавливает длину строки
    
- При увеличении добавляет нулевые символы
    
- При уменьшении обрезает строку

В случае вызова `setLength()` со значением меньше текущего значения, возвращаемого методом `length() `, символы, которые оказались за пределами новой длины, будут утрачены.

## 📙 4. charAt() и setCharAt 
С использованием метода 
- `charAt()` можно ==получать значение одиночного символа== из объекта `StringBuffer`, 
- `setCharAt()` можно ==устанавливать значение символа== в `StringBuffer`.

```java
char charAt(int where)
void setCharAt(int where, char ch)
```

## 📙 5. getChars()
Для ==копирования подстроки== из объекта StringBuffer в массив применяйте метод `getChars()`
```java
void getChars(
	int sourceStart, // индекс начала подстроки
	int sourceEnd,   // индекс, находящийся сразу после конца желаемой подстроки
	char[] target,   // массив, который получит нужные символы
	int targetStart  // Индекс внутри массива target, по которому будет копироваться подстрока
)
```

## 📙 6. append()
Метод `append()` ==добавляет строковое представление== любого другого типа данных ==в конец объекта `StringBuffer`==
```java
StringBuffer append(String str)
StringBuffer append(int num)
StringBuffer append(Object obj)
```
- Возвращает `this` (позволяет цепочку вызовов)

## 📙 7. insert()
Метод `insert()` ==вставляет одну строку в другую==.

В `index` указывается индекс, по которому строка будет вставлена в вызывающий объект StringBuffer.
```java
StringBuffer insert(int index, String str)
StringBuffer insert(int index, char ch)
```

## 📙 8. reverse()
Метод `reverse()` ==возвращает объект==, на котором был вызван, ==но с ПРОТИВОПОЛОЖНЫМ ПОРЯДКОМ СЛЕДОВАНИЯ символов==
```java
StringBuffer reverse()
```

## 📙 9. delete() и deleteCharAt()
С помощью методов `delete()` и `deleteCharAt()` можно ==удалять символы внутри объекта== `StringBuffer`
```java
// Удаляет последовательность символов
// В startindex задается индекс первого удаляемого символа, 
// В endindex указывается индекс, находящийся сразу за последним удаляемым символом.
StringBuffer delete(int startIndex, int endIndex) 

// Удаляет символ по индексу, заданному в loc
StringBuffer deleteCharAt(int loc)
```

## 📙 10. replace()
С применением метода `replace()` ==можно заменить один набор символов== внутри объекта `StringBuffer` ==другим набором==:

- `Заменяемая подстрока` указывается посредством индексов `startindex` и `endindex`, что приводит к замене подстроки от `startindex` до `endindex-1`. 
- Строка замены передается в `str`.
```java
StringBuffer replace(int startIndex, int endIndex, String str)
```

## 📙 11. substring()
Вызвав метод `substring()`, можно ==получить часть объекта== `StringBuffer`.
```java
String substring(int startIndex) // От индекса до конца строки
String substring(int startIndex, int endIndex) // Диапазон
```

# 📕 3. Дополнительные методы

| Метод                                                       | Назначение                                                                                            |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| **StringBuffer appendCodePoint(int ch)**                    | Добавляет кодовую точку Unicode в конец объекта. Возвращает ссылку на тот же `StringBuffer`.          |
| **int codePointAt(int i)**                                  | Возвращает кодовую точку Unicode в позиции `i`.                                                       |
| **int codePointBefore(int i)**                              | Возвращает кодовую точку Unicode в позиции, находящейся перед индексом `i`.                           |
| **int codePointCount(int start, int end)**                  | Возвращает количество кодовых точек в диапазоне от `start` до `end-1`.                                |
| **✅int indexOf(String str)**                                | Ищет первое вхождение подстроки `str` и возвращает её индекс, либо `-1`, если совпадение не найдено.  |
| **✅int indexOf(String str, int startIndex)**                | Ищет первое вхождение `str`, начиная с позиции `startIndex`. Возвращает индекс или `-1`.              |
| **✅int lastIndexOf(String str)**                            | Ищет последнее вхождение `str` и возвращает его индекс или `-1`.                                      |
| **✅int lastIndexOf(String str, int startIndex)**            | Ищет последнее вхождение `str`, начиная с позиции `startIndex`. Возвращает индекс или `-1`.           |
| **int offsetByCodePoints(int start, int num)**              | Возвращает индекс, который находится на `num` кодовых точек после позиции `start`.                    |
| **CharSequence subSequence(int startIndex, int stopIndex)** | Возвращает подстроку с диапазоном `[startIndex, stopIndex)`. Реализация интерфейса `CharSequence`.    |
| **✅void trimToSize()**                                      | Уменьшает размер внутреннего буфера `StringBuffer` так, чтобы он соответствовал текущему содержимому. |
