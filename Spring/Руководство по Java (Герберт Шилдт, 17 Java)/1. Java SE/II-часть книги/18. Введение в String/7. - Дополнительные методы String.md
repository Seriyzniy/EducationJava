# Таблица - Некоторые дополнительные методы класса String

| Метод                                                               | Описание                                                                                                         |
| ------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **int codePointAt(int i)**                                          | Возвращает кодовую точку Unicode в позиции `i`.                                                                  |
| **int codePointBefore(int i)**                                      | Возвращает кодовую точку Unicode, которая предшествует позиции `i`.                                              |
| **int codePointCount(int start, int end)**                          | Возвращает количество кодовых точек в строке между индексами `start` и `end-1`.                                  |
| **boolean contains(CharSequence str)**                              | Возвращает `true`, если строка содержит подстроку `str`.                                                         |
| **boolean contentEquals(CharSequence str)**                         | Возвращает `true`, если строка совпадает с содержимым `str`.                                                     |
| **boolean contentEquals(StringBuffer str)**                         | Возвращает `true`, если строка совпадает с содержимым `StringBuffer`.                                            |
| **static String format(String fmtstr, Object... args)**             | Форматирует строку по шаблону `fmtstr` и аргументам `args`.                                                      |
| **static String format(Locale loc, String fmtstr, Object... args)** | Форматирует строку по шаблону `fmtstr`, учитывая локаль `loc`.                                                   |
| **String formatted(Object... args)**                                | Возвращает строку, отформатированную на основе текущей строки как шаблона и аргументов.                          |
| **String indent(int num)**                                          | Добавляет или убирает отступы у каждой строки: положительное `num` → добавляет пробелы, отрицательное → удаляет. |
| **boolean isEmpty()**                                               | Возвращает `true`, если строка пустая (длина = 0).                                                               |
| **Stream lines()**                                                  | Разбивает строку по символам переноса строки и возвращает поток `Stream<String>`.                                |
| **boolean matches(String regExp)**                                  | Проверяет, соответствует ли строка регулярному выражению `regExp`.                                               |
| **int offsetByCodePoints(int start, int num)**                      | Возвращает индекс, который на `num` кодовых точек больше индекса `start`.                                        |
| **String replaceFirst(String regExp, String newStr)**               | Заменяет первую подстроку, соответствующую регулярному выражению, на `newStr`.                                   |
| **String replaceAll(String regExp, String newStr)**                 | Заменяет все подстроки, соответствующие регулярному выражению, на `newStr`.                                      |
| **String[] split(String regExp)**                                   | Разбивает строку на массив подстрок по регулярному выражению.                                                    |
| **String[] split(String regExp, int max)**                          | Разбивает строку на массив подстрок по регулярному выражению, ограничивая количество элементов `max`.            |
| **CharSequence subSequence(int startIndex, int stopIndex)**         | Возвращает подстроку с `startIndex` до `stopIndex`. Требуется интерфейсом `CharSequence`.                        |
| `R transform(Function<? super String, ? extends R> func)`           | Применяет функцию `func` к строке и возвращает результат.                                                        |

***
# Примеры
***
## 🔶 int codePointAt(int i)
Возвращает кодовую точку Unicode в позиции `i`
```java
public class Demo {
    public static void main(String[] args) {
        String s = "A😊B";
        System.out.println(s.codePointAt(1)); // кодовая точка эмодзи
    }
    // Вывод
    // 128522
}
```

## 🔶 int codePointBefore(int i)
Возвращает кодовую точку Unicode, которая предшествует позиции `i`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "A😊B";
        System.out.println(s.codePointBefore(2)); // кодовая точка эмодзи
        // 55357
    }
}
```

## 🔶int codePointCount(int start, int end)
Возвращает количество кодовых точек в строке между индексами `start` и `end-1`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "A😊B";
        System.out.println(s.codePointCount(0, s.length())); // 3
        
        // 3
    }
}
```

***
## 🔶 boolean contains(CharSequence str)
Возвращает `true`, если ==строка содержит подстроку== `str`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "Hello World";
        System.out.println(s.contains("World")); // true
    }
}
```

## 🔶 boolean contentEquals(CharSequence str)
Возвращает `true`, если ==строка совпадает с содержимым== `str`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "Java";
        System.out.println(s.contentEquals("Java")); // true
    }
}
```

## 🔶 boolean contentEquals(StringBuffer str)
Возвращает `true`, если ==строка совпадает с содержимым== `StringBuffer`.
```java
public class Demo {
    public static void main(String[] args) {
        StringBuffer sb = new StringBuffer("Java");
        System.out.println("Java".contentEquals(sb)); // true
    }
}
```

***

## 🔶 static String format(String fmtstr, Object... args)
==Форматирует строку по шаблону== `fmtstr` и аргументам `args`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = String.format("Name: %s, Age: %d", "Tom", 25);
        System.out.println(s); // Name: Tom, Age: 25
    }
}
```

## 🔶 static String format(Locale loc, String fmtstr, Object... args)
Форматирует строку по шаблону `fmtstr`, учитывая локаль `loc`.
```java
import java.util.Locale;

public class Demo {
    public static void main(String[] args) {
        String s = String.format(Locale.FRANCE, "%,.2f", 12345.67);
        System.out.println(s); // 12 345,67
    }
}

```

## 🔶 String formatted(Object... args)
Возвращает строку, ==отформатированную на основе текущей строки== как шаблона и аргументов.
```java
public class Demo {
    public static void main(String[] args) {
        String template = "Hello, %s!".formatted("Alice");
        System.out.println(template); // Hello, Alice!
    }
}
```

***
## 🔶 String indent(int num)
==Добавляет или убирает отступы у каждой строки==: положительное `num` → добавляет пробелы, отрицательное → удаляет.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "Line1\nLine2";
        System.out.println(s.indent(2));
        //   Line1
        //   Line2
    }
}

```

## 🔶 boolean isEmpty()
Возвращает `true`, если ==строка пустая== (длина = 0).
```java
public class Demo {
    public static void main(String[] args) {
        String s = "";
        System.out.println(s.isEmpty()); // true
    }
}
```

## 🔶 `Stream<String> lines()`
==Разбивает строку СИМВОЛАМ ПЕРЕНОСА== строки и возвращает поток `Stream<String>`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "a\nb\nc";
        s.lines().forEach(System.out::println);
        // a
        // b
        // c
    }
}
```

## 🔶 boolean matches(String regExp)
Проверяет, соответствует ли строка регулярному выражению `regExp`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "123";
        System.out.println(s.matches("\\d+")); // true
    }
}
```

## 🔶 int offsetByCodePoints(int start, int num)
Возвращает индекс, который на `num` кодовых точек больше индекса `start`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "A😊B";
        System.out.println(s.offsetByCodePoints(0, 2)); // 3
    }
}

```

***
## 🔶 String replaceFirst(String regExp, String newStr)
==Заменяет первую подстроку==, соответствующую регулярному выражению, на `newStr`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "one two two";
        System.out.println(s.replaceFirst("two", "X")); // one X two
    }
}

```

## 🔶 String replaceAll(String regExp, String newStr)
==Заменяет все подстроки==, соответствующие регулярному выражению, на `newStr`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "one two two";
        System.out.println(s.replaceAll("two", "X")); // one X X
    }
}

```

***
## 🔶 `String[] split(String regExp)`
==Разбивает строку на массив подстрок== по регулярному выражению.
```java
import java.util.Arrays;

public class Demo {
    public static void main(String[] args) {
        String s = "a,b,c";
        String[] arr = s.split(",");
        System.out.println(Arrays.toString(arr)); // [a, b, c]
    }
}

```

## 🔶 `String[] split(String regExp, int max)`
==Разбивает строку на массив подстрок== по регулярному выражению, ==ограничивая количество элементов== `max`.
```java
import java.util.Arrays;

public class Demo {
    public static void main(String[] args) {
        String s = "a,b,c";
        String[] arr = s.split(",", 2);
        System.out.println(Arrays.toString(arr)); // [a, b,c]
    }
}

```

***
## 🔶 CharSequence subSequence(int startIndex, int stopIndex)
Возвращает подстроку с `startIndex` до `stopIndex`. Требуется интерфейсом `CharSequence`.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "Hello";
        System.out.println(s.subSequence(1, 4)); // ell
    }
}

```

## 🔶 `<R> R transform(Function<? super String, ? extends R> func)`
==Применяет функцию== `func` ==к строке== и возвращает результат.
```java
public class Demo {
    public static void main(String[] args) {
        String s = "hello";
        Integer len = s.transform(String::length);
        System.out.println(len); // 5
    }
}

```
