> [!NOTE] ⭐Класс `Class`
> Класс `Class` представляет метаинформацию о классах и интерфейсах. 
> 
> Класс `Class` является обобщенным и объявлен следующим образом:
> ```java
> class Class<T>
> ```
> 
> Класс `Class` реализует несколько интерфейсов, в том числе `Constable` и `TypeDescriptor`.


Позволяет:
- Получать информацию о полях, методах, конструкторах.

Как правило, объект Class получается путем вызова метода getClass ( ) , определенного в Object. 
```java
Class<?> clazz = obj.getClass();
System.out.println(clazz.getName());
```

## Пример работы с getNestHost
```java
class Outer {
    class Inner {}
}

public class Test {
    public static void main(String[] args) {
        System.out.println(Outer.class.getNestHost()); // class Outer
        System.out.println(Outer.Inner.class.getNestHost()); // class Outer
    }
}
```

## Пример работы с isNestMateOf
```java
class Outer {
    class Inner {}
    static class Nested {}
}

public class Test {
    public static void main(String[] args) {
        System.out.println(Outer.Inner.class.isNestMateOf(Outer.Nested.class)); // true
        System.out.println(Outer.Inner.class.isNestMateOf(Test.class)); // false
    }
}
```
