# 📕 1. Класс Package
Инкапсулирует **метаданные о пакете**:
- имя (`getName()`),
    
- версия (`getImplementationVersion()`),
    
- поставщик (`getImplementationVendor()`),
    
- спецификация (`getSpecificationTitle()`, `getSpecificationVersion()`),
    
- аннотации и т. д.

```java
class PkgTest {
    public static void main(String[] args) {
        Package[] pkgs = Package.getPackages();
        for (Package p : pkgs) {
            System.out.println(p.getName() + " : " + p.getImplementationVendor());
        }
    }
}

```

# 📕 2. Класс Module
Появился с модульной системой Java (JPMS).  

Позволяет:
- узнать имя модуля (`getName()`),
    
- получить пакеты (`getPackages()`),
    
- проверить, экспортируется ли пакет (`isExported()`),
    
- добавить доступ к модулю во время выполнения (`addReads()`, `addExports()`),
    
- получить дескриптор (`getDescriptor()`).

```java
Module m = String.class.getModule();

System.out.println("Модуль: " + m.getName());
System.out.println("Пакеты: " + m.getPackages());
```
