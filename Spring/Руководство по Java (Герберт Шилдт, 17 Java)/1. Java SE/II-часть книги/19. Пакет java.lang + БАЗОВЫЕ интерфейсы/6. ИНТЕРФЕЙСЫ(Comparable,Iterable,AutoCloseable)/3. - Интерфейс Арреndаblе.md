## Методы:
К объекту класса, реализующего интерфейс `AppendaЫe`, можно добавлять символ или последовательность символов.
- `Арреndаblе append(char c)`
    
- `Арреndаblе append(CharSequence csq)`
    
- `Арреndаblе append(CharSequence csq, int start, int end)`
    

## Кто реализует:
- `StringBuilder`
    
- `StringBuffer`
    
- `Writer` (например, `FileWriter`, `PrintWriter`)

## Особенность
Очень часто методы пишут так: `void log(String msg)`. Но грамотнее было бы: `void log(Appendable out)`. Тогда ты сможешь подсовывать туда хоть `System.out`, хоть `StringBuilder`, хоть файл.

***
# Пример
Пример собственного класса, который собирает текст в верхнем регистре
```java
class UpperCaseAppender implements Appendable {
    private final StringBuilder sb = new StringBuilder();

    @Override
    public Appendable append(CharSequence csq) {
        sb.append(csq.toString().toUpperCase());
        return this;
    }

    @Override
    public Appendable append(CharSequence csq, int start, int end) {
        sb.append(csq.subSequence(start, end).toString().toUpperCase());
        return this;
    }

    @Override
    public Appendable append(char c) {
        sb.append(Character.toUpperCase(c));
        return this;
    }

    @Override
    public String toString() {
        return sb.toString();
    }
}

public class TestAppendable {
    public static void main(String[] args) {
        Appendable app = new UpperCaseAppender();
        try {
            app.append("hello").append(' ').append("world", 0, 5);
            System.out.println(app); // HELLO WORLD
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

```
