Базовый класс-обертка для `EventListener`. Позволяет расширять слушателей событий с дополнительными свойствами.

## Конструктор
```java
protected EventListenerProxy(EventListener listener) 
// listener — слушатель, который будет обернут
```

## Основные методы
```java
public EventListener getListener() // Возвращает исходный слушатель
```

## Особенности
- Абстрактный класс.
    
- Используется как база для специализированных прокси-слушателей (например, `ChangeListenerProxy`).

## Пример
```java
EventListener myListener = e -> System.out.println("event!");
EventListenerProxy proxy = new EventListenerProxy(myListener) {};
```
