Константы для управления форматированием объектов при использовании `Formattable`.

## Конструктор
Нет публичных конструкторов — это utility-класс с константами

## Основные поля
```java
public static final int LEFT_JUSTIFY
public static final int UPPERCASE
public static final int ALTERNATE
```

## Особенности
- Используется в `Formatter`.
- Задает флаги форматирования (`%-10s`, `%S` и т.п.).
