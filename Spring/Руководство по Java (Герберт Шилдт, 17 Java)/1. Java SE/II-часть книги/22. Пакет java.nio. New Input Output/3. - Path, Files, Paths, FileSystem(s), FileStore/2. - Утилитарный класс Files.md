
> [!NOTE] Класс `Files` — это **утилитарный класс** из пакета `java.nio.file`
> ==Содержит **статические методы** для работы с файлами и каталогами==: 
> 

Основные возможности:
- создание, копирование, перемещение, удаление файлов и папок
- чтение и запись содержимого
- работа с метаданными (размер, дата изменения, атрибуты)
- обход файловой системы


Он ==работает с объектами== `Path`, предоставляя современный, безопасный и кроссплатформенный API для файловых операций.

## Конструкторы
Утилитарный класс не содержит конструкторов. Используются статические методы

## 💡Особенности и подводные камни💡
1. 💡Методы `readAllBytes()` и `readAllLines()` полностью читают файл в память — не использовать для больших файлов.
    
2. `Files.walk()` и `Files.list()` возвращают **стримы**, которые **нужно закрывать** (`try-with-resources`).
    
3. Использование `LinkOption.NOFOLLOW_LINKS` предотвращает автоматическое разыменование симлинков.
    
4. Методы `createFile()` и `copy()` выбрасывают исключение, если файл уже существует (если не указан `REPLACE_EXISTING`).

---
## ✅ Best Practices ✅

- Используй `Path` вместо `File` для кроссплатформенности.
    
- Для записи больших файлов — **буферизованные потоки** (`BufferedReader/Writer`).
    
- Для обхода папок используй `Files.walk()` или `Files.newDirectoryStream()`.

## Пример
```java
import java.nio.file.*;
import java.io.IOException;
import java.nio.charset.StandardCharsets;
import java.util.List;

public class FilesExample {
    public static void main(String[] args) {
        Path file = Path.of("data.txt");

        try {
            // Создание файла и запись строк
            Files.write(file, List.of("Java NIO", "Класс Files"), StandardCharsets.UTF_8);


            // Чтение всех строк
            List<String> lines = Files.readAllLines(file, StandardCharsets.UTF_8);
            lines.forEach(System.out::println);


            // Копирование
            Path copy = Path.of("copy.txt");
            Files.copy(file, copy, StandardCopyOption.REPLACE_EXISTING);


            // Проверка атрибутов
            System.out.println("Размер файла: " + Files.size(file) + " байт");
            System.out.println("Файл существует? " + Files.exists(file));


            // Удаление копии
            Files.deleteIfExists(copy);

        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
