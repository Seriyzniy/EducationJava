
> [!NOTE] Класс **`java.net.http.HttpHeaders`** представляет `набор HTTP-заголовков`  (как из запроса, так и из ответа).
> Он предоставляет **неизменяемый, потокобезопасный** интерфейс  для чтения, фильтрации и итерации по заголовкам.

## Пару слов о назначении
- ❗❗❗`HttpHeaders` используется для хранения **пары “имя–значения”** заголовков HTTP.
    
- Применяется как в `HttpRequest` (заголовки, установленные клиентом),  
    так и в `HttpResponse` (заголовки, полученные от сервера).
    
- ❗❗❗Заголовки могут иметь **множество значений** для одного ключа.
    
- ==ЗНАЧЕНИЯ== заголовков ==НЕ ИЗМЕНЯЮТСЯ== ==после создания объекта== (`immutable`)

---
## Конструкторы / Фабричные методы
Конструктор приватный — объекты создаются через фабрики или из API `HttpRequest` / `HttpResponse`.
```java
//✅ Создаёт экземпляр из карты с заголовками.
// Параметр:
//  map — ключ: имя заголовка, значение: список значений.
public static HttpHeaders of(
	Map<String, List<String>> map,
    BiPredicate<String, String> filter)
```
💬 Этот метод используется редко напрямую.  

Обычно объект `HttpHeaders` возвращается методами:
- 🟢`HttpResponse.headers()` — для заголовков ответа,
- 🟢`HttpRequest.headers()` — для заголовков запроса.

## Методы
### 1️⃣ Получение заголовков
```java
//✅ Возвращает немодифицируемую Map всех заголовков.
// Ключи — имена заголовков (в нижнем регистре), значения — списки строк.
public Map<String, List<String>> map()

//✅ Возвращает список значений указанного заголовка.
// Параметр:
//  name — имя заголовка (регистр не имеет значения).
// Возвращает: List<String> или пустой список, если заголовок отсутствует.
public List<String> allValues(String name)


//✅ Возвращает первое значение указанного заголовка (если есть).
// Возвращает Optional<String>.
public Optional<String> firstValue(String name)

//✅ Возвращает первое значение указанного заголовка, приведённое к long.
// Бросает NumberFormatException, если значение не число.
public OptionalLong firstValueAsLong(String name)
```

### 2️⃣ Проверка и фильтрация
```java
//✅ Проверяет, содержит ли заголовок с указанным именем.
public boolean hasHeader(String name)

//✅ Применяет фильтр и возвращает новый HttpHeaders с подходящими парами.
public HttpHeaders filter(BiPredicate<String, String> predicate)
```

### 3️⃣ Итерация по заголовкам
```java
//✅ Возвращает Stream<Map.Entry<String, List<String>>> для функциональных операций.
public Stream<Map.Entry<String, List<String>>> entries()

//✅ Возвращает Set<String> имён всех заголовков.
public Set<String> keySet()
```

### 4️⃣Общие методы
```java
//✅ Сравнивает два объекта HttpHeaders (по содержимому).
public boolean equals(Object o)

//✅ Возвращает хэш-код.
public int hashCode()

//✅ Возвращает строковое представление (имя: значение).
public String toString()
```

## 💡 Особенности и подводные камни 💡

1️⃣ `HttpHeaders` — **==неизменяемый класс==**: нельзя добавить или удалить заголовок после создания.  

2️⃣ 🧠Имена заголовков ==**регистр-независимы**==, но возвращаются в нижнем регистре.  

3️⃣ 🧠Некоторые заголовки могут иметь **несколько значений** (например, `Set-Cookie`, `Accept`).  

4️⃣ ⭐⭐Пустой список вместо `null` — безопасно для NPE.  

5️⃣ 🧠Метод `firstValueAsLong()` полезен для заголовков вроде `Content-Length`, но может выбросить исключение.  

6️⃣ Для фильтрации удобно использовать `filter()`, чтобы извлечь нужные пары.  

7️⃣ 🧠Заголовки ответа могут включать системные поля (`Date`, `Server`, `Content-Type`, `Content-Length` и др.),  
которые клиент не всегда задаёт явно.  

8️⃣ `HttpRequest.Builder.header()` не принимает `null` — только строки.  

9️⃣ ❗❗❗Для работы с cookies через `HttpHeaders` нужно использовать методы `allValues("Set-Cookie")`.

## 💎 Best Practices 💎

✅ 💡Для получения ВСЕХ заголовков из ответа:
```java
HttpHeaders headers = response.headers();

headers.map().forEach((name, values) ->
    System.out.println(name + " -> " + values));
```

✅ 💡Для получения ОДНОГО конкретного заголовка:
```java
response.headers().firstValue("Content-Type")
        .ifPresent(System.out::println);
```

✅ Для ЗАГОЛОВКОВ С ЧИСЛОВЫМИ значениями:
```java
long contentLength = response.headers()
        .firstValueAsLong("Content-Length")
        .orElse(-1);
```

✅ Для фильтрации по условию:
```java
HttpHeaders serverHeaders = response.headers()
        .filter((name, value) -> name.startsWith("server"));
```

✅ Для вывода в человекочитаемом виде:
```java
System.out.println(response.headers().toString());
```

✅ Для чтения `Set-Cookie`:
```java
List<String> cookies = response.headers().allValues("Set-Cookie");
```

## Пример
```java
import java.net.http.*;
import java.net.URI;
import java.util.*;

public class HttpHeadersDemo {
    public static void main(String[] args) throws Exception {
        // 1️⃣ Создаём HttpClient
        HttpClient client = HttpClient.newHttpClient();

        // 2️⃣ Отправляем запрос
        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://httpbin.org/headers"))
                .header("User-Agent", "HttpHeadersDemo/1.0")
                .GET()
                .build();

        HttpResponse<String> response =
                client.send(
	                request, 
	                HttpResponse.BodyHandlers.ofString()
	            );

        // 3️⃣ Извлекаем заголовки
        HttpHeaders headers = response.headers();

        // 4️⃣ Просмотр всех заголовков
        System.out.println("--- Все заголовки ---");
        headers
	        .map()
	        .forEach((k, v) -> System.out.println(k + " -> " + v));

        // 5️⃣ Получение одного заголовка
        System.out.println("\nContent-Type: " +
                headers.firstValue("Content-Type").orElse("не найден"));

        // 6️⃣ Пример числового заголовка
        long len = headers.firstValueAsLong("Content-Length").orElse(-1);
        System.out.println("Content-Length: " + len);

        // 7️⃣ Фильтрация
        System.out.println("\n--- Заголовки, содержащие 'server' ---");
        headers
	        .filter((name, val) -> name.contains("server"))
            .map()
            .forEach((n, v) -> System.out.println(n + " -> " + v));
    }
}
```
