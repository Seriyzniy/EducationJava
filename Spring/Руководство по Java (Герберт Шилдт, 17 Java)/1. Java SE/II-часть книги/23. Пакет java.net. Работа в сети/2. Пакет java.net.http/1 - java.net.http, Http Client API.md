Появился в **Java 11** — современная альтернатива `HttpURLConnection`.

Для многих видов сетей НТТР возможности, определенные АРI интерфейсом в java.net.http, могут обеспечить превосходные решения. 

Помимо оптимизированного и простого в использовании АРI-интерфейса к другим преимуществам относятся 
- ⭐⭐поддержка асинхронной связи, ⭐⭐
- НТТР/2 и 
- управление потоком. 

> [!NOTE] ⭐Он также поддерживает протокол `WebSocket` для `ДВУНАПРАВЛЕННОЙ связи`.⭐
> 

В целом НТТР Client API разработан в качестве более совершенной альтернативы функциональности класса HttpURLConnection.

## Ключевые классы:
| Класс          | Назначение                                                    |
| -------------- | ------------------------------------------------------------- |
| `HttpClient`   | управляет соединением, `отправляет` запрос и `получает` ответ |
| `HttpRequest`  | инкапсулирует HTTP-`запрос`                                   |
| `HttpResponse` | инкапсулирует `ответ`                                         |
