
> [!NOTE] Класс **`java.net.http.HttpRequest`** представляет `HTTP-запрос`, который можно отправить с помощью `HttpClient`
> Он инкапсулирует всё, что нужно для отправки HTTP-запроса:  
> - метод (GET, POST, PUT и т.д.), 
> - URI, 
> - заголовки, 
> - тело запроса, 
> - а также таймаут и версия протокола.

### Пару слов о назначении
- `HttpRequest` — это **immutable (==неизменяемый==)** объект, описывающий HTTP-запрос.
    
- ❗❗Создаётся через **Builder (паттерн "Строитель (Builder)")**.
    
- Поддерживает установку заголовков, таймаутов, тел запросов и указание версии HTTP (1.1 / 2).
    
- 💡Работает в паре с `HttpClient` и `HttpResponse`.
    
- 💡Поддерживает ==синхронную== и ==асинхронную== отправку.

## Конструктор / Фабричные методы
Конструктор приватный — создание запросов производится через билдер.
```java
//✅ Создаёт новый экземпляр билдера для указанного URI.
// Параметр:
//  uri — адрес ресурса (например, "https://example.com")
public static HttpRequest.Builder newBuilder(URI uri)


//✅ Создаёт пустой билдер (URI можно указать позже методом uri()).
public static HttpRequest.Builder newBuilder()
```

💬 ⭐⭐После конфигурации запроса нужно вызвать `build()`,  
чтобы получить готовый `HttpRequest`.

## 📙 Вложенный интерфейс Builder
Используется для пошагового создания `HttpRequest`.
```java
public static interface HttpRequest.Builder {

    //✅ Устанавливает URI запроса.
    HttpRequest.Builder uri(URI uri);

    //✅ Устанавливает HTTP-метод и тело запроса.
    HttpRequest.Builder method(String method, HttpRequest.BodyPublisher bodyPublisher);

    //✅ Устанавливает GET-запрос без тела.
    HttpRequest.Builder GET();

    //✅ Устанавливает POST-запрос с телом.
    HttpRequest.Builder POST(HttpRequest.BodyPublisher bodyPublisher);

    //✅ Устанавливает PUT-запрос с телом.
    HttpRequest.Builder PUT(HttpRequest.BodyPublisher bodyPublisher);

    //✅ Устанавливает DELETE-запрос без тела.
    HttpRequest.Builder DELETE();

    //✅ Устанавливает PATCH-запрос с телом.
    HttpRequest.Builder PATCH(HttpRequest.BodyPublisher bodyPublisher);

    //✅ Устанавливает HEAD-запрос без тела.
    HttpRequest.Builder HEAD();

    //✅ Устанавливает OPTIONS-запрос без тела.
    HttpRequest.Builder OPTIONS();

    //✅ Добавляет один заголовок.
    HttpRequest.Builder header(String name, String value);

    //✅ Добавляет несколько заголовков через varargs.
    HttpRequest.Builder headers(String... headers);

    //✅ Устанавливает таймаут выполнения запроса.
    HttpRequest.Builder timeout(Duration duration);

    //✅ Устанавливает версию HTTP (HTTP_1_1 или HTTP_2).
    HttpRequest.Builder version(HttpClient.Version version);

    //✅ Устанавливает политику перенаправлений.
    HttpRequest.Builder followRedirects(HttpClient.Redirect policy);

    //✅ Устанавливает приоритет запроса (HTTP/2 feature).
    HttpRequest.Builder priority(int priority);

    //✅ Включает/выключает ожидание тела ответа.
    HttpRequest.Builder expectContinue(boolean enable);

    //✅ Устанавливает обработчик тела запроса с возможностью отмены.
    HttpRequest.Builder bodyPublisher(HttpRequest.BodyPublisher bodyPublisher, 
                                     java.util.function.BiConsumer<HttpRequest, Throwable> onRequestBodySent);

    //✅ Устанавливает пользовательский селектор протокола.
    HttpRequest.Builder sslContext(javax.net.ssl.SSLContext sslContext);

    //✅ Устанавливает аутентификацию Basic Auth.
    HttpRequest.Builder auth(java.net.Authenticator authenticator);

    //✅ Устанавливает пользовательский таймаут для подключения.
    HttpRequest.Builder connectTimeout(Duration duration);

    //✅ Устанавливает кеширующую политику.
    HttpRequest.Builder cacheControl(java.net.CacheControl cacheControl);

    //✅ Завершает построение и создаёт объект HttpRequest.
    HttpRequest build();
}
```
### Вспомогательные методы Builder
```java
//✅ Устанавливает URI из строки (удобная обёртка)
default HttpRequest.Builder uri(String uri) {
    return uri(URI.create(uri));
}

//✅ Устанавливает JSON-тело с автоматическим Content-Type
default HttpRequest.Builder jsonBody(String json) {
    return header("Content-Type", "application/json")
           .POST(HttpRequest.BodyPublishers.ofString(json));
}

//✅ Устанавливает form-data тело
default HttpRequest.Builder formData(String formData) {
    return header("Content-Type", "application/x-www-form-urlencoded")
           .POST(HttpRequest.BodyPublishers.ofString(formData));
}

//✅ Устанавливает Bearer токен авторизации
default HttpRequest.Builder bearerAuth(String token) {
    return header("Authorization", "Bearer " + token);
}

//✅ Устанавливает Basic авторизацию
default HttpRequest.Builder basicAuth(String username, String password) {
    String credentials = java.util.Base64.getEncoder()
        .encodeToString((username + ":" + password).getBytes());
    return header("Authorization", "Basic " + credentials);
}
```

## 📙 Вложенный `интерфейс` `BodyPublisher`
Используется ==для ПЕРЕДАЧИ ТЕЛА запроса==.

```java
public static interface BodyPublisher extends Flow.Publisher<ByteBuffer> 
{
    long contentLength();
}
```
💬 Создаётся через статические фабричные методы `HttpRequest.BodyPublishers`.

### Вспомогательная `фабрика` `BodyPublisherS `
```java
public static class BodyPublishers {

    //✅ Без тела (например, для GET или DELETE)
    public static BodyPublisher noBody();


    //✅ Из строки (обычно для JSON)
    public static BodyPublisher ofString(String body);
    
    //✅ Создаёт BodyPublisher из строки с указанной кодировкой.
	public static BodyPublisher ofString(String body, Charset charset)


    //✅ Из массива байт
    public static BodyPublisher ofByteArray(byte[] bytes);
    
	//✅ Создаёт BodyPublisher из части массива байт.
	public static BodyPublisher ofByteArray(
		byte[] bytes, 
		int offset, 
		int length)


    //✅ Из файла
    public static BodyPublisher ofFile(Path path);
    
    //✅ Создаёт BodyPublisher, читающий часть файла 
    // (с позиции offset и длиной count байт).
	public static BodyPublisher ofFile(
		Path path, 
		long offset, 
		long count)


    //✅ Из потока байт (InputStream)
    public static BodyPublisher ofInputStream(
	    Supplier<? extends InputStream> streamSupplier);
	    
	
	//✅ Объединяет несколько BodyPublisher’ов в один.
	// Параметр: Iterable<BodyPublisher> — список частей.
	public static BodyPublisher concat(
		Iterable<? extends BodyPublisher> publishers)
		
	//✅ Создаёт BodyPublisher из Publisher<ByteBuffer> напрямую.
	public static BodyPublisher fromPublisher(
		Flow.Publisher<ByteBuffer> publisher)

	//✅ (Java 21+) Создаёт multipart/form-data publisher из частей (файлы, текст).
	// Позволяет отправлять сложные формы (аналог POST форм).
	public static BodyPublisher ofMultipartData(
		Map<String, Object> data, 
		String boundary)

	
}
```

## 💡 Особенности и подводные камни 💡

1️⃣ `HttpRequest` **==неизменяем==** — после создания через `build()` изменить ничего нельзя.  

2️⃣ Используется **Builder**, поэтому можно безопасно конфигурировать запрос пошагово.  

3️⃣ ❗❗❗При создании `POST`, `PUT`, `PATCH` — обязательно нужно указать `BodyPublisher`, иначе `IllegalArgumentException`.  

4️⃣ 🧠Если не указать метод явно, по умолчанию используется **GET**.  

5️⃣ ⭐⭐Таймаут запроса (`timeout()`) ОТНОСИТСЯ ТОЛЬКО к **ВРЕМЕНИ ВЫПОЛНЕНИЯ**,  А НЕ к УСТАНОВЛЕНИЮ соединения — это задаётся в `HttpClient.connectTimeout()`.  

6️⃣ HTTP/2 не всегда поддерживается сервером — клиент может перейти на HTTP/1.1 автоматически.  

7️⃣ ⭐⭐Если тело запроса большое — лучше использовать `BodyPublishers.ofFile(...)` вместо `ofString()`.  

8️⃣ ⭐⭐При работе с ==JSON== не забывай ==ЗАГОЛОВОК==:⭐⭐
```java
.header("Content-Type", "application/json")
```

9️⃣ 🧠Для авторизации — можно установить `Authenticator` в `HttpClient`,  
но не в `HttpRequest`.

## 💎 Best Practices 💎

✅ **Создавай HttpRequest через builder** — это читаемо и безопасно:
```java
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://example.com"))
    .header("Accept", "application/json")
    .GET()
    .build();
```

✅ **Для POST и PUT** — используй `BodyPublishers.ofString()` или `ofFile()`:
```java
HttpRequest request = HttpRequest.newBuilder(uri)
    .header("Content-Type", "application/json")
    .POST(HttpRequest.BodyPublishers.ofString("{\"name\":\"Seriy\"}"))
    .build();
```

✅ **Добавляй таймаут для надёжности:**
```java
.timeout(Duration.ofSeconds(10))
```

✅ ⭐⭐**Для больших тел** — применяй ==потоковую передачу== (`ofFile` / `ofInputStream`).  

✅ ❗❗❗**Не создавай `HttpRequest` заново** для одинаковых запросов — ==можно переиспользовать== объект.  

✅ **Используй UTF-8** при кодировании тела запроса, особенно для JSON.

## Пример
```java
import java.net.http.*;
import java.net.URI;
import java.time.Duration;

public class HttpRequestDemo {
    public static void main(String[] args) throws Exception {
        // 1️⃣ Создаём HttpClient
        HttpClient client = HttpClient.newHttpClient();


        // 2️⃣ Формируем GET-запрос
        HttpRequest getRequest = HttpRequest.newBuilder()
                .uri(URI.create("https://httpbin.org/get"))
                .header("User-Agent", "JavaHttpRequestDemo/1.0")
                .timeout(Duration.ofSeconds(5))
                .GET()
                .build();

        HttpResponse<String> getResponse = 
	        client.send(
                getRequest, 
                HttpResponse.BodyHandlers.ofString()
            );

        System.out.println("GET статус: " + getResponse.statusCode());
        System.out.println("GET тело:\n" + getResponse.body());


        // 3️⃣ Формируем POST-запрос
        String json = "{\"login\":\"Seriy\",\"role\":\"dev\"}";

        HttpRequest postRequest = HttpRequest.newBuilder()
                .uri(URI.create("https://httpbin.org/post"))
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(json))
                .timeout(Duration.ofSeconds(10))
                .build();

        HttpResponse<String> postResponse =
                client.send(
	                postRequest, 
	                HttpResponse.BodyHandlers.ofString()
	            );

        System.out.println("\nPOST статус: " + postResponse.statusCode());
        System.out.println("POST тело:\n" + postResponse.body());
    }
}
```
