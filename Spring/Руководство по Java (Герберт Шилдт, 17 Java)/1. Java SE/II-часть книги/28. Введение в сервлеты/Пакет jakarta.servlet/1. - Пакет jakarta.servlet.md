Пакет `jakarta.servlet` содержит несколько интерфейсов и классов, образующих инфраструктуру, в которой функционируют сервлеты



> [!NOTE] `Базовые интерфейсы` Servlet и  ServletRequest и ServletResponse
> Наиболее важным из них является `Servlet`.
> ⭐ `Все сервлеты должны реализовы вать интерфейс Servlet` или расширять реализующий его класс. 
> 
> Интерфейсы `ServletRequest` и `ServletResponse` тоже очень важны.

## Основные ИНТЕФРЕЙСЫ в пакете jakarta.servlet

| Интерфейс        | Назначение                                                                                                     |
| ---------------- | -------------------------------------------------------------------------------------------------------------- |
| ⭐**Servlet**     | Объявляет `методы жизненного цикла` сервлета                                                                   |
|                  |                                                                                                                |
| 🧠ServletConfig  | Позволяет сервлетам `ПОЛУЧАТЬ` `параметры инициализации`                                                       |
| 🧠ServletContext | Позволяет сервлетам `регистрировать события в журнале` и `получать доступ к информации`, касающейся `их среды` |
|                  |                                                                                                                |
| ServletRequest   | Используется `для чтения данных` из клиентского `запроса`                                                      |
| ServletResponse  | Используется `для записи данных` в клиентский `ответ`                                                          |
|                  |                                                                                                                |

## Основные КЛАССЫ в пакете jakarta.servlet

| Класс                | Назначение                                               |
| -------------------- | -------------------------------------------------------- |
| GenericServlet       | Реализует интерфейсы Servlet и ServletConfig             |
|                      |                                                          |
| ServletInputStream   | Инкапсулирует поток ввода для чтения запросов от клиента |
| ServletOutputStream  | Инкапсулирует поток вывода для записи ответов клиенту    |
|                      |                                                          |
| ServletException     | Указывает на то, что в сервлете произошла ошибка         |
| UnavailableException | Указывает на то, что сервлет недоступен                  |
