
> [!NOTE] `HashMap` — это `основная реализация интерфейса` `Map`. 
> Класс `HashМap` расширяет `AbstractMap` и реализует интерфейс `Мар`. 
> 
> Для `хранения карты` он ==применяет хеш-таблицу==, что позволяет времени выполнения операций `get()` и `put()` оставаться постоянным даже для крупных карт.

### Особенности реализации
⭐В основе лежит **хэш-таблица** (⭐массив бакетов, где коллизии разрешаются через связные списки или красно-чёрные деревья при больших нагрузках).

Ключи в `HashMap` **уникальны**, значения могут повторяться.  

Порядок элементов **не гарантируется** (для упорядоченной версии — `LinkedHashMap`, для сортированной — `TreeMap`).

## 💡 Особенности и подводные камни 💡
- 🔶**Неупорядоченность**: порядок хранения элементов произвольный, зависит от хеш-функции.
    
- 🔶**Null**: допускается **один `null`-ключ** и любое количество `null`-значений.
    
- 🔶**Коллизии**: при большом числе коллизий список превращается в **красно-чёрное дерево** (с Java 8).
    
- 🔶**Не потокобезопасна** — для многопоточности использовать `ConcurrentHashMap`.
    
- 🔶**capacity & loadFactor**: слишком маленькая capacity приведёт к частым перераспределениям, слишком большая — к перерасходу памяти.
    
- 🔶Итерация по `HashMap` **fail-fast** — при изменении карты во время обхода выбрасывается `ConcurrentModificationException`.

***
## ✅ Best Practices ✅ 
- Для многопоточной среды → использовать `ConcurrentHashMap`, а не синхронизацию `HashMap`.
    
- Задавать **правильный `initialCapacity`**, если заранее известно количество элементов (экономия на resize).
    
- 💡Всегда переопределять `hashCode()` и `equals()` для ключевых объектов (иначе могут теряться данные).
    
- 💡Использовать `entrySet()` для итерации, если нужны и ключ, и значение.

## Пример
```java
import java.util.*;

public class HashMapDemo {
    public static void main(String[] args) {
        Map<String, Integer> ages = new HashMap<>();

        // Добавление элементов
        ages.put("Alice", 30);
        ages.put("Bob", 25);
        ages.put("Charlie", 28);

        // Получение значения
        System.out.println("Возраст Alice: " + ages.get("Alice"));

        // Проверка наличия ключа
        if (ages.containsKey("Bob")) {
            System.out.println("Есть данные для Bob");
        }

        // Итерация по ключам и значениям
        for (Map.Entry<String, Integer> entry : ages.entrySet()) {
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }

        // Использование computeIfAbsent
        ages.computeIfAbsent("Diana", k -> 22);

        // Сортировка по ключу (через Stream API)
        ages.entrySet().stream()
            .sorted(Map.Entry.comparingByKey())
            .forEach(System.out::println);
    }
}
```
