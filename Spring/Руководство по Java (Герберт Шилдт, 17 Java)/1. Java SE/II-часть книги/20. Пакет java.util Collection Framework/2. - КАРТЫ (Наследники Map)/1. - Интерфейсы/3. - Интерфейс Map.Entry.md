
> [!NOTE] `Map.Entry` — это `вложенный интерфейс` внутри Map, ==представляющий одну пару ключ-значение==

Используется при работе с методами `entrySet()`, ==когда НУЖНО ИТЕРИРОВАТЬСЯ по _ключам и значениям одновременно_==.

```java
public interface Map.Entry<K, V> {
    // Ключ пары
    K getKey();

    // Значение пары
    V getValue();

    // Заменить значение, вернуть старое
    V setValue(V value);

    // Сравнение по ключу и значению
    boolean equals(Object o);

    // Хеш по ключу и значению
    int hashCode();

    // (Java 8) Сравнение по ключу с заданным компаратором
    public static <K extends Comparable<? super K>, V> Comparator<Map.Entry<K, V>> comparingByKey();

    // (Java 8) Сравнение по ключу с пользовательским компаратором
    public static <K, V> Comparator<Map.Entry<K, V>> comparingByKey(Comparator<? super K> cmp);

    // (Java 8) Сравнение по значению (естественный порядок)
    public static <K, V extends Comparable<? super V>> Comparator<Map.Entry<K, V>> comparingByValue();

    // (Java 8) Сравнение по значению с пользовательским компаратором
    public static <K, V> Comparator<Map.Entry<K, V>> comparingByValue(Comparator<? super V> cmp);
}

```

***
## ✅Best Practices✅

- 💡 Использовать `for (Map.Entry<K,V> e : map.entrySet())` для итерации💡, когда нужны **и ключ, и значение**.
    
- 💡Для сортировки💡 `Map` по ключам или значениям удобно применять `entrySet().stream().sorted(...)`.
    
- ❗❗Никогда не менять `Map` во время итерации❗❗ по `entrySet()` без использования `Iterator.remove()`.

***
## Пример
```java
import java.util.*;

public class EntryDemo {
    public static void main(String[] args) {
        Map<String, Integer> scores = new HashMap<>();
        scores.put("Alice", 90);
        scores.put("Bob", 75);
        scores.put("Charlie", 85);

        // ✅Итерация по Entry
        for (Map.Entry<String, Integer> entry : scores.entrySet()) {
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }

        // ✅Изменение значения через setValue()
        for (Map.Entry<String, Integer> entry : scores.entrySet()) {
            if (entry.getKey().equals("Bob")) {
                entry.setValue(80); // обновляем оценку
            }
        }

        // Сортировка по значению
        scores.entrySet().stream()
              .sorted(Map.Entry.comparingByValue())
              .forEach(System.out::println);
    }
}
```
