
> [!NOTE] `InputEvent` — это абстрактный класс, представляющий `общее поведение` `всех событий ввода` пользователя:
> нажатие клавиш, клики мышью, перемещения курсора, прокрутка колеса и т.п
> 
> Он хранит информацию о **времени**, **модификаторах клавиатуры (Ctrl, Alt, Shift)** и **состоянии кнопок мыши** в момент генерации события.

## Иерархия
```java
java.lang.Object
   └── java.util.EventObject
        └── java.awt.AWTEvent
             └── java.awt.event.ComponentEvent
                  └── java.awt.event.InputEvent
```
🧠 От `InputEvent` наследуются:
- `KeyEvent` — события клавиатуры,
- `MouseEvent` — события мыши,
- `MouseWheelEvent` — прокрутка колесика мыши.

### 🧭 Назначение
`InputEvent` описывает **общие свойства всех событий ввода**.  
Он не используется напрямую, а служит **базовым классом** для конкретных событий.

Хранит:
- флаги нажатых клавиш-модификаторов (`Ctrl`, `Alt`, `Shift`, `Meta`),
- время генерации события,
- состояние кнопок мыши,
- информацию о том, были ли они ==нажаты или отпущены.==

---
## ! Конструктор - protected !
Поскольку `InputEvent` — **абстрактный класс**, его нельзя создавать напрямую.  
Однако его конструктор используется наследниками (`KeyEvent`, `MouseEvent`, `MouseWheelEvent`).
```java
// Конструктор для инициализации базовой информации о событии ввода
// src — источник события (компонент, например Frame, Button)
// id — тип события (например, KEY_PRESSED, MOUSE_CLICKED и т.д.)
// when — отметка времени события в миллисекундах
// modifiers — набор битовых флагов, описывающих нажатые клавиши (Shift, Ctrl, Alt, Meta)
protected InputEvent(Component src, int id, long when, int modifiers)
```

## Методы
Класс `InputEvent` наследует общие методы `ComponentEvent` (`getComponent()`, `paramString()`),  
но добавляет **собственные методы**, обеспечивающие доступ к информации о клавишах, мыши и времени события.

### Работа с модификаторами клавиш
```java
// Возвращает набор битовых флагов, описывающих модификаторы, активные при событии
// Например, комбинации Ctrl, Alt, Shift
public int getModifiers()

// Возвращает расширенные модификаторы (для новых клавиш и кнопок мыши)
// Этот метод предпочтителен начиная с Java 1.4
public int getModifiersEx()

// Проверяет, была ли нажата клавиша Alt
public boolean isAltDown()

// Проверяет, была ли нажата клавиша Control
public boolean isControlDown()

// Проверяет, была ли нажата клавиша Meta (Command на macOS, Windows key на Windows)
public boolean isMetaDown()

// Проверяет, была ли нажата клавиша Shift
public boolean isShiftDown()
```

#### Константы модификаторов (для getModifiersEx()):
|Константа|Описание|
|---|---|
|`SHIFT_DOWN_MASK`|клавиша Shift нажата|
|`CTRL_DOWN_MASK`|клавиша Ctrl нажата|
|`META_DOWN_MASK`|клавиша Meta нажата|
|`ALT_DOWN_MASK`|клавиша Alt нажата|
|`ALT_GRAPH_DOWN_MASK`|клавиша AltGr нажата|
|`BUTTON1_DOWN_MASK`|нажата левая кнопка мыши|
|`BUTTON2_DOWN_MASK`|нажата средняя кнопка мыши|
|`BUTTON3_DOWN_MASK`|нажата правая кнопка мыши|

### Время события
```java
// Возвращает отметку времени (в миллисекундах) момента генерации события
public long getWhen()
```
> Полезно для анализа производительности и фильтрации событий.

### Состояние кнопок мыши
```java
// Возвращает true, если была нажата конкретная кнопка мыши (1, 2, 3)
// Использует маски BUTTON1_DOWN_MASK и т.д.
public static boolean isButtonDown(int modifiers, int button)
```

### Управление распространением событий
```java
// Устанавливает флаг, что событие уже обработано
public void consume()

// Проверяет, было ли событие "поглощено"
// Если true — другие обработчики не должны реагировать на него
public boolean isConsumed()
```
> Используется для предотвращения многократной реакции на одно и то же событие (например, если слушателей несколько).

## 💡 Особенности и подводные камни 💡

1️⃣ 💡**Это абстрактный класс.**  
Его нельзя использовать напрямую — только через наследников (`KeyEvent`, `MouseEvent`, `MouseWheelEvent`).

2️⃣ 💡**Используй `getModifiersEx()` вместо `getModifiers()`.**  
Старый метод возвращает устаревшие коды клавиш; ==с Java 1.4 введён новый формат== с поддержкой дополнительных кнопок мыши и клавиш.

3️⃣ 💡**События могут приходить с несколькими модификаторами одновременно.**  
Например, `Ctrl+Shift+Click` → комбинация битовых флагов.

4️⃣ ⭐⭐**Метод `consume()` очень важен.**  ⭐⭐
Если ты вызвал `e.consume()`, то событие считается “обработанным”,  
и AWT больше не будет его передавать другим слушателям или компонентам.

5️⃣ **Некоторые события могут быть "синтетическими"**,  
то есть созданы программно, а не пользователем — у них `when` может быть 0.

---

## 💎 Best Practices 💎

✅ Для проверки модификаторов всегда используй **битовые маски `getModifiersEx()`**:
```java
if ((e.getModifiersEx() & InputEvent.CTRL_DOWN_MASK) != 0) {
    // Ctrl был зажат
}
```

✅ При работе с событиями мыши проверяй `consume()` —  
если событие уже обработано, не дублируй реакцию.

✅ Используй `getWhen()` для измерения времени между событиями —  
например, чтобы отфильтровать "дребезг" или двойные клики.

✅ Если реализуешь собственный компонент с пользовательским вводом,  
наследуйся от `Component` и обрабатывай `InputEvent` в своих `processEvent()` методах.

✅ Всегда различай устаревшие `getModifiers()` и новые `getModifiersEx()`  
(особенно при совместимости с кодом до Java 1.4).

## Пример использования (через MouseEvent и KeyEvent)
Поскольку `InputEvent` — абстрактный, приведём пример его использования **через наследников**:
```java
import java.awt.*;
import java.awt.event.*;

public class InputEventDemo extends Frame implements KeyListener, MouseListener {
    private TextArea info;

    public InputEventDemo() {
        setLayout(new BorderLayout());
        info = new TextArea();
        add(info, BorderLayout.CENTER);

        addKeyListener(this);
        addMouseListener(this);

        setSize(400, 300);
        setVisible(true);

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
    }

    @Override
    public void keyPressed(KeyEvent e) {
        info.append("Нажата клавиша: " + e.getKeyChar() + "\n");
        if ((e.getModifiersEx() & InputEvent.CTRL_DOWN_MASK) != 0) {
            info.append("→ Нажат Ctrl\n");
        }
    }

    @Override
    public void mousePressed(MouseEvent e) {
        info.append("Клик мыши: кнопка " + e.getButton() + "\n");
        if ((e.getModifiersEx() & InputEvent.SHIFT_DOWN_MASK) != 0) {
            info.append("→ Нажат Shift во время клика\n");
        }
    }

    public void keyReleased(KeyEvent e) {}
    public void keyTyped(KeyEvent e) {}
    public void mouseClicked(MouseEvent e) {}
    public void mouseReleased(MouseEvent e) {}
    public void mouseEntered(MouseEvent e) {}
    public void mouseExited(MouseEvent e) {}

    public static void main(String[] args) {
        new InputEventDemo();
    }
}

```
