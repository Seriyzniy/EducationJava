
> [!NOTE] `MouseWheelEvent` — это специализированный подкласс `MouseEvent`,  который описывает `вращение колеса прокрутки мыши` (scroll wheel).
> 🧠 Это событие сообщает приложению: 
> - на сколько щелчков прокручено колесо,
> - в каком направлении (вверх или вниз),
> - а также — было ли это «постраничное» или «построчное» прокручивание.

## Иерархия
```java
java.lang.Object
   └── java.util.EventObject
        └── java.awt.AWTEvent
             └── java.awt.event.ComponentEvent
                  └── java.awt.event.InputEvent
                       └── java.awt.event.MouseEvent
                            └── java.awt.event.MouseWheelEvent
```

### 🧭 Назначение
`MouseWheelEvent` используется для обработки **прокрутки колесом мыши**.  

Он генерируется компонентами, у которых включена поддержка колеса,  
и обрабатывается объектами, реализующими интерфейс:
```java
public interface MouseWheelListener {
    void mouseWheelMoved(MouseWheelEvent e);
}
```
> Когда пользователь крутит колесо, компоненту отправляется объект `MouseWheelEvent`,  
> содержащий всю информацию о направлении, интенсивности и типе прокрутки.

## Конструктор
```java
// Создаёт событие прокрутки колесом мыши
// src — компонент, над которым произошло событие
// id — тип события (обычно MouseEvent.MOUSE_WHEEL)
// when — отметка времени (в мс)
// modifiers — модификаторы клавиш (Ctrl, Shift, Alt и т.п.)
// x, y — координаты курсора внутри компонента
// xAbs, yAbs — координаты курсора на экране
// clickCount — количество кликов (обычно 0 для колеса)
// popupTrigger — является ли событие всплывающим
// scrollType — тип прокрутки (WHEEL_UNIT_SCROLL или WHEEL_BLOCK_SCROLL)
// scrollAmount — количество единиц на один щелчок
// wheelRotation — направление вращения (отрицательное вверх, положительное вниз)
public MouseWheelEvent(
	Component src, 
	int id, 
	long when, 
	int modifiers,
	int x, 
	int y, 
	int xAbs, 
	int yAbs, 
	int clickCount,
    boolean popupTrigger, 
    int scrollType, 
    int scrollAmount, 
    int wheelRotation)
```

### Константы класса
|Константа|Значение|Описание|
|---|---|---|
|`WHEEL_UNIT_SCROLL`|0|Прокрутка **по строкам** (типичная ситуация)|
|`WHEEL_BLOCK_SCROLL`|1|Прокрутка **по блокам/страницам**|
|`MOUSE_WHEEL`|507|ID события колеса мыши|

### Основные поля
|Поле|Тип|Описание|
|---|---|---|
|`scrollType`|`int`|Тип прокрутки (`WHEEL_UNIT_SCROLL` или `WHEEL_BLOCK_SCROLL`)|
|`scrollAmount`|`int`|Количество единиц (строк) на один щелчок|
|`wheelRotation`|`int`|Направление и количество шагов прокрутки (отрицательное — вверх, положительное — вниз)|
|`preciseWheelRotation`|`double`|Точное (неокруглённое) значение вращения колеса|

---
## Методы
Класс `MouseWheelEvent` наследует все методы `MouseEvent`  
(`getX()`, `getY()`, `getButton()`, `getClickCount()`, `isPopupTrigger()` и др.),  

и добавляет свои, относящиеся к колесу.

### Направление и величина прокрутки
```java
// Возвращает количество щелчков колеса
// Отрицательное значение — прокрутка вверх
// Положительное — вниз
public int getWheelRotation()

// Возвращает "точное" значение вращения (double), может быть дробным
// Используется для более плавной прокрутки (например, в трекпадах)
public double getPreciseWheelRotation()
```

### Тип и масштаб прокрутки
```java
// Возвращает тип прокрутки:
// WHEEL_UNIT_SCROLL — построчная,
// WHEEL_BLOCK_SCROLL — блочная (например, постраничная)
public int getScrollType()

// Возвращает количество единиц для прокрутки за один щелчок
public int getScrollAmount()

// Возвращает количество единиц (строк), которые нужно прокрутить
// Вычисляется как getScrollAmount() * getWheelRotation()
public int getUnitsToScroll()
```

### Отладочная информация
```java
// Возвращает строковое представление события
public String paramString()
```

## 💡 Особенности и подводные камни 💡

1️⃣ 💡**Знак `wheelRotation`:**
- Отрицательное значение — прокрутка **вверх** (к началу),
    
- Положительное — **вниз** (к концу).  
    Это легко перепутать, особенно при ручной обработке.
    

2️⃣ 💡**Тип `scrollType` определяет поведение:**
- `WHEEL_UNIT_SCROLL` — прокрутка контента по строкам (например, текст),
- `WHEEL_BLOCK_SCROLL` — постранично (например, большие панели).


3️⃣ **`getPreciseWheelRotation()` даёт более точные значения**,  
особенно при работе с современными трекпадами или плавным колесом.

4️⃣ 💡**Некоторые системы генерируют дробные вращения** (0.5, 1.2 и т. д.),  
поэтому при использовании `getPreciseWheelRotation()` не стоит округлять слишком агрессивно.

5️⃣ **Используй `consume()`**, если не хочешь, чтобы событие распространялось дальше  
(например, при кастомной прокрутке панели).

6️⃣ **Колесо работает только при активном фокусе окна.**  
Компоненты без фокуса не получают `MouseWheelEvent`.

---
## 💎 Best Practices 💎

✅ 💡**Используй** `getPreciseWheelRotation()` для плавной прокрутки, особенно в Swing и UI с touchpad.

✅ Учитывай `scrollType`:
```java
if (e.getScrollType() == MouseWheelEvent.WHEEL_UNIT_SCROLL) {
    scroll(e.getUnitsToScroll());
}
```

✅ Используй `consume()` в кастомных компонентах, чтобы событие не дублировалось в родительских контейнерах.

✅ Не обрабатывай `MouseWheelEvent` через `MouseListener` —  
он имеет **отдельный интерфейс** `MouseWheelListener`.

✅ 💡Для инверсии направления прокрутки не меняй `wheelRotation` напрямую —  используй настройку пользовательского интерфейса (иначе логика будет ломаться при смене ОС).

## Пример
```java
import java.awt.*;
import java.awt.event.*;

public class MouseWheelEventDemo extends Frame implements MouseWheelListener {
    private int fontSize = 16;
    private Label label;

    public MouseWheelEventDemo() {
        setLayout(new FlowLayout());
        label = new Label("Прокрути колесо для изменения размера шрифта");
        label.setFont(new Font("Arial", Font.PLAIN, fontSize));

        add(label);
        addMouseWheelListener(this);

        setSize(400, 200);
        setVisible(true);

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
    }

    @Override
    public void mouseWheelMoved(MouseWheelEvent e) {
        int rotation = e.getWheelRotation(); // +1 вниз, -1 вверх
        fontSize -= rotation; // уменьшаем/увеличиваем шрифт
        fontSize = Math.max(8, Math.min(48, fontSize)); // ограничим диапазон

        label.setFont(new Font("Arial", Font.PLAIN, fontSize));
        label.setText("Размер шрифта: " + fontSize);
    }

    public static void main(String[] args) {
        new MouseWheelEventDemo();
    }
}
```
