
# 📕1. Основные принципы создания окон в AWT

1. **Окна верхнего уровня** создаются ==расширением класса== `Frame`, который предоставляет стандартные функции управления окном (сворачивание, разворачивание, закрытие, изменение размеров).
    
2. **Вывод в окно** осуществляется через переопределение метода `paint()`, который автоматически вызывается исполняющей средой при первом отображении окна и после его повторного показа.
    
3. **Для обновления вывода** используется метод `repaint()`, который сообщает инфраструктуре AWT о необходимости вызова `paint()` (напрямую `paint()` не вызывается).
    
4. **Завершение программы** должно обрабатываться явно - при закрытии окна Frame необходимо вызвать `System.exit()`, поскольку щелчок на кнопке закрытия сам по себе не завершает программу.

# 📕2. Использование модели делегирования

Использовать модель делегирования обработки событий на самом деле довольно легко. 
Понадобится просто выполнить два описанных ниже шага. 
1. Реализовать подходящий интерфейс в прослушивателе, чтобы он мог принимать события желаемого типа. 
2. Реализовать код для регистрации и отмены регистрации (при необходимости) прослушивателя как получателя уведомлений о событиях.

---

Существуют компоненты(источники), способные генерировать события.

Эти события принимают и обрабатывают СЛУШАТЕЛИ.

Kor да происходит событие, источник события вызывает надлежащий метод, который определен в классе прослушивателя, и предоставляет объект события в качестве аргумента.

Для того, чтобы создать слушатель, необходимо РЕАЛИЗОВАТЬ ИНТЕРФЕЙС, связанный с необходимым событием. 
Реализовав методы интерфейса, соответствующие событие будет обрабатываться именно там

```java
public class MyMain extends Frame implements ActionListener     {
	private TextField tf;
	
	public MyMain() {
		setLayout(new FlowLayout());
        tf = new TextField(20);
        Button b1 = new Button("OK");
        Button b2 = new Button("Clear");

        b1.addActionListener(this);
        b2.addActionListener(this);

        add(tf);
        add(b1);
        add(b2);

        setSize(250, 100);
        setVisible(true);

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
	}
	
	// ⭐ПРИ ВОЗНИКНОВЕНИИ СОБЫТИЯ, выолняется данный метод⭐
    @Override
    public void actionPerformed(ActionEvent e) {
        String cmd = e.getActionCommand();
        System.out.println("In handler");
        if ("OK".equals(cmd)) {
            tf.setText("Button OK pressed");
        } else if ("Clear".equals(cmd)) {
            tf.setText("");
        }
    }

    
	public static void main(String[] args){
		MyMain m = new MyMain();
		
		// ⭐Вручную создали событие и передали обработчикау
		ActionEvent a = new ActionEvent(m, 1001, "OK");
		m.actionPerformed(a);
	}
}
```
