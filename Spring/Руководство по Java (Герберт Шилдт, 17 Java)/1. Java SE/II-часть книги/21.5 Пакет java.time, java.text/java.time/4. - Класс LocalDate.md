
> [!NOTE] Представляет дату без времени и часового пояса — `год, месяц и день`.
> Иммутабельный, потокобезопасный. 
> Реализует: `Temporal`, `TemporalAdjuster`, `ChronoLocalDate`, `Serializable`.

## Фабричные методы:
```java
// Текущая дата по системным часам
public static LocalDate now();

// Текущая дата в заданной временной зоне
public static LocalDate now(ZoneId zone);

// Создание по компонентам
public static LocalDate of(int year, int month, int dayOfMonth);

// Создание из объекта TemporalAccessor (например, Instant)
public static LocalDate from(TemporalAccessor temporal);
```

---
## 💡 Особенности и подводные камни 💡
1️⃣ Не содержит времени суток или зоны.  

2️⃣ Все методы возвращают **новый объект** (иммутабельность).  

3️⃣ Учитывает високосные годы.  

4️⃣ Потокобезопасен.  

5️⃣ Не путать с `Instant` — это относительное представление.

---

## 💎 Best Practices 💎
✅ Используй `LocalDate` для календарных расчётов (например, «через 3 дня»).  

✅ Не для хранения UTC — для этого `Instant`.  

✅ Для БД можно сохранять как `DATE`.

## Пример
```java
LocalDate today = LocalDate.now();
LocalDate nextWeek = today.plusWeeks(1);
System.out.println(today + " → " + nextWeek);
```
