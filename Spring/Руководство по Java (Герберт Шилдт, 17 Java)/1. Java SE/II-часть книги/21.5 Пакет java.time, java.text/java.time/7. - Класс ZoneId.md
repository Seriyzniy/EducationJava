
> [!NOTE] Определяет `идентификатор часового пояса`, например `"Europe/Moscow"`.
> 

##  Фабричные методы:
```java
public static ZoneId of(String zoneId);
public static ZoneId systemDefault();
public static Set<String> getAvailableZoneIds();
```

## 💡 Особенности
1️⃣ Хранит полное имя зоны, а не просто смещение.  
2️⃣ ⭐⭐==Идентификаторы берутся из базы IANA== (например, `"Europe/London"`).

## Пример
```java
ZoneId zone = ZoneId.of("Europe/Moscow");
System.out.println(ZoneId.systemDefault());
```
