# 📕 Модификаторы доступа в Java
| Модификатор                                   | Доступен в том же классе | В том же пакете | В подклассе (другой пакет) | Вне пакета |
| --------------------------------------------- | ------------------------ | --------------- | -------------------------- | ---------- |
| `private`                                     | ✅                        | ❌               | ❌                          | ❌          |
| _(`package-private`)_  <br>(без модификатора) | ✅                        | ✅               | ❌                          | ❌          |
| `protected`                                   | ✅                        | ✅               | ✅                          | ❌          |
| `public`                                      | ✅                        | ✅               | ✅                          | ✅          |
## 🔸 `private`
- **Самый ограниченный** модификатор.
- ==Доступен **только внутри того же класса**.==
- Недоступен даже в других классах **в том же пакете**.
- Часто используется для **инкапсуляции состояния**.
```java
class Example {
    private int value;

    private void secretMethod() {
        System.out.println("Can't touch this");
    }
}
```
***
## 🔸 _(package-private)_ (по умолчанию)
- Отсутствие модификатора.
- Элемент доступен только **внутри пакета**.
- Часто используется для **вспомогательных классов**, к которым не нужен внешний доступ.
```java
class Helper {
	int x = 5; // доступно внутри пакета
	
    void assist() { // доступно внутри пакета
        
    }
}
```
***
## 🔸 `protected`
- Доступен:
    - ==в **текущем классе**==
    - ==в **пакете**==
    - ==и в **наследниках**==, даже из других пакетов

- Используется для **предоставления ограниченного доступа в наследуемых классах**.
```java
class Parent {
    protected void show() {
        System.out.println("Protected method");
    }
}

//Другой пакет
class Child extends Parent {
    void access() {
        show();  // работает даже в другом пакете
    }
}
```
***
## 🔸 `public`
- Полный доступ **откуда угодно**.
- Применяется к:
    - классам (но только **один public-класс на файл**)
    - методам, полям, конструкторам
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Доступен везде");
    }
}
```

***
## 💡Применение модификаторов

|Элемент|Возможные модификаторы|
|---|---|
|**Классы**|`public`, _(package-private)_|
|**Методы**|Все четыре|
|**Поля**|Все четыре|
|**Конструкторы**|Все четыре|
|**Вложенные классы**|Все четыре|
