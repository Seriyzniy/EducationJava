Для удобства проект создадим на основе `spring-boot-starter`, с помощью которого сервер `Tomcat` автоматически развернется с нашим приложением

Для создания проекта мы использовали `Spring Initializr` (веб сервис), который создал необходимую файловую структуру на основе `Maven`, а также внедрил необходимые нам зависимости:
- Sring DevTools
- Spring Web
- Thymeleaf
