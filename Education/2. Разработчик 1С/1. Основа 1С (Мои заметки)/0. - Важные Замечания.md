# 1. Имя и синоним
**Имя** - название объекта/переменной для разработчика. Именуется согласно правилам именований переменных в программировании

**Синоним** - название того же объекта, но которое отображается пользователю

---
# 2. Поле Код не изменяеть
Есть поле Код, которое пользователь видит при заполнении данных

Так вот, его НЕЛЬЗЯ заполнять, он заполняется сам автоматически (автоинкрементное значение)

---
# 3. Доступ к переменным через &
Доступ к значению переменной осуществляется через `&`

> Пример 
> `Дата1 = &СтандартПериодПерем.ДатаОкончания`
> 
> Значение в переменную `Дата1` подставится из значение переменной `СнатдартПериодПерем.ДатаОкончания`

---
# 4. Полезный тип переменной СтандартныйПериод 
Переменная типа СтандартныйПериод позволяет пользователю при помощи UI выбрать период даты

Обычно значение из этой переменной подставляются в другие переменные, которые используются в запросе

---
# 5. Изменение формата переменной
Изменить формат вывода переменной в отчете, можно в окне "Набор Данных", во внутренней вкладке "Оформление-> Параметры редактирования"

Два раза нажать, и слева появится `...`. Там можно настроить цвет, формат и прочее. В формате нужно также нажать 2 раза на поле и перейти в `...`. И там через выражения заполнить формат ввода для
- даты
- числа
- булевой переменной
