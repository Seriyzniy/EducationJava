Иногда Kafka может **переотправить уже обработанное сообщение** — например, при _rebalance_.  

Фреймворк не знает, что ты его уже обработал — это ответственность приложения. Это называется **Idempotent Receiver Pattern**.

Spring Kafka помогает в этом с помощью:
# 1.FilteringMessageListenerAdapter
Это обёртка вокруг твоего `MessageListener`, которая позволяет **отфильтровать дубликаты** или ненужные сообщения.

Она использует интерфейс:
### ➤ `RecordFilterStrategy`

Ты реализуешь метод `filter()`, который должен вернуть:
- **true** — если сообщение нужно _отфильтровать_ (пропустить)
- **false** — если сообщение нужно обработать
    

Также есть свойство: `ackDiscarded`
Если true → Kafka-коммит будет сделан **даже для отбракованного** сообщения.

---
# 2.Использование с @KafkaListener
Чтобы включить фильтрацию:
- стратегию фильтра задаёшь в **container factory**,  
    или
- начиная с **Spring Kafka 2.8.4**, можно прямо в аннотации:
```java
@KafkaListener(
	topics = "topic", 
	filter = "myFilter")
public void listen(Thing thing) { }
```

---
# 3.Фильтрация батчей
Если слушатель принимает `List<ConsumerRecord>`, используется:
- **FilteringBatchMessageListenerAdapter**

❗❗Если слушатель принимает `ConsumerRecords<?, ?>`, то фильтрация **не применяется**, потому что объект immutable.

---
# 4. **Новое с версии 3.3 — ignoreEmptyBatch()**

Фильтр может полностью очистить batch (вернуть пустой список).  

Что делать дальше — решает метод: `ignoreEmptyBatch()``
- **true** → если batch пустой, метод @KafkaListener **НЕ вызывается**, но коммит offset’ов делается
    
- **false** → метод @KafkaListener вызывается **даже если batch пуст**

---
# Пример
### Фильтр
Фильтр: пропустить сообщения, у которых key = "system"
```java
@Component("filterByKey")
public class FilterByKey 
implements RecordFilterStrategy<String, String> 
{
    @Override
    public boolean filter(ConsumerRecord<String, String> record) {
        return "system".equals(record.key());
    }
}
```

### Listener
```java
@KafkaListener(topics = "events", filter = "filterByKey")
public void listen(String data) {
    System.out.println("Accepted: " + data);
}
```
