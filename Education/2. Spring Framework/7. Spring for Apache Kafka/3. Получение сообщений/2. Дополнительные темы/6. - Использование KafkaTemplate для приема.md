Этот раздел рассказывает о том, как использовать `KafkaTemplate` для получения сообщений.

Начиная с версии 2.8, шаблон имеет четыре метода `receive()`:
```java
ConsumerRecord<K, V> receive(
	String topic, 
	int partition, 
	long offset);

ConsumerRecord<K, V> receive(
	String topic, 
	int partition, 
	long offset, 
	Duration pollTimeout);

ConsumerRecords<K, V> receive(
	Collection<TopicPartitionOffset> requested);

ConsumerRecords<K, V> receive(
	Collection<TopicPartitionOffset> requested, 
	Duration pollTimeout);
```

Как вы можете видеть, вам необходимо знать **раздел (partition)** и **смещение (offset)** записи(ей), которые нужно получить; для каждой операции создается новый `Consumer` (и затем закрывается).

С последними двумя методами каждая запись извлекается индивидуально, и результаты собираются в объект `ConsumerRecords`. При создании `TopicPartitionOffsets` для запроса поддерживаются только **положительные, абсолютные смещения**.
