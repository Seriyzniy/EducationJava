Контейнеры слушателей 
(KafkaMessageListenerContainer/ ConcurrentMessageListenerContainer) 

могут обрабатывать **события перераспределения партиций** (rebalance).
- **Свойство `consumerRebalanceListener`** в ContainerProperties *принимает реализацию интерфейса* Kafka `ConsumerRebalanceListener`.
    
- *Если слушатель не указан, контейнер использует* **логирующий слушатель**, который пишет события перераспределения на уровень INFO

## Интерфейс ConsumerAwareRebalanceListener
```java
public interface ConsumerAwareRebalanceListener extends ConsumerRebalanceListener {
    void onPartitionsRevokedBeforeCommit(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions);
    void onPartitionsRevokedAfterCommit(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions);
    void onPartitionsAssigned(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions);
    void onPartitionsLost(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions);
}
```

### Особенности:
1. **onPartitionsRevokedBeforeCommit** – вызывается сразу при отзыве партиций, полезно для ручного подтверждения сообщений (`Acknowledgment`).
    
2. **onPartitionsRevokedAfterCommit** – вызывается после коммита любых ожидающих смещений, удобно для хранения смещений во внешнем репозитории.
    
3. **onPartitionsAssigned** – вызывается после назначения партиций; можно выполнить `consumer.seek()` для восстановления смещений.
    
4. **onPartitionsLost (с версии 2.4)** – вызывается при потере партиций; по умолчанию ничего не делает. **Важно:** не делегируйте вызов `onPartitionsRevoked` внутри `onPartitionsLost`, иначе метод будет вызван дважды.

---
# Пример
```java
containerProperties.setConsumerRebalanceListener(new ConsumerAwareRebalanceListener() {
    @Override
    public void onPartitionsRevokedBeforeCommit(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions) {
        // подтверждаем все ожидающие ack
    }

    @Override
    public void onPartitionsRevokedAfterCommit(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions
	) {
        // сохраняем смещения
        store(consumer.position(partition));
    }

    @Override
    public void onPartitionsAssigned(
	    Consumer<?, ?> consumer, 
	    Collection<TopicPartition> partitions
	) {
        // восстанавливаем смещения
        consumer.seek(partition, offsetTracker.getOffset() + 1);
    }
});
```
