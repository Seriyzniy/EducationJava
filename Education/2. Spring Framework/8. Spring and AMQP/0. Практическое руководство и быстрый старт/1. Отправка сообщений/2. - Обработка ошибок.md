Ошибки возможны на трех этапах:
### 1 **Ошибки конвертации сообщений**
Происходят ДО отправки, при создании `Message`:
- Ошибки в `MessageConverter` (Jackson, etc.)
- NPE в теле сообщения
- Ошибки сериализации объектов
- Проблемы с кодировкой

### 2 **Ошибки соединения и сети**
Происходят при попытке установить соединение:
- RabbitMQ недоступен
- Ошибки аутентификации
- Проблемы с виртуальным хостом 
- Network timeout

### 3 **Ошибки доставки (после отправки)**
Происходят ПОСЛЕ вызова `send()`:
- Exchange не существует
    
- Routing key не найден (при `mandatory=true`)
    
- Queue полна или недоступна
    
- TTL истек
- Ошибки подтверждений (Publisher Confirms)

## Конвертация ошибок в RabbitTemplate и Container
Многие операции Java-клиента RabbitMQ выбрасывают **checked-исключения** (чаще всего `IOException`). 

Компоненты Spring AMQP 
- `RabbitTemplate`, 
- `SimpleMessageListenerContainer` и др.
**перехватывают их и ПРЕОБРАЗУЮТ** в исключения иерархии `AmqpException` (`org.springframework.amqp`).

Если исключение выбрасывается **внутри listener’а**, оно оборачивается в `ListenerExecutionFailedException`. По умолчанию сообщение **rejected и requeued** (сообщение отклоняется и ставится в очередь)

# 1. Способы обработки ошибок
## **Способ 1 — Callback (`ConfirmCallback` и `ReturnsCallback`)**

Для **обработки** 
- ***подтверждений*** 
- и ***возвращенных сообщений***.
### **ConfirmCallback (Publisher Confirms)**
```java
@Bean
public RabbitTemplate rabbitTemplate(ConnectionFactory connectionFactory) {
    RabbitTemplate template = new RabbitTemplate(connectionFactory);
    
    // Включение confirms
    CachingConnectionFactory ccf = 
	    (CachingConnectionFactory) connectionFactory;
	    
    ccf.setPublisherConfirmType(
	    CachingConnectionFactory.ConfirmType.CORRELATED
	);
    
    // Обработка подтверждений
    template.setConfirmCallback((correlationData, ack, cause) -> {
        if (ack) {
            log.info("Message confirmed: {}", 
                correlationData != null 
	                ? correlationData.getId() 
	                : "unknown");
        } else {
            log.error("Message NOT confirmed. ID: {}, Cause: {}", 
                correlationData != null 
	                ? correlationData.getId() 
	                : "unknown", 
                cause);
                
            // Повторная отправка или сохранение в локальную очередь
            handleFailedMessage(correlationData, cause);
        }
    });
    
    return template;
}
```

### **ReturnsCallback (для mandatory сообщений)**
```java
@Bean
public RabbitTemplate rabbitTemplate(ConnectionFactory connectionFactory) {
    RabbitTemplate template = new RabbitTemplate(connectionFactory);
    
    // Mandatory = true - немаршрутизируемые сообщения вернуться producer
    template.setMandatory(true);
    
    // Обработка возвращенных сообщений
    template.setReturnsCallback(returned -> {
        log.error("""
            Message returned!
            Reply Code: {}
            Reply Text: {}
            Exchange: {}
            Routing Key: {}
            Message Body: {}
            """,
            returned.getReplyCode(),
            returned.getReplyText(),
            returned.getExchange(),
            returned.getRoutingKey(),
            new String(returned.getMessage().getBody())
        );
        
        // Сохранить в Dead Letter или обработать иначе
        handleReturnedMessage(returned);
    });
    
    return template;
}
```

## **Способ 2 — RetryTemplate (повторные попытки)**
Для автоматических **повторных попыток** *при сетевых ошибках.*

```java
@Bean
public RabbitTemplate resilientRabbitTemplate(
        ConnectionFactory connectionFactory,
        RetryTemplate retryTemplate) {
    
    RabbitTemplate template = new RabbitTemplate(connectionFactory);
    
    // Настройка RetryTemplate
    template.setRetryTemplate(retryTemplate);
    
    // Recovery callback
    template.setRecoveryCallback(context -> {
        log.error(
	        "All retry attempts failed", context.getLastThrowable());
	        
        // Сохранить сообщение для ручной обработки
        Message failedMessage = (Message) context.getAttribute("message");
        saveForManualRetry(failedMessage);
        return null;
    });
    
    return template;
}

@Bean
public RetryTemplate retryTemplate() {
    RetryTemplate retryTemplate = new RetryTemplate();
    
    // Политика повторов
    SimpleRetryPolicy retryPolicy = new SimpleRetryPolicy();
    retryPolicy.setMaxAttempts(3);
    retryTemplate.setRetryPolicy(retryPolicy);
    
    // Стратегия задержки
    ExponentialBackOffPolicy backOffPolicy = 
	    new ExponentialBackOffPolicy();
	    
    backOffPolicy.setInitialInterval(1000); // 1 секунда
    backOffPolicy.setMultiplier(2.0); // увеличиваем в 2 раза каждый раз
    backOffPolicy.setMaxInterval(10000); // максимум 10 секунд
    retryTemplate.setBackOffPolicy(backOffPolicy);
    
    // Какие исключения ретраить
    Map<Class<? extends Throwable>, Boolean> retryableExceptions = 
	    new HashMap<>();
    retryableExceptions.put(AmqpConnectException.class, true);
    retryableExceptions.put(AmqpIOException.class, true);
    
    // Добавление мапы с исплюкниями
    retryPolicy.setRetryableExceptionMap(retryableExceptions);
    
    return retryTemplate;
}
```

## **Способ 3 — ErrorHandler в ConnectionFactory**
Глобальная обработка **ошибок соединения.**

```java
@Bean
public ConnectionFactory connectionFactory() {
    CachingConnectionFactory factory = new CachingConnectionFactory("localhost");
    
    // Настройка ErrorHandler для соединения
    factory.addConnectionListener(new ConnectionListener() {
        @Override
        public void onCreate(Connection connection) {
            log.info("Connection created: {}", connection);
        }
        
        @Override
        public void onClose(Connection connection) {
            log.warn("Connection closed: {}", connection);
        }
        
        @Override
        public void onShutDown(ShutdownSignalException signal) {
            log.error("Connection shutdown: {}", signal.getMessage());
            // Можно запустить процесс восстановления
            startConnectionRecovery();
        }
    });
    
    // Настройка блокировки соединения (flow control)
    factory.addBlockedListener(new BlockedListener() {
        @Override
        public void handleBlocked(String reason) {
            log.warn("Connection blocked: {}", reason);
            // Приостановить отправку сообщений
            pauseMessageSending();
        }
        
        @Override
        public void handleUnblocked() {
            log.info("Connection unblocked");
            // Возобновить отправку
            resumeMessageSending();
        }
    });
    
    return factory;
}
```

## **Способ 4 — Custom MessageConverter с обработкой ошибок**

Для обработки о**шибок сериализации.**
```java
@Bean
public MessageConverter errorHandlingMessageConverter() {

    return new MessageConverter() {
    
        private final Jackson2JsonMessageConverter delegate = 
            new Jackson2JsonMessageConverter();
        
        @Override
        public Message toMessage(
	        Object object, 
	        MessageProperties messageProperties) 
                throws MessageConversionException 
        {
            try {
                return delegate.toMessage(object, messageProperties);
                
            } catch (MessageConversionException e) {
                log.error(
	                "Failed to convert message: {}", e.getMessage());
                
                // Создать сообщение об ошибке вместо оригинального
                ErrorMessage errorMessage = new ErrorMessage(
                    "Serialization failed", 
                    e.getMessage(), 
                    object.toString()
                );
                
                return delegate.toMessage(
	                errorMessage, 
	                messageProperties
	            );
            }
        }
        
        @Override
        public Object fromMessage(Message message) throws MessageConversionException {
            try {
                return delegate.fromMessage(message);
                
            } catch (MessageConversionException e) {
                log.error(
	                "Failed to deserialize message: {}", e.getMessage());
                throw e;
            }
        }
    };
}

// Использование
@Bean
public RabbitTemplate rabbitTemplate(
        ConnectionFactory connectionFactory,
        MessageConverter messageConverter) 
{
    RabbitTemplate template = new RabbitTemplate(connectionFactory);
    
    template.setMessageConverter(messageConverter);
    
    return template;
}
```
