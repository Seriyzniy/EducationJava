## Для чего DTO
Модель используется для взаимодействия В БИЗНЕС ЛОГИКЕ (на свервере)
DTO используется для взаимодействия КЛИЕНТА и СЕРВЕРА

> [!NOTE] Модель и DTO
> В процессе обработки данных мы можем не захотеть некоторые данные (Модель) передавать пользователю, и для этого формируем объект DTO из Модели

> [!NOTE] DTO
> Обычно классы DTO помещаются в отдельный пакет с названием DTO
