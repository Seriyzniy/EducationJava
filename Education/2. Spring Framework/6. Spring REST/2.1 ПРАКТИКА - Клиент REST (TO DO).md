### 1. Получение JSON данных с сервиса
Создадим простой Java класс
```java
public class ClientRest {  
    public static void main(String[] args) {  
        RestTemplate restTemplate = new RestTemplate();  
        String url = "https://reqres.in/api/users/2";  
          
        String responce = restTemplate.getForObject(url, String.class);  
        System.out.println(responce);  
    }  
}
```

Здесь создаем объект класса `RestTemplate` и вызываем метод `getForObject()`, который отправит GET-запрос (по первому слову метода) по указанному адресу, и сохранить данные в строку (String.class)

# 2. Обращение к сервису TravelPayOuts
Создадим приложение которое будет получать информацию о путешествиях для своего сайта или блога, тенденции цен на авиабилеты и находите популярные направления для своих клиентов

Инструкция:
[about-api – Справочник по API](https://travelpayouts.github.io/slate/)

Для этого необходимо зарегаться на сайте, после чего будет выдан Токен

Вот пример запроса который должен отправляться
``` bash
curl --request GET \
  --url 'https://api.travelpayouts.com/v1/prices/cheap?origin=MOW&destination=HKT&depart_date=2019-11&return_date=2019-12' \
  --header 'x-access-token: 321d6a221f8926b5ec41ae89a3b2ae7b'
```

Рассмотрим параметры:
- `origin` - IATA код города отправления
- `destination` - IATA-код города назначения (для выбора всех городов ввести `-`)
- `depart_date` - День или месяц отъезда (гггг-мм-дд или гггг-мм).
- `return_date` - День или месяц возврата (гггг-мм-дд или гггг-мм).
- `page` - Необязательный параметр, используется для отображения найденных данных (по умолчанию на странице отображается 100 найденных цен. Если **место назначения** не выбрано, может быть больше данных. В этом случае используйте **страницу** для отображения следующего набора данных, например, page=2).
- `token` - Индивидуальный партнерский токен.
- `currency` - Валюта

В результате должен получиться такой запрос (летим из Ставрополя в Москву)
```bash
https://api.travelpayouts.com/v1/prices/cheap?origin=STW&destination=MOW&depart_date=2024-12&return_date=2025-01&token=570c3b035ac7b52faf3d292b43cbbe51
```

В Java-коде просто собираем URL адрес

```java
public class ClientRest {
```

# 3. Преобразование JSON файла с помощью Jackson

### 1. ObjectMapper
Чтобы получить данные из JSON так, чтобы их можно было удобно считывать/обрабатывать, необходимо эти данные преобразовать

Для этого можно:
- Создать новый класс с полями соответствующими ключам JSON файла
- Считать необходимые данные сразу при помощи `ObjectMapper`

```java
ObjectMapper objectMapper = new ObjectMapper();  
JsonNode node = objectMapper.readTree(result);  
  
System.out.println(node.get("data").get("ABA").get("2").get("airline"));
```

В результате получим авиалинию ABA

### 2. Создание класса по JSON файлу
Обычно при создании сервисов, формат JSON файла известен заранее, поэтому можно создать класс с полями соответствующим названиям ключей в JSON

Поскольку JSON файл представляет собой такую структуру
```json
{
"data":{
	"ABA":{
		"2":{
			"airline":"DP",
			"departure_at":"2024-12-26T12:15:00+03:00",
			"return_at":"2025-01-12T20:25:00+07:00",
			"expires_at":"2024-12-23T09:03:50Z",
			"price":56574,
			"flight_number":6920
		}
	},
	"AER":{
		"1":{
			"airline":"DP",
			"departure_at":"2024-12-26T12:15:00+03:00",
			"return_at":"2024-12-30T07:50:00+03:00",
			"expires_at":"2024-12-23T09:03:50Z",
			"price":20379,
			"flight_number":6920
		}
	}
}
}
```

> [!NOTE] Как правильно определить структуру?
> - Внимательно смотри на вложенность
> - Каждая пара `{ }` это объект
> - Если в `{ }` идет структура вида `ключ: {}` то это Map (ассоциативный массив)

Теперь определим структуру классов
Начальные `{ }` говорят о том, что ==нужен корневой класс==, в котором будет лежать переменная/объект
- Поскольку работаем с авиабилетами, пусть корневым классом будет класс `Airlines`
```json
{//Мы тут
	"data": {
		. . .
	}
}
```

В этой переменной `data` находится структура `ключ: значение`, ==ПРИЧЕМ значением выступает другой класс==. 
Значит переменная `data` - это ассоциативный массив `строка, объект_класса` 
- Классом, отвечающим за IATA-коды городов будет `myIATACity`
```json
{
"data":{    //Мы тут
	"ABA":{
		. . .
	},
	"AER":{
		. . .
	}
}
}
```

Внутри вложенного класса `myIATACity`, находится такая же структура `ключ: значение`, ==значением также является другой класс==, значит используем также ассоциативный массив Map
- Классом отвечающим за билеты будет `myTickets`
```json
{
"data":{
	"ABA":{ // Мы тут
		"2":{
			
		}
	},
	"AER":{ // Мы тут
		"1":{
			
		}
	}
}
}
```

Нам понадобятся несколько НОВЫХ классов
- `Airlines` - корневой класс, содержащий поле `data`
- `myIATACity` - класс для маппинга `номерБилета = myTicket`
- `myTicket` - класс, содержащий данные о билете

Класс `myTicket`

```java
package myModelsJSON;  
  
import com.fasterxml.jackson.annotation.JsonProperty;  
  
public class myTicket {  
    @JsonProperty("airline")

    @JsonProperty("departure_at")

    @JsonProperty("return_at")  
    private String returnAt;  
  
    @JsonProperty("expires_at")  
    private String expiresAt;  
  
    @JsonProperty("price")  
    private int price;  
  
    @JsonProperty("flight_number")  
    private int flightNumber;  
  
    // Getters and Setters  
    public String getAirline() {  
        return airline;  
    }  
  
    public void setAirline(String airline) {  
        this.airline = airline;  
    }  
  
    public String getDepartureAt() {  
        return departureAt;  
    }  
  
    public void setDepartureAt(String departureAt) {  
        this.departureAt = departureAt;  
    }  
  
    public String getReturnAt() {  
        return returnAt;  
    }  
  
    public void setReturnAt(String returnAt) {  
        this.returnAt = returnAt;  
    }  
  
    public String getExpiresAt() {  
        return expiresAt;  
    }  

    public void setExpiresAt(String expiresAt) {  
        this.expiresAt = expiresAt;  
    }  
  
    public int getPrice() {  
        return price;  
    }  
  
    public void setPrice(int price) {  
        this.price = price;  
    }  
  
    public int getFlightNumber() {  
        return flightNumber;  
    }  

    public void setFlightNumber(int flightNumber) {  
        this.flightNumber = flightNumber;  
    }  
}
```

Класс myIATACity
```java
package myModelsJSON;  
  
import com.fasterxml.jackson.annotation.JsonAnySetter;  
  
import java.util.HashMap;  
import java.util.Map;  
  
public class myIATACity {  
    private Map<String, myTicket> tickets = new HashMap<>();  
  
    public Map<String, myTicket> getTickets() {  
        return tickets;  
    }  
    @JsonAnySetter  
    public void setTickets(String key, myTicket ticket) {  
        this.tickets.put(key,ticket);  
    }  
}
```

> [!NOTE] ВСТАВКА ЗНАЧЕНИЙ В `Map`
> Не нужно передавать в качестве параметра set метода целую структуру, ==необходимо передавать ключи и значения==, иначе Jackson не поймет что делать, т.к обрабатывает JSON поэлементно 

> [!NOTE] Аннотация `@JsonAnySetter`
> `@JsonAnySetter` в библиотеке Jackson **позволяет определить логический «любой сеттер» с помощью нестатического двухаргументного метода**.

Класс `Airlines`

Функция main
```java
public class ClientRest {  
    public static void main(String[] args) throws JsonProcessingException {  
        RestTemplate restTemplate = new RestTemplate();  
        String url = "https://api.travelpayouts.com/v1/prices/cheap?";  
  
        String origin = "STW";  
        String destination = "MOW";  
        String depart_date = "2024-12-26";  
        String return_date = "2025-01";  
        String token = "570c3b035ac7b52faf3d292b43cbbe51";  
  
        String resultUrl = url + "origin=" + origin + "&depart_date=" + depart_date + "&return_date" + return_date +  
                "&token=" + token;  
  
        Airlines result = restTemplate.getForObject(resultUrl, Airlines.class);  
		
		// Перебираем города (IATA-коды)
        for(Map.Entry<String, myIATACity> iataCities : result.getData().entrySet()){  
            String iataCode = iataCities.getKey();  
            myIATACity iataCity = iataCities.getValue();  
  
            System.out.println("\nIATA-код: " + iataCode);  
  
            // Получаем билеты  
            Map<String, myTicket> tickets = iataCity.getTickets();  
  
            for (Map.Entry<String, myTicket> ticketEntry : tickets.entrySet()) {  
                String ticketKey = ticketEntry.getKey();  
                myTicket ticket = ticketEntry.getValue();  
  
                System.out.println("Билет ID: " + ticketKey);  
                System.out.println("Авиакомпания: " + ticket.getAirline());  
                System.out.println("Цена: " + ticket.getPrice());  
                System.out.println("Дата вылета: " + ticket.getDepartureAt());  
            }  
        }  
    }  
}
```
