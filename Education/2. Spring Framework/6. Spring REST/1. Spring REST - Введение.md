Spring приложение НЕ РЕАЛИЗУЕТ ПРЕДСТАВЛЕНИЯ, а всего лишь ОТПРАВЛЯЕТ ДАННЫЕ (работает только с БЭКЕНДОМ)

***
> [!NOTE] `REST приложения`
> Такие приложения, которые ОТДАЮТ ДАННЫЕ называются REST API (синонимы RESTful API, сервис, бэкенд, API)
