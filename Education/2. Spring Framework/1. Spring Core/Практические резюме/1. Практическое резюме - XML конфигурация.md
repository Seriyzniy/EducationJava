Описание разработки простого Spring Core приложения на базе Maven

# 1. Создание XML файла с конфигурацией
Для того, чтобы Spring приложение работало, ему необходимо наличие файла .xml с конфигурацией

Для этого необходимо создать ==файл XML== в папке ==src/main/resources== со следующим содержимым:
```xml
<beans  xmlns="http://www.springframework.org/schema/beans"
        xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xmlns:context="http://www.springframework.org/schema/context"
        xsi:schemaLocation="http://www.springframework.org/schema/beans
        http://www.springframework.org/schema/beans/spring-beans.xsd
        http://www.springframework.org/schema/context
        http://www.springframework.org/schema/context/spring-context.xsd">

		<!-- Здесь создаем бины -->
</beans>
```

В XML файле бины создаются следующим кодом ==(см главу 2.1 Тег bean в XML)==
```java
public class MessageService {
	public String getMessage() { 
		return "Messge Service";
	}
}

public class UserService {
	private final MessageService messageService;
	
	public UserService(MessageService msgService) {
		this.messageService = msgService;
	}
	
	public void printMessage() {
		System.out.println("User service " + messageService.getMessage());
	}
}
```

```xml
<bean id="messageService" 
	class="ru.springcore.tasks.springCore.task1.MessageService">
</bean>
	
<bean id="userService" class="ru.springcore.tasks.springCore.task1.UserService">
		<constructor-arg ref="messageService"/>
</bean>
```

# 2. Запуск контекста с XML конфигурацией

==Для запуска приложения ИСПОЛЬЗУЮЩЕГО XML конфигурацию==, необходимо использовать класс `ClassPathXmlApplicationContext`
```java
public class App 
{
    public static void main( String[] args )
    {
        ClassPathXmlApplicationContext context = new ClassPathXmlApplicationContext("applicationContext.xml");
        
        UserService user = context.getBean("userService",UserService.class);
        
        user.printMessage();
    
        context.close();
    }
}
```
