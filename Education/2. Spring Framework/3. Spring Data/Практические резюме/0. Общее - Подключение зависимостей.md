# 1. Зависимости для JDBC API (JdbcTemplate)
### Минимальные зависимости
Для того, чтобы работать с БД через JDBS API необходимо подключить две зависимости

1. Драйвер для БД
```xml
<dependency>
    <groupId>org.postgresql</groupId>
    <artifactId>postgresql</artifactId>
    <version>42.7.4</version>
</dependency>
```

2. Spring JDBC - JdbsTemplate
```xml
<dependency>
	<groupId>org.springframework</groupId>
	<artifactId>spring-jdbc</artifactId>
	<version>${spring.version}</version>
</dependency>
```

### Дополнительные зависимости
 - Jackson-databind
```xml
<dependency>
    <groupId>com.fasterxml.jackson.core</groupId>
    <artifactId>jackson-databind</artifactId>
    <version>2.18.1</version>
</dependency>
```
Для конвертации данных в JSON формат (позволяет конвертировать объекты в Json при помощи аннотации `@ResponseBody`)
