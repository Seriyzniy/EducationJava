Для начала развернем простую учебную программку в Docker.

Допустим у нас уже есть сервер (наш компьютер), с установленным Docker. Теперь зная имя Image (Welcome to docker), скачаем этот образ и запустим контейнер

## Скачивание Image
Скачать нужный Image можно прямиком из Docker.

> [!NOTE] Скачивание и запуск
> `Pull` - означает скачать Image (аналог git)
> `Run` - означает запуск Image. Даже если его не скачали, Docker сам его скачает

![[Pasted image 20251021131431.png]]

После этого, скаченный Image будет во вкладке `Images`, оттуда его можно запустить

## Настройка контейнера (настройка порта)
Перед этим выполним настройки контейнера на основе этого образа
- имя: myFirstContainer (либо другое)
- порт: 8088 (указано в документации к этому Image)
![[Pasted image 20251021132013.png]]

После, нажмем на кнопку Run (в скриншот не поместился)
![[Pasted image 20251021132216.png]]

## Консольный вывод контейнера
Если нажать на имя контейнера, появится консоль, в которой отображается вывод
