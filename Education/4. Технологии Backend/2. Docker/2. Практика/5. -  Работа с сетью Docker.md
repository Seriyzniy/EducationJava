# 📕 1. Создание/изменение/удаление сетей
## Просмотр существующих сетей
```bash
docker network ls
```

![[Pasted image 20251023181943.png]]

## Создание сети bridge
```bash
docker network create mybridgenet
```

![[Pasted image 20251023183526.png]]


## Просмотр настроек
```bash
docker network inspect mybridgenet
```

> [!NOTE] Адрес подсети (Subnet) можно изменить
>

![[Pasted image 20251023183756.png]]

## Создание сети с определенным IP сети
```bash
docker network create --subnet 192.168.10.0/24 --gateway 192.168.10.1 mynet2
```

Проверим настройки созданной сети
```bash
docker network inspect mynet2
```

![[Pasted image 20251023184320.png]]

## Удаление сети
```bash
docekr network rm НазваниеСети
```
