1️⃣ **Создание топика**:
```
/opt/kafka/bin/kafka-topics.sh --create --bootstrap-server localhost:9092 --topic my-topic --partitions 1 --replication-factor 1
```

2️⃣ **Проверка существующих топиков**:
```
/opt/kafka/bin/kafka-topics.sh --list --bootstrap-server localhost:9092
```

3️⃣ **Отправка сообщения в топик** (одно сообщение через echo):
```
/opt/kafka/bin/kafka-console-producer.sh --bootstrap-server localhost:9092 --topic my-topic
```
