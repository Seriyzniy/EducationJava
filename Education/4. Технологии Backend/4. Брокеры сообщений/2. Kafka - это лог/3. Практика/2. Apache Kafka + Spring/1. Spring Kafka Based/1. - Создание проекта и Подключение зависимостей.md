# Ошибки
В зависимостях МОЖЕТ быть указан scope (когда применяется зависимость)
```xml
<!-- https://mvnrepository.com/artifact/org.slf4j/slf4j-simple -->
<dependency>
    <groupId>org.slf4j</groupId>
    <artifactId>slf4j-simple</artifactId>
    <version>2.0.17</version>
    <scope>test</scope>
</dependency>
```

Если указан test - то зависимость работает только при запуске тестов (а нам нужно постоянно)

Поэтому удаляем это, и ПЕРЕСОБИРАЕМ ПРОЕКТ при помощи 
```bash
mvn clean compile
```
