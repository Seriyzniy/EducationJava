# 1. Классы
В состав Kafka Connect входит несколько широко применяемых преобразований данных и маршрутизации:
- **[Cast](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.Cast)** – Преобразует отдельные поля или весь ключ/значение в указанный тип.
    
- **[DropHeaders](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.DropHeaders)** – Удаляет заголовки сообщения по имени.
    
- **[ExtractField](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.ExtractField)** – Извлекает конкретное поле из Struct или Map и оставляет только это поле в результате.
    
- **[Filter](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.Filter)** – Удаляет сообщения из дальнейшей обработки. Используется вместе с [predicate](https://kafka.apache.org/documentation/#connect_predicates) для селективной фильтрации сообщений.
    
- **[Flatten](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.Flatten)** – Преобразует вложенную структуру данных в плоскую структуру.
    
- **[HeaderFrom](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.HeaderFrom)** – Копирует или перемещает поля из ключа или значения в заголовки записи.
    
- **[HoistField](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.HoistField)** – Оборачивает всё событие в одно поле внутри Struct или Map.
    
- **[InsertField](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.InsertField)** – Добавляет поле, используя статические данные или метаданные записи.
    
- **[InsertHeader](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.InsertHeader)** – Добавляет заголовок с использованием статических данных.
    
- **[MaskField](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.MaskField)** – Заменяет значение поля на корректное «нулевое» значение для типа (0, пустая строка и т.д.) или на пользовательское значение (непустая строка или число).
    
- **[RegexRouter](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.RegexRouter)** – Изменяет тему записи на основе исходной темы, строки замены и регулярного выражения.
    
- **[ReplaceField](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.ReplaceField)** – Фильтрует или переименовывает поля.
    
- **[SetSchemaMetadata](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.SetSchemaMetadata)** – Изменяет имя или версию схемы.
    
- **[TimestampConverter](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.TimestampConverter)** – Преобразует временные метки между разными форматами.
    
- **[TimestampRouter](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.TimestampRouter)** – Изменяет тему записи на основе исходной темы и временной метки. Полезно для sink-коннекторов, которые должны писать в разные таблицы или индексы в зависимости от времени.
    
- **[ValueToKey](https://kafka.apache.org/documentation/#org.apache.kafka.connect.transforms.ValueToKey)** – Заменяет ключ записи новым ключом, сформированным из подмножества полей значения записи.

# 2. Свойства 
## bootstrap.servers
**Описание:** Список хост:порт пар Kafka-брокеров, с которыми воркер будет устанавливать первоначальное соединение для обнаружения кластера.  
**Тип данных:** list  
**Значение по умолчанию:** `localhost:9092` [docs.redhat.com+1](https://docs.redhat.com/en/documentation/red_hat_streams_for_apache_kafka/3.0/html/kafka_configuration_properties/kafka-connect-configuration-properties-str?utm_source=chatgpt.com)

---
## group.id
**Описание:** Уникальная строка, идентифицирующая группу воркеров Kafka Connect (для режима распределённого запуска).  
**Тип данных:** string  
**Значение по умолчанию:** отсутствует (обязательно указать) [docs.confluent.io](https://docs.confluent.io/platform/current/installation/configuration/connect/index.html?utm_source=chatgpt.com)

---
## key.converter
**Описание:** Класс конвертера, который отвечает за преобразование ключей сообщений для Kafka Connect (из/в сериализованный формат). Это контролирует формат ключей, которые записываются или читаются из Kafka.  
**Тип данных:** class  
**Значение по умолчанию:** не задано (нужно явно указать) [docs.confluent.io+1](https://docs.confluent.io/platform/current/installation/configuration/connect/index.html?utm_source=chatgpt.com)

---
## value.converter
**Описание:** Класс конвертера для значений сообщений. Контролирует формат записей, отправляемых или читаемых воркером.  
**Тип данных:** class  
**Значение по умолчанию:** не задано (нужно указать) [docs.confluent.io+1](https://docs.confluent.io/platform/current/connect/references/allconfigs.html?utm_source=chatgpt.com)

## config.storage.topic
**Описание:** Имя темы Kafka, в которой хранятся конфигурации коннекторов (для распределённого режима).  
**Тип данных:** string  
**Значение по умолчанию:** отсутствует (обязательно указать) [docs.confluent.io+1](https://docs.confluent.io/platform/current/installation/configuration/connect/index.html?utm_source=chatgpt.com)

**Описание:** Имя темы Kafka, в которой хранится смещение (offset) источников данных для коннекторов.  
**Тип данных:** string  
**Значение по умолчанию:** отсутствует (обязательно указать) [docs.confluent.io](https://docs.confluent.io/platform/current/installation/configuration/connect/index.html?utm_source=chatgpt.com)

**Описание:** Имя темы Kafka, в которой хранится статус коннекторов и задач (tasks) — состояние, ошибки, статистика.  
**Тип данных:** string  
**Значение по умолчанию:** отсутствует (обязательно указать) [docs.confluent.io](https://docs.confluent.io/platform/current/installation/configuration/connect/index.html?utm_source=chatgpt.com)

**Тип данных:** string (разделённые запятыми)
