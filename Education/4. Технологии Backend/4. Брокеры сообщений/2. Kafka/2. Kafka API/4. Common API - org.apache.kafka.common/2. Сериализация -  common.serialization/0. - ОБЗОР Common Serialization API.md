## Основные классы и интерфейсы

| **Имя класса/интерфейса** | **Описание**                                                                                 |
| ------------------------- | -------------------------------------------------------------------------------------------- |
| `Serializer<T>` ✅         | Интерфейс для преобразования объектов Java в массив байтов перед отправкой в Kafka.          |
| `Deserializer<T>`✅        | Интерфейс для преобразования байтов, полученных из Kafka, обратно в объекты Java.            |
| `Serde<T>`✅               | Интерфейс-обёртка, объединяющий `Serializer` и `Deserializer` для определённого типа данных. |
| `Serdes`✅                 | Фабрика для создания стандартных `Serde` (например, для `String`, `Integer`, `Long` и т.д.). |

---

## Примитивные типы (числа, логика)

|**Имя класса/интерфейса**|**Описание**|
|---|---|
|`"BooleanSerializer"`|Сериализатор для `Boolean`. Преобразует `true/false` в байты.|
|`"BooleanDeserializer"`|Десериализатор для `Boolean`. Преобразует байты обратно в логическое значение.|
|`"IntegerSerializer"`|Сериализатор для `Integer`. Преобразует 4 байта в Big Endian формате.|
|`"IntegerDeserializer"`|Десериализатор для `Integer`. Преобразует 4 байта из Big Endian в `int`.|
|`"LongSerializer"`|Сериализатор для `Long`. Преобразует 8 байт в Big Endian.|
|`"LongDeserializer"`|Десериализатор для `Long`. Преобразует 8 байт в `long`.|
|`"ShortSerializer"`|Сериализатор для `Short`. Преобразует 2 байта в Big Endian.|
|`"ShortDeserializer"`|Десериализатор для `Short`. Преобразует 2 байта обратно в `short`.|
|`"FloatSerializer"`|Сериализатор для `Float`. Использует IEEE 754 формат (4 байта).|
|`"FloatDeserializer"`|Десериализатор для `Float`. Преобразует 4 байта обратно в `float`.|
|`"DoubleSerializer"`|Сериализатор для `Double`. Преобразует 8 байт в IEEE 754 формат.|
|`"DoubleDeserializer"`|Десериализатор для `Double`. Преобразует 8 байт обратно в `double`.|

---

## Байтовые типы и буферы

|**Имя класса/интерфейса**|**Описание**|
|---|---|
|`"ByteArraySerializer"`|Сериализатор для `byte[]`. Возвращает байты без изменений.|
|`"ByteArrayDeserializer"`|Десериализатор для `byte[]`. Возвращает массив байт без обработки.|
|`"ByteBufferSerializer"`|Сериализатор для `ByteBuffer`. Перед сериализацией всегда делает `rewind()` позиции буфера.|
|`"ByteBufferDeserializer"`|Десериализатор для `ByteBuffer`. Оборачивает байтовый массив в `ByteBuffer`.|
|`"BytesSerializer"`|Сериализатор для класса `org.apache.kafka.common.utils.Bytes`.|
|`"BytesDeserializer"`|Десериализатор для `Bytes`. Преобразует байты обратно в объект `Bytes`.|

---

## Строки, UUID, Void

|**Имя класса/интерфейса**|**Описание**|
|---|---|
|`"StringSerializer"`|Сериализатор для строк. Кодировка UTF-8 (можно изменить через свойство `serializer.encoding`).|
|`"StringDeserializer"`|Десериализатор для строк. Кодировка UTF-8 (можно изменить через `deserializer.encoding`).|
|`"UUIDSerializer"`|Сериализатор для `UUID`. Сначала преобразует UUID в строку, затем в байты.|
|`"UUIDDeserializer"`|Десериализатор для `UUID`. Сначала преобразует байты в строку, затем в UUID.|
|`"VoidSerializer"`|Сериализатор для `Void` (используется, если ключ/значение не нужны).|
|`"VoidDeserializer"`|Десериализатор для `Void` (всегда возвращает `null`).|

---

## Списки и сложные типы

|**Имя класса/интерфейса**|**Описание**|
|---|---|
|`"ListSerializer<Inner>"`|Сериализатор для списков (`List<T>`), сериализует каждый элемент с помощью заданного внутреннего сериализатора.|
|`"ListDeserializer<Inner>"`|Десериализатор для списков (`List<T>`), десериализует каждый элемент с помощью внутреннего десериализатора.|
