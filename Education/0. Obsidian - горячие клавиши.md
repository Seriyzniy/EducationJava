### **Основное форматирование**

| Действие          | Горячая клавиша (Windows/Linux) |
| ----------------- | ------------------------------- |
| **Жирный текст**  | `Ctrl + B`                      |
| _Курсив_          | `Ctrl + I`                      |
| ~~Зачёркнутый~~   | `Alt + Shift + 5`               |
| `Встроенный код`  | `Ctrl + E`                      |
| **Заголовок 1-6** | `Ctrl + 1/2/3/4/5/6`            |
***
### **Списки и блоки**

| Действие             | Горячая клавиша (Windows/Linux) |
| -------------------- | ------------------------------- |
| Нумерованный список  | `Ctrl + Shift + 7`              |
| Маркированный список | `Ctrl + Shift + 8`              |
| Чекбокс (`- [ ]`)    | `Ctrl + Enter` (в списке)       |
| Блок цитаты (`>`)    | `Ctrl + Shift + >`              |
***
### **Ссылки и вставки**

| Действие                     | Горячая клавиша (Windows/Linux) |
| ---------------------------- | ------------------------------- |
| Гиперссылка (`[текст](url)`) | `Ctrl + K`                      |
| Вставить изображение         | `Ctrl + Shift + I`              |
| Вставить таблицу             | Нет (используйте плагины)       |
***
### **Код и блоки**

| Действие                   | Горячая клавиша (Windows/Linux) |
| -------------------------- | ------------------------------- |
| Блок кода (```)            | `Ctrl + Shift + ```             |
| Математический блок (`$$`) | `Ctrl + Shift + M`              |
***


### **Дополнительно**

| ------------------------ | ------------------------------- |

| Свернуть/развернуть блок | `Ctrl + ↑/↓` (в заголовке)      |
