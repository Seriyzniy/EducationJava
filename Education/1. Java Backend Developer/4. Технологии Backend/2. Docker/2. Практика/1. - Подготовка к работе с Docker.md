Для установки Docker на Windows необходимо скачать Docker Desktop
[Docker Desktop: The #1 Containerization Tool for Developers | Docker](https://www.docker.com/products/docker-desktop/)

Docker Desktop - программа с графическим интерфейсом, которая может работать на Windows.  При этом она работает через WSL 2 (Windows Sybsystem for Linux)

А для Linux нужно установить Docker Engine

## Ограничение потребления памяти Vmmem
**vmmem в Windows 10** — это **процесс, который управляет ресурсами виртуальной памяти**/

Так как докер работает только на Linux, то в Windows запускается виртуальная машина Linux через WSL 2.

Этот процесс может потреблять слишком много системных ресурсов.

Для ограничения ресурсов для этого процесса vmmem, необходимо написать конфигурационный файл `.wslconfig`, и поместить его по пути `C:\Users\YOU\.wslconfig`

и ввести следующие:
```text
[wsl2]
memory=2GB
processors=2      
```

## ❗❗ Перенос всего WSL 2 на другой диск (не проверял) ❗❗
Чтобы докер не занимал много места на системной диске (из-за образов и контейнеров), его можно перенести на другой диск, при помощи перемещения всего WSL 2

## Самый чистый и рекомендуемый вариант — **переместить дистрибутив WSL2**

(так как Docker хранит данные внутри него)

> 🔸 Цель: перенести **всю файловую систему Linux** (и Docker вместе с ней) на другой диск, например `D:`.

### 🔹 Шаги:
1. **Отключи WSL и Docker:**
    
    `wsl --shutdown`
    
1. **Посмотри список установленных дистрибутивов:**
    `wsl --list --verbose`
    
    Например:
    `NAME      STATE           VERSION * Ubuntu   Stopped         2`
    
1. **Экспортируй дистрибутив на диск D:**
    `wsl --export Ubuntu D:\WSL\Ubuntu.tar`
    
2. **Удаляй старую версию (только сам дистрибутив, не Docker Desktop):**
    `wsl --unregister Ubuntu`
    
3. **Импортируй обратно — уже на диск D:**
    `wsl --import Ubuntu D:\WSL\Ubuntu D:\WSL\Ubuntu.tar --version 2`
    
4. **Проверь:**
    `wsl -d Ubuntu`

Всё должно работать.  
Теперь `ext4.vhdx` будет лежать в `D:\WSL\Ubuntu`,  
и именно туда Docker будет сохранять образы и контейнеры.
