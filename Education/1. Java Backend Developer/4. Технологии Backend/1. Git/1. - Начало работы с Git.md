# 📕2. Начальная настройка git
## Проверка установки
```bash
git --version

# Ввыод
git version 2.51.0.windows.2
```

## Регистрация пользователя git
 
> [!NOTE] У каждого коммита есть МЕТАДАННЫЕ, о том, кто создал коммит
> Поэтому, `при связывании` `локального` и `удаленного` репозитория, лучше указать одни и те же `emal`

При работе над чем либо, пользователь создает коммиты. Чтобы его можно было как-то идентифицировать, каждый пользователь,  должен НАСТРОИТЬ 
- ИМЯ автора
- EMAIL автора 

Эти настройки ПРИМЕНЯЮТСЯ ГЛОБАЛЬНО, для всех репозиториев на компьютере

```bash
git config --global user.name ИМЯ_ПОЛЬЗОВАТЕЛЯ
git config --global user.name ПОЧТА_ПОЛЬЗОВАТЕЛЯ
```

Для ПРОСМОТРА глобальных настроек используется
```bash
git config --list
```
