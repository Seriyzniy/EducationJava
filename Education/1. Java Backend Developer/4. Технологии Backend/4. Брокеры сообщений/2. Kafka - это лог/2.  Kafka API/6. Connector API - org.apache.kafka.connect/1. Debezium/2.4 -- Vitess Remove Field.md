SMT **RemoveField** ==удаляет указанное поле из записи события==. Когда коннектор настроен на предоставление метаданных транзакций, используйте этот SMT, чтобы удалить поле `transaction.id` и избежать дублирования записи VGTID. 

Эта трансформация предназначена для использования только с коннектором Debezium для Vitess.

SMT использует опцию `field_names`, чтобы указать список путей к полям, которые нужно удалить, через запятую. SMT удаляет каждое указанное поле из записи события и из схемы события.
