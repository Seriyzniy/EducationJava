# 1. Создадим таблицы `Person` и `Passport`
![[Pasted image 20241215103611.png]]

Теперь создадим классы сущности. Начнем с `Passport`
```java
@Entity
@Table(name = "Passports")
public class Passport implements Serializable{
	@Id
	@OneToOne
	@JoinColumn(name = "fk_person_id", referencedColumnName = "person_id")
	private Person passport_person_id;
	
	@Column(name = "passport_number")
	private int passport_number;
	
	public Passport(){};
	public Passport(Person pers, int passport_number) {
		this.passport_number = passport_number;
		this.passport_person_id = pers;
	}
	public int getPassport_number() {
		return passport_number;
	}
	public void setPassport_number(int passport_number) {
		this.passport_number = passport_number;
	}
	
	public Person getPassport_person_id() {
		return passport_person_id;
	}
	public void setPassport_person_id(Person passport_person_id) {
		this.passport_person_id = passport_person_id;
	}
	@Override
	public String toString() {
		return "Passport serial " + passport_number;
	}
}
```

Теперь реализуем класс `Person`
```java
@Entity
@Table(name = "Persons")
	public class Person {
		@Id
		@Column(name = "person_id")
		@GeneratedValue(strategy = GenerationType.IDENTITY)
		private int id;
		
		@Column(name="person_name")
		private String name;
		
		@Column(name = "person_age")
		private int age;
		
		@OneToOne(mappedBy = "passport_person_id")
		@Cascade(CascadeType.SAVE_UPDATE)
		private Passport passport;
		
		public Person() {};
		public Person(String name, int age) {
			this.name = name;
			this.age = age;
		}
		public String getName() {
			return name;
		}
		public void setName(String name) {
			this.name = name;
		}
		public int getAge() {
			return age;
		}
		public void setAge(int age) {
			this.age = age;
		}
		public Passport getPassport() {
			return passport;
		}
		public void setPassport(Passport passport) {
			this.passport = passport;
		}
		public int getId() {
			return id;
		}
		@Override
		public String toString() {
			return "Person" + name + ", age " + age + ", " + passport;
		}
}
```

# 2. Добавление записей с отношениями в БД

```java
public class App {
	public static void main(String[] args) {
		Configuration myConfigHiber = new Configuration();
		
		myConfigHiber.addAnnotatedClass(Person.class);
		myConfigHiber.addAnnotatedClass(Passport.class);
		
		SessionFactory mySessionFact = myConfigHiber.buildSessionFactory();
		Session mySession = mySessionFact.getCurrentSession();
		try {
			mySession.beginTransaction();
			
			Person pers = new Person("Tom", 25);
			Passport passport = new Passport(pers, 12345);

			pers.setPassport(passport);
			
			mySession.save(pers);
			mySession.getTransaction().commit();
		}finally {
			mySessionFact.close();
		}
	}
}
```
