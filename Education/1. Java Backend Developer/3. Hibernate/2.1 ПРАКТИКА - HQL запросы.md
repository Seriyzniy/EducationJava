# 1. Простая выборка данных из таблицы при помощи HQL

> [!NOTE] Создание запроса HQL
> Чтобы создать HQL запрос, необходимо вызвать соответствующий метод `createQuery()`у объекта `Session`
>  

```java
...
try {
	mySession.beginTransaction();
	
	List<Person> people = mySession.createQuery("FROM Person").getResultList();
	
	for(Person pers : people) {
		System.out.println(pers);
	}
	
	mySession.getTransaction().commit();
	
}finally {
	mySessionFact.close();
}
...
```
При помощи метода `createQuery()` мы задаем запрос HQL. А при помощи метода `getResultList()` результат запроса помещаем в объект List
