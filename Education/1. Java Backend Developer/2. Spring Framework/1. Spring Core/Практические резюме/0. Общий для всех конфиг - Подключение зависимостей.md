Сборка проекта на базе Maven, архетип quick-start
# 1. Подключение зависимостей
***
##### Минимальная зависимость для Spring Core - `Spring-Context`
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-context</artifactId>
    <version>5.3.30</version> <!-- или актуальная версия -->
</dependency>
```
Эта зависимость ==ВКЛЮЧАЕТ в себя:==
- **Spring Core** (IoC контейнер, управление бинами)
- **Spring Beans** (механизм работы с бинами)
- **Spring Context** (расширение контейнера, поддержка аннотаций `@Component`, `@Autowired` и других)
***
##### Дополнительные зависимости (по необходимости)
1. **Spring AOP** (если используешь аспекты)
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-aop</artifactId>
    <version>5.3.30</version>
</dependency>
```

2. **Spring Expression Language (SpEL)** (если используешь SpEL в аннотациях)
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-expression</artifactId>
    <version>5.3.30</version>
</dependency>
```

1. **Spring Test** (если пишешь тесты для Spring-бинов)
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-test</artifactId>
    <version>5.3.30</version>
    <scope>test</scope>
</dependency>
```

2. **Spring Context Support** (дополнительные утилиты, например, работа с `@PropertySource`)
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-context-support</artifactId>
    <version>5.3.30</version>
</dependency>
```
