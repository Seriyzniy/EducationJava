# 1. Создание метода для ОТПРАВКИ пользователю формы заполнения данных

Для того, чтобы не внедрять значения полей из формы в объект вручную, будем передавать в HTML-форму объект класса `Person`. А для этого необходимо использовать HTML-формы Thymleaf

```java
@GetMapping("/new")
public String getCreatePersonForm(Model model) {
	model.addAttribute("newPerson", new Person());
	return "people/new";
}
```
Данный метод будет получать `Get-запрос` на адрес `people/new` и передавать HTML-форму для заполнения данных.  
Для того ==чтобы передать объект на HTML==-форму Thymleaf, ==создадим атрибут в модели== под именем `newPerson`

# 2. Создание метода для ПОЛУЧЕНИЯ от пользователя данных с формы
Реализуем данный метод следующим образом.

После того как пользователь захочет создать пользователя в БД, мы отравим ему HTML-форму с полями для заполнения. В ==качестве приемного адреса== (куда будут отправляться данные) укажем адрес `/people`. (POST запросы будут отлавливаться по адресу `/people`)

Далее создадим метод с `@PostMapping`, который будет получать POST-запросы на адрес `/people` и сохранять данные в БД (список людей). А также будет перенаправлять пользователя на "домашнюю" страницу

```java
@PostMapping()
public String addPersonInDataBase(@ModelAttribute("person") Person person) {
	myPersonDAO.saved(person);
	return "redirect:/people";
}
```

Однако ==нужно реализовать метод== `saved()` в классе `PersonDAO`, который будет добавлять объект класса `Person` в список.

# 3. Реализация метода добавления данных в БД (список)

Реализуем метод `saved()`
```java
@Component
public class PersonDAO {
	private List<Person> personsList;
	private static int ID;
	. . . 
	public void saved(Person pers) {
		pers.setId(++ID);
		personsList.add(pers);
	}
}
```
# 4. Реализация представлений

```html
<!DOCTYPE html>
<html xmlns:th="https://thymeleaf.org">
<head>
	<meta charset="UTF-8">
	<title>Create new Person</title>
</head>
<body>
	<form th:method="POST" th:action="@{/people}" th:object="${newPerson}">
		<label for="Name">Enter name person</label>
		<input type="text" th:field="*{name}" id="Name"/>
		<br>
		<input type="submit" value="Create!"/>
	</form>
</body>
</html>
```

> [!NOTE] Title
>Как и планировали:
>- Вместе с формой создали атрибут `newPerson` и отправили форму пользователю для заполнения полей
>.
>- При нажатии на кнопку `Create!` данные будут приходить на сервер на адрес `/people` с методом `POST`. 
>.
>- Все `POST` запросы на этот адрес будет обрабатывать метод `addPersonInDataBase()`, который с помощью `@ModelAtribute` создает новую пару ключ-значение
>	- Ключ - person
>	- Значение - объект класса `Person` с полями из HTML-формы
