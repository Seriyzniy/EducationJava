Регулярные выражения - особый вид выражений, используемый для описания определенных правил

Для их составления можно воспользоваться сайтом [Шпаргалка по регулярным выражениям](https://rexegg.com/regex-quickstart.php)
# 1. Добавления валидации для поля адрес

Добавим новое поле `adress` в `Person`
```java
...

private String adress;

public Person() {};
public Person(String name, int age, String email, String adress) {
	this.name = name;
	this.age = age;
	this.email = email;
	this.adress = adress;
}
...
```

У поля адрес должна быть определенная структура, например (Страна, Город, Почтовый индекс (6 цифр) )

Чтобы создать правило для валидации такого поля, необходимо использовать аннотацию `@Pattern`

Получим такое поле (см шпаргалку выше)
```java
@Pattern(regexp = "[A-Z]\\w+ , [A-Z]\\w+, \\d{6}")
private String adress;
```


> [!NOTE] ВАЖНО
> Также необходимо добавить поле `adress` в качестве параметра в методы создания/обновления человека, в PersonMapper, а также в HTML-формы, и в БД в виде отдельного столбца
> 
> А также добавить get/set методы для поля `adress`в классе `Person`
