Spring - фреймворк (каркас) упрощающий разработку Java приложений. 
Он реализует спецификации [[Java EE]].

Работа с фреймворком Spring в основном заточена на работу с аннотациями. Аннотации упрощает работу с фреймворком. Здесь собраны основные аннотации [[Шпаргалка по аннотация]]
Spring предоставляет множество готовых решений.

Sprint состоит из модулей:
	
- [[1. Spring Core (IoC&DI)]] - **основной модуль**
	 
- Spring AOP (Аспектно-ориентированное программирование)
	
- [[1. Spring Data - Введение (базы данных)]]
	
- [[1. Spring Web(MVC)]] - (Сервлеты)
	
- Spring Security 
	
- Spring Reactive
	
- Spring Integration
	
Spring Boot тот же Spring, предоставляющий функционал авто-конфигурирования.

Spring отлично подходит для микросервисной архитектуры. Для этого у него есть:
	
- Spring Actuator
	
- Spring Cloud
