# 2. Подключение к PostgreSQL из Eclipse

#### 1. Подключение зависимости
Чтобы подключиться к БД PostgreSQL необходимо установить драйвер `PostgreSQL`.
Если без сборщика проекта Maven, то необходимо скачать его вручную
А если используем сборщик проекта Maven то в `pom.xml` добавить зависимость

```xml
<dependency>
	<groupId>org.postgresql</groupId>
	<artifactId>postgresql</artifactId>
	<version>42.7.4</version>
</dependency>
```
#### 2. Поиск драйвера .jar
После этого, ==Maven сам установит драйвер Postgresql==, и сохранит его в ==локальном репозитории==

> [!NOTE] Локальный репозиторий `Maven`
> Локальный репозиторий Maven обычно находится по следующему пути:
> `C:\Users\<ваш_пользователь>\.m2\repository`

Чтобы найти установленный драйвер, необходимо идти по структурам папок соответствующим названию `groupID` в `pom.xml`, а именно:
- Перейти в папку локального репозитория Maven `org`
- Перейти в папку/папки/версия `postgresql`

> [!NOTE] Файл драйвера
> Имя файла драйвера `postgresql-42.7.4.jar`

Этот файл нам пригодится для создания профиля
#### 3. Создание профиля подключения

Теперь в Eclipse необходимо перейти `Window -> Show view -> Other -> Data Managment -> Data Source Explorer`

==Отроется окно== `Data Source Explorer` в нем создадим новое соединение с БД.
Нажмем на `New`
![[Pasted image 20241123204209.png]]
Выберем готовый профиль `Generic JDBC` и установим имя нашего нового профиля `myFirstJDBC` и нажмем на `Next`

Отроется окно с выбором готового профиля (у нас нет, создаем). Справа от поля ввода есть кнопка `New Driver Definition`

Вводим имя нашего нового профиля драйвера
![[Pasted image 20241123204657.png]]

Здесь переходим ==по пути драйвера PostgreSQL установленного Maven== и добавляем ==JAR файл==
![[Pasted image 20241123204734.png]]

Здесь ==указываем параметры:==
- **URL:** `jdbc:postgresql://localhost:5432/myFirstDB`
		Важно указать в URL имя БД, иначе подключимся к системной БД (postgres)!!!
		
- **Database Name**: имя БД в PostgreSQL (myFirstDB)
	
- **Driver Class**: имени класса драйвера JDBC, который отвечает за управление подключением между вашим приложением и базой данных (жми на Browse и копируй оттуда имя и в Type Name)
![[Pasted image 20241123204903.png]]


После этого вводим имя пользователя и пароль
- пароль от БД (12345)
- имя пользователя `postgres`

# 5. Установка соединения с БД в Java-коде

Для подключения к БД нам необходимо знать URL, имя пользователя и пароль. Эти данные должны храниться в файле `.properties` , но для простоты реализации будем эти данные хранить в коде

```java
@Component
public class PersonDAO {
	private static int ID;
	
	private static final String DB_URL = 
			"jdbc:postgresql://localhost:5432/myFirstDB";
	private static final String DB_USER = "postgres";
	private static final String DB_PASS = "12345";
	. . .
```

Теперь нам необходимо ==установить соединение==. Для этого используется класс `Connection`
Здесь мы добавим класс нашего драйвера в память (ОЗУ), чтобы потом найти его через DriverManager
> [!NOTE] Класс `Connetion`
> ```java
> 
> private static Connection myConnection;
> 
> static {
> 	try {
> 		Class.forName("org.postgresql.Driver");
> 	} catch (ClassNotFoundException e) {
> 		e.printStackTrace();
> 	}
> 
> 	try {
> 		myConnection = DriverManager.getConnection(DB_URL, DB_USER, DB_PASS);
> 	} catch (SQLException e) {
> 		// TODO Auto-generated catch block
> 		e.printStackTrace();
> 	}
> }

##### 1. **Загрузка драйвера PostgreSQL**:
```java
try {
	Class.forName("org.postgresql.Driver");
} catch (ClassNotFoundException e) {
	e.printStackTrace();
}
```
- Этот код загружает класс драйвера PostgreSQL, который предоставляет JDBC-интерфейс для подключения к базе данных.
- Метод `Class.forName("org.postgresql.Driver")` динамически загружает класс драйвера в память.
- Это важно для регистрации драйвера в `DriverManager`, чтобы он мог устанавливать соединения с базой данных.
##### 2. **Создание соединения с базой данных**:
```java
try { 
	myConnection = DriverManager.getConnection(DB_URL, DB_USER, DB_PASS); 
} catch (SQLException e) { 
	e.printStackTrace(); 
}
```
- Здесь создается подключение к базе данных с помощью `DriverManager.getConnection`.
- Метод принимает три параметра:
    - `DB_URL` — URL базы данных (в данном случае, локальный сервер PostgreSQL).
    - `DB_USER` — имя пользователя базы данных.
    - `DB_PASS` — пароль для этого пользователя.
- Это подключение сохраняется в статической переменной `myConnection`, что позволяет использовать его повторно в методах класса, избегая повторного подключения.

# 6. Реализация метода `getList()` в PersonDAO

Теперь когда у нас есть соединение с бд, мы можем реализовать метод вывода всех записей из бд
#### 1. Класс Statment

> [!NOTE] `java.sql.Statment`
> Класс **`Statement`** является частью пакета **`java.sql`** и представляет собой ==интерфейс для выполнения SQL-запросов== в базах данных. 
> 
>  Он используется для отправки SQL-команд (запросов) к базе данных ==через соединение (`Connection`).==

```java
public List<Person> getList(){
	List<Person> personsList = new ArrayList<>();
	try {
		Statement myStatement = myConnection.createStatement();
		String mySQLQuere = "SELECT * FROM persons";
		
		ResultSet myResultSet = myStatement.executeQuery(mySQLQuere);
		
		while(myResultSet.next()) {
			Person pers = new Person();
			pers.setId(myResultSet.getInt("id"));
			pers.setName(myResultSet.getString("personname"));
			pers.setAge(myResultSet.getInt("age"));
			pers.setEmail(myResultSet.getString("email"));
			
			personsList.add(pers);
		}
	}catch(SQLException e) {
		e.printStackTrace();
	}
	return personsList;
}
```

# 7. Реализация метода `saved()` в PersonDAO

Чтобы сохранить данные с формы в БД, нам необходимо принять данные с формы и с помощью SQL запроса `INSERT INTO` добавить данные в БД


> [!NOTE] Выполнение запроса `INSERT INTO`
> Для выполнения запроса на ВСТАВКУ данных, необходимо использовать метод класса `Statement` а именно `executeUpdate(String SQL)`

```java
public void saved(Person pers) {
	try {
		Statement myStatement = myConnection.createStatement();
		String insertSQL = "INSERT INTO persons VALUES(" + 5 + ",'" + pers.getName() + "'," + pers.getAge() + ",'" + pers.getEmail() + "')";
		
		myStatement.executeUpdate(insertSQL);
		
	} catch (SQLException e) {
		e.printStackTrace();
	}
}
```
