Там где данные с форм включаются в запрос, будем использовать `PreparedStatement`
# 1. Улучшение метода `saved()`

```java
public void saved(Person pers) {
	try {
		PreparedStatement myStatement =
			myConnection.prepareStatement("INSERT INTO persons VALUES(1, ?, ?, ?)");
	
		myStatement.setString(1, pers.getName());
		myStatement.setInt(2, pers.getAge());
		myStatement.setString(3, pers.getEmail());
		
		myStatement.executeUpdate();
		
	} catch (SQLException e) {
		e.printStackTrace();
	}
}
```

# 2. Реализация метода `getPerson()`

```java
public Person getPerson(int id) {
	Person searchedPerson = null;
	try {
		PreparedStatement myStatement = myConnection.prepareStatement("SELECT * FROM persons WHERE id = ?");
		
		myStatement.setInt(1, id);
		ResultSet myResult = myStatement.executeQuery();
		
		myResult.next();
		searchedPerson = new Person();
		
		searchedPerson.setId(myResult.getInt("id"));
		searchedPerson.setName(myResult.getString("personname"));
		searchedPerson.setAge(myResult.getInt("age"));
		searchedPerson.setEmail(myResult.getString("email"));
	
	} catch (SQLException e) {
		// TODO Auto-generated catch block
		e.printStackTrace();
	}
	return searchedPerson;
}
```
