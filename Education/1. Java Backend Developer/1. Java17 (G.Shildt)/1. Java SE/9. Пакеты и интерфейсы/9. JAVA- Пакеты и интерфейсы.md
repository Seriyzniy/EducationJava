**Пакет** — это **логическая группа классов, интерфейсов и под-пакетов**. Он:
- ==помогает избежать конфликтов имён,==
- упрощает управление проектом,
- и определяет уровень **доступа** (например, `default` доступ работает внутри одного пакета).

##### Объявление пакета
В начале Java-файла пишется:
```java
package mypackage;
```

Это значит, что файл `MyClass.java` должен находиться в папке `mypackage`, иначе возникнет ошибка компиляции.

##### Пакет соответствует ПАПКЕ (Директории)
Пакет в Java напрямую соответствует структуре папок.
```java
package com.example.utils;
```
→ Класс должен лежать по пути:  
`com/example/utils/MyClass.java`

##### 🔍 JVM ищет классы по **пути поиска классов (classpath)**:
- `CLASSPATH` — переменная среды
    
- Или `-cp` (или `-classpath`) в командной строке:

> [!NOTE] Поиск пакетов по умолчанию
> JVM ищет **в текущей директории (`.`)**, где запущена программа.

# 📕 1. Базовые пакеты 
### ✅ `java.lang`
**Импортируется по умолчанию**. Ты можешь использовать его классы **без `import`**.
Содержит:
- ⚙️ Базовые классы:  
    `Object`, `Class`, `System`, `Runtime`, `Math`, `Thread`, `Throwable`, `Exception`
    
- 🔤 Работа со строками:  
    `String`, `StringBuilder`, `StringBuffer`, `CharSequence`
    
- 📌 Классы-обёртки:  
    `Integer`, `Double`, `Boolean`, и т.д.
    
- 👷 Потоки: `Thread`, `Runnable`
    
- 🔒 Синхронизация: `ThreadLocal`, `Throwable`, `Error`
    

---
### ✅ `java.util`
**Один из самых больших и часто используемых пакетов.**
Содержит:
- 📚 Коллекции:
    - `List`, `ArrayList`, `LinkedList`
        
    - `Set`, `HashSet`, `TreeSet`
        
    - `Map`, `HashMap`, `TreeMap`, `Hashtable`
        
    - `Queue`, `Deque`, `PriorityQueue`
        
- 🧱 Структуры данных: `Stack`, `Vector`
    
- 📅 Работа с датами/временем (до Java 8): `Date`, `Calendar`, `TimeZone`
    
- 📆 Новый API даты и времени (с Java 8): `java.time.*`
    
- 🔁 Итераторы: `Iterator`, `Enumeration`
    
- 🎲 Случайные числа: `Random`, `SplittableRandom`
    
- 🧮 Утилиты: `Collections`, `Arrays`, `Objects`, `Optional`
    

---
### ✅ `java.io`
Для **ввода-вывода** через потоки (streams).
Содержит:
- 📂 Классы для работы с файлами:
    - `File`, `FileReader`, `FileWriter`
        
- 📥 Потоки:
    - `InputStream`, `OutputStream`
        
    - `FileInputStream`, `BufferedReader`, `PrintWriter`
        
- 🧱 Объекты:
    - `ObjectInputStream`, `ObjectOutputStream` — сериализация объектов
        

---
### ✅ `java.nio` (и `java.nio.file`)
Более **новый и производительный API ввода-вывода** (Non-blocking IO, Java 7+)
Содержит:
- Буферы: `ByteBuffer`, `CharBuffer`
    
- Каналы: `FileChannel`, `SocketChannel`
    
- Работа с файлами и путями: `Paths`, `Files`, `Path`
    

---
### ✅ `java.net`
Для работы с **сетевыми подключениями**.
Содержит:
- 🌐 Протоколы: `URL`, `URLConnection`, `HttpURLConnection`
    
- 🔌 Сокеты: `Socket`, `ServerSocket`, `DatagramSocket`, `InetAddress`
    

---

### ✅ `java.time`
Новый API даты/времени, введённый в Java 8.
Содержит:
- 📅 Дата и время:
    - `LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime`
        
- 📐 Временные интервалы: `Duration`, `Period`
    
- 🕒 Форматирование: `DateTimeFormatter`
    
- 🗺️ Зоны: `ZoneId`, `ZoneOffset`
    

---
### ✅ `java.math`
Для работы с **большими и точными числами**:
- `BigInteger` — целые числа произвольной длины
    
- `BigDecimal` — точные десятичные числа (для финансовых вычислений)
    
- `MathContext` — настройки точности и округления
    

---
### ✅ `java.sql`
Для работы с **базами данных** через JDBC:
- `Connection`, `Statement`, `ResultSet`, `PreparedStatement`
    
- `DriverManager`
    
- `SQLException`, `SQLWarning`
    

---
### ✅ `java.util.concurrent`
Для работы с **многопоточностью** и **синхронизацией** (современный API):
- Потоки: `ExecutorService`, `ThreadPoolExecutor`, `Future`
    
- Синхронизация: `Semaphore`, `ReentrantLock`, `CountDownLatch`
    
- Коллекции: `ConcurrentHashMap`, `CopyOnWriteArrayList`
    

---
### ✅ `java.security`
Инструменты для **шифрования**, **подписей**, **ключей**, **доступа**:

- `MessageDigest`, `Signature`, `KeyStore`, `AccessController`
