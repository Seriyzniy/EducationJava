# 📕 1. Вывоз конструкторов при иерархии наследования 

> [!NOTE] Порядок вызова конструкторов
> В иерархии классов, конструкторы завершают свое выполнение ==в порядке наследования от суперкласса к подклассу==, не смотря на то вызывается ли super()

```java
class A {
    A () {
	    System.out.println("Inside constructor A");
    }
}

class B extends A {
    B () {
        System.out.println("Inside constructor B");
    }
}

class C extends B{
    C () {
        System.out.println("Inside constructor C");
    }
}

//Main
public class MyMain {
	public static void main(String[] args) {
		C c = new C();
	}
}

//Вывод
Inside constructor A
Inside constructor B
Inside constructor C
```
