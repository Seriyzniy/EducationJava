
> [!NOTE] `BiPredicate<T, U>`
> Проверка условия на двух аргументах.

## Интерфейс
```java
@FunctionalInterface
public interface BiPredicate<T, U> {
    💡 boolean test(T t, U u);

    default BiPredicate<T, U> and(BiPredicate<? super T, ? super U> other);
    default BiPredicate<T, U> or(BiPredicate<? super T, ? super U> other);
    default BiPredicate<T, U> negate();
}
```

## 💡 Особенности 💡
- Используется в `Map.remove(key, value)` (сравнение ключ/значение).

## ✅Best Practices✅
- Полезен, если условие зависит от двух объектов.

## Пример
```java
BiPredicate<String, Integer> lengthEquals = (s, n) -> s.length() == n;

System.out.println(lengthEquals.test("Java", 4)); // true
```
