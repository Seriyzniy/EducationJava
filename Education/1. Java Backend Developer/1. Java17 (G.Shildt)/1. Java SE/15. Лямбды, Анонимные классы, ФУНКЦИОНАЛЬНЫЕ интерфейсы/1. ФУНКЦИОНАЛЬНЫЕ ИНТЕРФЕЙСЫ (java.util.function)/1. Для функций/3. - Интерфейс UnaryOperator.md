
> [!NOTE] `UnaryOperator<T>`
> Частный случай `Function<T,T>`: 
> - принимает и возвращает один и тот же тип

## Интерфейс
```java
@FunctionalInterface
public interface UnaryOperator<T> extends Function<T, T> {
    // ⭐ Наследует apply(T t)
}
```

## 💡 Особенности💡
- Используется для "трансформации" значения.
    
- Часто встречается в `List.replaceAll(UnaryOperator)`.

## Пример
```java
UnaryOperator<String> toUpper = String::toUpperCase;

System.out.println(toUpper.apply("java")); // "JAVA"
```
