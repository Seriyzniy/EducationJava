# 📕 1. Базовые

|Интерфейс|Абстрактный метод|Описание|Пример использования|
|---|---|---|---|
|`Consumer<T>`|`void accept(T val)`|Принимает один аргумент и не возвращает результат|`list.forEach(x -> System.out.println(x))`|
|`BiConsumer<T, U>`|`void accept(T tVal, U uVal)`|Принимает два аргумента и не возвращает результат|`map.forEach((k, v) -> System.out.println(k + ":" + v))`|
|`Supplier<T>`|`T get()`|Не принимает аргументов, возвращает результат|`() -> "Hello World"`|
|`Function<T, R>`|`R apply(T val)`|Принимает один аргумент и возвращает результат|`str -> str.length()`|
|`BiFunction<T, U, R>`|`R apply(T tVal, U uVal)`|Принимает два аргумента и возвращает результат|`(a, b) -> a + b`|
|`Predicate<T>`|`boolean test(T val)`|Принимает аргумент и возвращает boolean|`str -> str.isEmpty()`|
|`BiPredicate<T, U>`|`boolean test(T tVal, U uVal)`|Принимает два аргумента и возвращает boolean|`(a, b) -> a.equals(b)`|
