
> [!NOTE] `Consumer<T>`
> Операция над объектом без возврата (побочный эффект).

## Интерфейс
```java
@FunctionalInterface
public interface Consumer<T> {
    void accept(T t);

    default Consumer<T> andThen(Consumer<? super T> after);
}
```

## 💡Особенности💡
- Используется в `forEach`.
    
- Может вызывать побочные эффекты (печать, запись в БД).

## ✅Best Practices ✅
- Не смешивайте логику и побочные эффекты.

## Пример
```java
Consumer<String> printer = System.out::println;

printer.accept("Hello"); // Hello
```
