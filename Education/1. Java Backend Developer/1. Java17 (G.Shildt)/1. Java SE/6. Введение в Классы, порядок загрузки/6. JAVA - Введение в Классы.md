# 📕 1. Объекты и ссылки
==Все классы в Java находятся в куче (heap)==, и должны размещаться динамически при помощи оператора `new`

> [!NOTE] Оператор `new`
> Оператор `new` выделяет память для объекта во время ВЫПОЛНЕНИЯ программы

```java
class Box{
	double weigth;
	double height,
	double depth;
}

class BoxMain{
	public static void main(String[] args){
		Box box1;         //Объявление ссылки на объект
		box1 = new Box(); //Размещение в памяти объекта Box
	}
}
```

![[Pasted image 20250603112138.png]]

> [!NOTE] ==Объект класса == ссылка на объект!==
> При объявлении переменной типа класса, мы оперируем ссылкой!
> 

Поэтому, при присвоении одной ссылки другой, обе ссылки будут указывать на один и тот же объект
```java
Box box1 = new Box();
Box box2 = box1;
```

# 📕 2. Конструкторы класса
> [!NOTE] Возвращаемое значение конструктора
> НЕЯВНЫМ возвращаемым типом является сам класс
## Конструктор По умолчанию:
Если ты **не определяешь** конструктор — Java **автоматически добавит** конструктор **без параметров**, который просто ничего не делает:
```java
class A {
    int x;
}
// автоматический конструктор: A() {}
```
## Пользовательский конструктор:
```java
class A {
    int x;
    A(int val) {
        x = val;
    }
}
```

> [!NOTE] Отмена конструктора по умолчанию
> Если ты определяешь **любой** конструктор, **по умолчанию Java больше не добавляет пустой!**
