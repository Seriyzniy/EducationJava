### Основная идея
- В JDK 15 добавили **многострочные строковые литералы**.
    
- ⭐⭐Открываются и закрываются `"""`.⭐⭐
    
- Начинаются со строки после открывающих кавычек

Текстовые блоки — это многострочные строковые литералы, которые упрощают работу с длинным текстом (например, HTML, JSON, SQL).
Позволяют не вставлять спецсимволы (`\n, \t`) для установки отступов

# 📕 1. Основы текстовых блоков (синтаксис)
Текстовый блок поддерживается новым разделителем, который представляет собой ==три символа двойных кавычек==: """.

```java
String str = """ 
	Текстовые блоки облегчают
	работу с длинными строками
	и сохраняют отступы.
	""";
	
//Можно и так, если не нужен перевод на новую строку
String str = """ 
	Текстовые блоки облегчают
	работу с длинными строками
	и сохраняют отступы. """;
```

## Удаление пробельный символов

> [!NOTE] Ведущие пробелы удаляются по минимальному отступу
> Если 
> - одна строчка имеет отступ два пробела, 
> - вторая - четыре пробела
> - третья - шесть пробелов, 
> 
> тогда в начале каждой строчки удаляются только два пробела.

Если закрывающий разделитель выровнен по левому краю, то пробельные символы не удаляются. 
В противном случае пробельные символы удаляются, пока не встретится первый символ текста или закрывающий разделитель.
```java
//⭐ЕСЛИ ЗАКРЫВАЮЩИЕ кавычки находятся вот так, то текст выведится С ТАБУЛЯЦИЕЙ⭐
String str = """ 
	Текстовые блоки облегчают
	работу с длинными строками
	и сохраняют отступы. 
""";
```

# 📕 Итог
- Ведущие пробелы удаляются по минимальному отступу (удобно для форматирования).
    
- 💡Двойные кавычки `" "` можно писать без экранирования.
    
- 💡Поддерживаются спец-последовательности `\s` (пробел) и `\` в конце строки (чтобы убрать перенос).

➡️ Код с длинными строками (например, SQL-запросы) становится читаемым.
