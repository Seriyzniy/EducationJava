
> [!NOTE] **`CopyOnWriteArrayList`** — потокобезопасная реализация интерфейса `List`, 
> Использует стратегию «==копирования при записи==» (Copy-on-Write).
> 

💡Она идеально подходит для **сценариев, где число операций чтения значительно превышает число модификаций**.  

💡💡Все ==операции чтения происходят **без блокировок**==, а при изменении списка создается **новая копия массива**, что гарантирует отсутствие `ConcurrentModificationException`.

🧠 **Под капотом:**🧠
- Основан на обычном массиве (`transient volatile Object[] array`), хранящем элементы.
    
- При изменении списка (add, remove, set и т.д.) создается **новая копия массива**, в которую вносятся изменения.
    
- Все итераторы работают с **снимком массива на момент создания итератора**, что делает их безопасными без синхронизации.

## Конструкторы
```java
// Создает пустой список.
public CopyOnWriteArrayList()

// Создает список, содержащий элементы из другой коллекции.
public CopyOnWriteArrayList(Collection<? extends E> c)
// c — коллекция, чьи элементы будут скопированы в новый список.

// Создает список, содержащий элементы из указанного массива.
public CopyOnWriteArrayList(E[] toCopyIn)
// toCopyIn — массив, из которого копируются элементы.
```

## Методы
```java
//✅ Основные методы ДОБАВЛЕНИЯ и УДАЛЕНИЯ
/* 
add(E e)
Добавляет элемент в конец списка.
При каждом добавлении создается новая копия внутреннего массива.
Возвращает true, если элемент добавлен успешно.
*/
public boolean add(E e)

/*
add(int index, E element)
Вставляет элемент по заданному индексу.
Создает новую копию массива с добавленным элементом.
Бросает IndexOutOfBoundsException, если индекс некорректен.

public void add(int index, E element)

/*
addIfAbsent(E e)
Добавляет элемент только если он отсутствует в списке.
Проверка выполняется на основании equals().
Возвращает true, если элемент был добавлен.

public boolean addIfAbsent(E e)

/*
addAll(Collection<? extends E> c)
Добавляет все элементы из указанной коллекции.

Возвращает true, если список изменился.

public boolean addAll(Collection<? extends E> c)

/*
addAllAbsent(Collection<? extends E> c)

public boolean addAllAbsent(Collection<? extends E> c)

/*
remove(Object o)
Удаляет первое вхождение указанного объекта.

Возвращает true, если элемент найден и удалён.

public boolean remove(Object o)

remove(int index)
Удаляет элемент по указанному индексу.

public boolean retainAll(Collection<?> c)

/*
clear()
Очищает список, заменяя массив на пустой.

public void clear()

get(int index)

public E get(int index)

/*
set(int index, E element)

public boolean isEmpty()

indexOf(Object o)

/*
contains(Object o)
Проверяет, содержится ли элемент в списке (через equals()).

public boolean contains(Object o)

/*
containsAll(Collection<?> c)

//✅ Итерация и представления

iterator()
Возвращает итератор, основанный на snapshot-массиве.

listIterator()

public List<E> subList(int fromIndex, int toIndex)
```

## 💡 Особенности и подводные камни 💡

- **Каждая модификация создает новую копию массива**, поэтому при частых изменениях растут затраты по памяти и CPU.
    
- 💡**Операции чтения полностью неблокирующие**, т.к. работают с неизменяемым snapshot.
    
- 💡**Итераторы видят снимок**💡 — они не отражают последующие изменения и не кидают `ConcurrentModificationException`.
    
- ❌Не подходит для больших коллекций с активными записями — дорого.
    
- ✅ Отлично работает в сценариях "много читаем — редко пишем"

## Пример
```java
import java.util.concurrent.CopyOnWriteArrayList;

public class CopyOnWriteArrayListExample {
    public static void main(String[] args) {
        CopyOnWriteArrayList<String> list = new CopyOnWriteArrayList<>();

        list.add("A");
        list.add("B");
        list.addIfAbsent("C");

        // Потокобезопасная итерация
        for (String s : list) {
            System.out.println("Элемент: " + s);
            list.add("D"); // Не вызывает ConcurrentModificationException
        }

        System.out.println("Итоговый список: " + list);
    }
}
```

```java
import java.util.concurrent.CopyOnWriteArrayList;

public class CopyOnWriteArrayListExample {
    public static void main(String[] args) {
        List<String> list = new CopyOnWriteArrayList<>();
        
		// Добавление элементов
		list.add("Element1");
		list.add("Element2");
		
		// Итерация безопасна даже при модификации
		new Thread(() -> {
			for (String element : list) {
				System.out.println("Чтение: " + element);
				try { 
					Thread.sleep(100); 
				} catch (InterruptedException e) {}
			}
		}).start();
		
		// Модификация в другом потоке
		new Thread(() -> {
			try { Thread.sleep(50); } catch (InterruptedException e) {}
			list.add("Element3");
			list.remove("Element1");
		}).start();
    }
}
```
