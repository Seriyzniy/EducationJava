
> [!NOTE] **==Синхронизатор==** — это объект, который ==УПРАВЛЯЕТ ПОРЯДКОМ выполнения потоков== на основе своего состояния.
> 

**Примеры:**
- `BlockingQueue` — блокирует `put()` и `take()`, если очередь полная/пустая.
    
- `CountDownLatch` — потоки ждут, пока не произойдёт X событий.
    
- `Semaphore` — ограничивает количество одновременных доступов к ресурсу.
    
- `CyclicBarrier` — потоки ждут друг друга на "точке сбора".
    
- `FutureTask` — ожидает завершения асинхронного вычисления.
    
- `Exchanger` — два потока обмениваются данными.
