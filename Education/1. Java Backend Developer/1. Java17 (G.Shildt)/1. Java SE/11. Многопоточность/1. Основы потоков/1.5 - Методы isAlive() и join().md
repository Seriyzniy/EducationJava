Иногда требуется, чтобы главный поток завершился после того как завершаться все дочерние потоки.

Для этого используются ==следующие методы класса== `Thread`
## 🔶 1. Проверка завершенности потока
```java
final boolean isAlive();
```
Данный метод возвращает `true` если поток завершил свое выполнение, `false` в противном случае
## 🔶2. Ожидание дочерних потоков
```java
public final void join() throws InterruptedException
public final void join(long millis) throws InterruptedException
```
Метод `join()` ==**приостанавливает текущий== (выполняющийся) поток** до тех пор, ==пока **не завершится поток, на котором был вызван== `join()`**.

```java
class MyRunnable implements Runnable{
	private final String name;
	
	public MyRunnable(String name) {
		this.name = name;
	}
	@Override
	public void run() {
		for(int i = 0; i < 5; i++) {
			System.out.println("Thread " + name + ": i = " + i);
		}
	}
}

public class MyMain {
	public static void main(String[] args) {
		MyRunnable r1 = new MyRunnable("ONE");
		MyRunnable r2 = new MyRunnable("TWO");
		MyRunnable r3 = new MyRunnable("THREE");
		
		Thread t1 = new Thread(r1);
		Thread t2 = new Thread(r2);
		Thread t3 = new Thread(r3);
		
		t1.start();
		t2.start();
		t3.start();
		
		System.out.println("Поcле запуска потоков");
		
		try {
			System.out.println("Ожидание");
			t1.join();
			t2.join();
			t3.join();
		} catch (InterruptedException e) {
			e.printStackTrace();
		}
		
		System.out.println("После завершения");
		
	}
}

// Вывод
Поcле запуска потоков
Ожидание
Thread TWO: i = 0
Thread TWO: i = 1
Thread TWO: i = 2
Thread ONE: i = 0
Thread ONE: i = 1
Thread TWO: i = 3
Thread TWO: i = 4
Thread THREE: i = 0
Thread THREE: i = 1
Thread THREE: i = 2
Thread THREE: i = 3
Thread THREE: i = 4
Thread ONE: i = 2
Thread ONE: i = 3
Thread ONE: i = 4
После завершения
```
