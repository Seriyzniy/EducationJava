Эти классы предоставляют оболочки для `float` и `double`, а также константы (`NaN`, `POSITIVE_INFINITY`) и методы для проверки специальных значений:

- `isNaN()`, `isInfinite()`.

```java
Double d1 = Double.valueOf(3.14);
Double d2 = Double.valueOf("3.14");

System.out.println(d1.equals(d2)); // true
```

> [!NOTE] Начиная с версии JDK 12, классы `Float` и `Double` также реализуют интерфейсы `Constable` и `ConstantDesc`
>

В классах Float и Double ==определены следующие константы==:

| Константа             | Описание                                                                         |
| --------------------- | -------------------------------------------------------------------------------- |
| **BYTES**             | Ширина в байтах значения типа `float` или `double`.                              |
| **MAX_EXPONENT**      | Максимальный показатель степени.                                                 |
| **MAX_VALUE**         | Максимальное положительное значение.                                             |
| **MIN_EXPONENT**      | Минимальный показатель степени.                                                  |
| **MIN_NORMAL**        | Минимальное положительное **нормальное** значение.                               |
| **MIN_VALUE**         | Минимальное положительное значение (денормализованное, > 0).                     |
| **NaN**               | Специальное значение «не число» (Not-a-Number).                                  |
| **POSITIVE_INFINITY** | Положительная бесконечность.                                                     |
| **NEGATIVE_INFINITY** | Отрицательная бесконечность.                                                     |
| **SIZE**              | Ширина в битах значения внутри оболочки (`float` = 32, `double` = 64).           |
| **TYPE**              | Объект `Class`, представляющий примитивный тип (`float.class` / `double.class`). |
