В настоящей главе обсуждаются классы и интерфейсы, определенные в `java.lang`.

Как вам уже известно, пакет java.lang автоматически импортируется во все программы. Он содержит классы и интерфейсы, лежащие в основе практически всего кода Java.
Это наиболее широко используемый пакет Java

# 📕 1. Оболочки примитивных типов
==Примитивные типы== (например, `int`, `char`) ==не являются объектами==, но иногда требуется представить их в виде объектов (например, для использования в коллекциях).

Для этого существуют **классы-оболочки**:
- `Byte`, `Short`, `Integer`, `Long` — для целочисленных типов.
    
- `Float`, `Double` — для чисел с плавающей точкой.
    
- `Character` — для символов.
    
- `Boolean` — для логических значений.

***
# 📕 2. Абстрактный класс Number
Абстрактный класс `Number` ==является СУПЕРКЛАССОМ для ЧИСЛОВЫХ ОБОЛОЧЕК==. 

Он определяет методы для преобразования числового значения в различные примитивные типы:
- `intValue()`, `doubleValue()`, `byteValue()` и т.д.
