Класс `Character` — оболочка для `char`. 

Чтобы ==получить значение== `char`, содержащееся в объекте `Character`, нужно вызвать метод `charValue()`: 
```java
char charValue() // Он возвращает символ.
```


⭐Содержит методы для ==классификации символов==:
- `isDigit()`, `isLetter()`, `isUpperCase()`, `isWhitespace()`.

```java
char ch = 'A';
System.out.println(Character.isUpperCase(ch)); // true
```

**Поддержка Unicode**: Класс `Character` также поддерживает 32-битные кодовые точки (символы за пределами Basic Multilingual Plane), используя суррогатные пары.

В классe Character определено несколько констант, в том числе:
- BYTES - Ширина в байтах значения типа char
- MAX_RADIX-  Наибольшее основание системы счисления
- MIN RADIX - Наименьшее основание системы счисления
- MAX_VALUE - Наибольшее символьное значение
- MIN VALUE - Наименьшее символьное значение
- TYPE - Объект Class для char

# 📕 2. Дополнения класса Character для поддержки кодовых точек Unicode
### ❗❓Проблема
- Изначально все символы Unicode умещались в `char` (16 бит, диапазон `0–FFFF`).
    
- Unicode расширили до диапазона `0–10FFFF`, появились **дополнительные символы** (выше `FFFF`).
    
- `char` стал недостаточен → нужен способ обрабатывать символы, не помещающиеся в 16 бит.


> [!NOTE] ⭐❗Два важных термина❗⭐
> `Кодовая точка` - это символ в диапазоне от 0 до 10FFFF. 
> Символы, значения которых превышают FFFF, называются `дополнительными символами`. 
> 
> Основная многоязычная плоскость (basic multilingual plane - ВМР) охватывает символы от 0 до FFFF.

### ✅ Решение

1. 🔶 **Суррогатные пары**
    - ⭐Дополнительный символ кодируется двумя `char`:
        - старший суррогат, (от U+D800 до U+DBFF)
            
        - младший суррогат. (от U+DC00 до U+DFFF)
            
    - Методы вроде `codePointAt()` позволяют преобразовывать между кодовыми точками и суррогатами.
        
2. 🔶**Перегрузка методов**
    - Ранее существующие методы (например, `isDigit(char)`) перегружены версиями, работающими с `int`, чтобы поддерживать любые кодовые точки.
        
    - Примеры:
        
        - `static boolean isDigit(int cp)`
            
        - `static boolean isLetter(int cp)`
            
        - `static int toLowerCase(int cp)`
            

В `Character` добавлены методы для работы с кодовыми точками
