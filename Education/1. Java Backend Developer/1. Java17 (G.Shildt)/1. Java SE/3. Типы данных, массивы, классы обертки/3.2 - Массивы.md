# 📕 Массивы и ссылки
Массив — это **ссылка** на объект, а не сам объект. 
(==В Java массивы реализованы в виде ОБЪЕКТОВ==, вспомни переменную `length` - показывает общий размер массива, а не кол-во элементов в нем)
```java
int[] x = new int[5];

int[] y = x;
y[0] = 99;

System.out.println(x[0]); // 99 — потому что x и y указывают на один объект
```

> [!NOTE] Получение массива
> 🔷Во первых:
> Необходимо объявить переменную нужного типа массива
> 
> 🔷Во вторых:
> Необходимо ==выделить память== в которой будет хранится массив, при помощи оператора `new`
> 
> ==Примечание:==
> Несмотря на то, что массивы хранятся в **куче** (`heap`), они ==имеют **фиксированный размер**==, заданный при создании

```java
int[] a = new int[3];             // массив целых чисел
String[] names = new String[5];   // массив строк
double[] data = {1.1, 2.2, 3.3};  // инициализация с значениями
```
***
# 📕 Особенности массивов в Java

| Свойство                        | Описание                                                              |
| ------------------------------- | --------------------------------------------------------------------- |
| **Фиксированный размер**        | Размер задаётся при создании и **не может быть изменён**.             |
| **Тип данных**                  | Все элементы имеют **один и тот же тип**.                             |
| **Объекты в heap**              | ==Массивы хранятся в **куче** (heap), даже если содержат примитивы.== |
| **Наследование от `Object`**    | Любой массив — это объект, наследник `Object`.                        |
| **Поля массива**                | У массива есть поле `length` — количество элементов.                  |
| **Массив может содержать null** | Если это массив ссылочных типов, его элементы могут быть `null`.      |
***
## Доступ к элементам

```java
a[0] = 42;
System.out.println(a[0]);
```

***
## Длина массива
```java
int[] arr = new int[10];
System.out.println(arr.length); // 10
```
> [!NOTE] !!!
> ⚠ `length` — **поле**, не метод. Поэтому `arr.length() // ошибка`

***
## Вспомогательные методы
```java
import java.util.Arrays;

int[] arr = {3, 1, 2};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr)); // [1, 2, 3]

//------- Example 2 -------
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};

System.out.println(a == b); // false — разные объекты
System.out.println(Arrays.equals(a, b)); // true — содержимое одинаково
```
