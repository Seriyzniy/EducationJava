
> [!NOTE] Представляет **время суток без даты и зоны** (`часы, минуты, секунды, наносекунды`).
> Иммутабельный, потокобезопасный.  
> Реализует:  `Temporal`, `Comparable<LocalTime>`, `Serializable`.

## Фабричные методы:
```java
public static LocalTime now();
public static LocalTime now(ZoneId zone);
public static LocalTime of(int hour, int minute);
public static LocalTime of(int hour, int minute, int second);
public static LocalTime of(int hour, int minute, int second, int nano);
```

---
## 💡 Особенности и подводные камни 💡

1️⃣ Работает в пределах суток (00:00:00–23:59:59.999999999).  

2️⃣ Не знает о датах и зонах.  

3️⃣ Для измерения длительности используй `Duration`.  

4️⃣ Потокобезопасный.

---

## 💎 Best Practices 💎
✅ Подходит для локального времени (например, расписание).  

✅ ⭐Для timestamp'ов — используй `Instant` или `LocalDateTime`.  

✅ Не храни в UTC — это относительное представление.

---
## Пример
```java
LocalTime start = LocalTime.of(9, 30);
LocalTime end = LocalTime.of(17, 45);
System.out.println(Duration.between(start, end).toHours() + " часов работы");
```
