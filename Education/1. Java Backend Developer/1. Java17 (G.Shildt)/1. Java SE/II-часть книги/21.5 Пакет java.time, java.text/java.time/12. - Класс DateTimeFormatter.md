
> [!NOTE] DateTimeFormatter
> **Неизменяемый** (immutable) и **потокобезопасный** класс, 
> предназначенный `для форматирования и парсинга дат и времени` (`LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime` и др.).  
> 
> ❗❗❗Он полностью заменяет `SimpleDateFormat`.

Реализует интерфейсы:
- `TemporalAccessor` (через возвращаемые типы)
    
- `Serializable`

---
## Конструктор

```java
// Конструкторы приватные — экземпляры создаются через фабричные методы:

public static DateTimeFormatter ofPattern(String pattern)
// Создает форматтер по указанному шаблону (например, "dd.MM.yyyy HH:mm:ss").
// Эквивалент SimpleDateFormat(String pattern)

public static DateTimeFormatter ofPattern(String pattern, Locale locale)
// Создает форматтер с учетом локали.

public static DateTimeFormatter ofLocalizedDate(FormatStyle dateStyle)
// Создает форматтер для даты с заданным стилем (FULL, LONG, MEDIUM, SHORT)

public static DateTimeFormatter ofLocalizedTime(FormatStyle timeStyle)
// Форматтер только для времени.

public static DateTimeFormatter ofLocalizedDateTime(FormatStyle dateStyle, FormatStyle timeStyle)
// Форматтер для даты и времени.

public static final DateTimeFormatter ISO_LOCAL_DATE
// Предопределённый ISO формат: "yyyy-MM-dd"

public static final DateTimeFormatter ISO_LOCAL_DATE_TIME
// Предопределённый ISO формат: "yyyy-MM-dd'T'HH:mm:ss"
```

## Методы

### Форматирование
```java
// Форматирует дату/время в строку.
public String format(TemporalAccessor temporal)
```

### Разбор (парсинг)
```java
// Разбирает строку в объект типа TemporalAccessor.
public TemporalAccessor parse(CharSequence text)

// Разбирает строку в объект указанного типа (например, LocalDate.class).
public <T> T parse(CharSequence text, TemporalQuery<T> query)
```

### Локаль и настройки
```java
// Возвращает новый форматтер с заданной локалью.
public DateTimeFormatter withLocale(Locale locale)

// Возвращает новый форматтер с заданной временной зоной.
public DateTimeFormatter withZone(ZoneId zone)
```

---
## 💡 Особенности и подводные камни 💡

1️⃣ `DateTimeFormatter` — **потокобезопасен** и **immutable** (можно использовать общий экземпляр).  

2️⃣ Символы шаблонов отличаются от `SimpleDateFormat`, но в целом похожи (`y`, `M`, `d`, `H`, `m`, `s`, `E`).  

3️⃣ ISO-форматы поддерживаются "из коробки" — без шаблонов.  

4️⃣ Поддерживает **временные зоны** (`ZonedDateTime`) и **локали**.  

5️⃣ Можно легко кастомизировать формат через `ofPattern()`, комбинировать с `Locale` и `ZoneId`.

## Пример
```java
import java.time.*;
import java.time.format.*;

public class Example {
    public static void main(String[] args) {
        LocalDateTime now = LocalDateTime.now();
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("dd.MM.yyyy HH:mm:ss");

        String formatted = now.format(fmt);
        System.out.println("Formatted: " + formatted);

        LocalDateTime parsed = LocalDateTime.parse("15.10.2025 08:45:00", fmt);
        System.out.println("Parsed: " + parsed);
    }
}

Formatted: 15.10.2025 08:58:50

Parsed: 2025-10-15T08:45
```

## Пример парсинга из строки
```java
// Разбор строки с датой и временем.
import java.time.*;
import java.time.format.*;

class DateTimeDemo4 {
    public static void main(String[] args) {
        // Получить объект LocalDateTime путем разбора строки с датой и временем
        LocalDateTime curDateTime = LocalDateTime.parse(
            "June 30, 2021 12:01 AM",
            DateTimeFormatter.ofPattern("MMMM d', 'yyyy hh':'mm a", Locale.ENGLISH)
        );
        
        // Отобразить результаты разбора
        System.out.println(curDateTime.format(
            DateTimeFormatter.ofPattern("MMMM d', 'yyyy h':'mm a", Locale.ENGLISH)
        ));
    }
}

June 30, 2021 12:01 AM
```
