Используется в графике (OpenGL/Vulkan bindings через LWJGL), в научных вычислениях, при работе с бинарными протоколами и форматами, где данные хранятся в виде float.


> [!NOTE] `FloatBuffer` — специализированный буфер для хранения **32-битных чисел с плавающей точкой (`float`)**.
> - Наследуется от `Buffer`.
> - Реализует `Comparable<FloatBuffer>`. 
> - Может быть создан напрямую (`allocate`, `wrap`) или как **view buffer** над `ByteBuffer` (`asFloatBuffer()`).

## Конструкторы - нет конструкторов
`FloatBuffer` **не имеет публичных конструкторов**. Создание происходит через статические методы:
```java
// Создаёт новый буфер в куче
public static FloatBuffer allocate(int capacity)

// Оборачивает существующий массив float[]
public static FloatBuffer wrap(float[] array)

// Оборачивает часть массива
public static FloatBuffer wrap(float[] array, int offset, int length)
```

## 💡 Особенности и подводные камни💡

1. **View buffer**  
    Обычно `FloatBuffer` создают через `ByteBuffer.asFloatBuffer()`. Это позволяет читать те же байты как `float`.
```JAVA
ByteBuffer bb = ByteBuffer.allocate(16).order(ByteOrder.LITTLE_ENDIAN);
FloatBuffer fb = bb.asFloatBuffer();
```

2. **Endianness**  
    При использовании `asFloatBuffer()` порядок байт (`ByteOrder`) критичен. По умолчанию — BIG_ENDIAN.
    
3. **compact()**  
    Переносит все непрочитанные float в начало. Очень полезно для стриминга данных из канала.
4. **Размерность**  
    ==Каждый float = 4 байта.== Если использовать через `ByteBuffer`, то capacity в байтах делится на 4.

## ✅ Best Practices ✅
- Используй `ByteBuffer.allocateDirect()` + `asFloatBuffer()` для интеграции с нативными API (например, OpenGL, TensorFlow).
    
- Всегда явно указывай `ByteOrder`, если работаешь с бинарными протоколами.

## Пример
```java
import java.nio.*;

public class FloatBufferExample {
    public static void main(String[] args) {
        // --- Прямое создание ---
        FloatBuffer fb = FloatBuffer.allocate(4);

        fb.put(1.5f).put(2.5f).put(3.5f);
        fb.flip();

        while (fb.hasRemaining()) {
            System.out.println("Value: " + fb.get());
        }

        // --- View Buffer через ByteBuffer ---
        ByteBuffer bb = ByteBuffer
	        .allocate(16)
	        .order(ByteOrder.LITTLE_ENDIAN);
	        
        FloatBuffer view = bb.asFloatBuffer();

        view.put(10.1f).put(20.2f);
        view.flip();

        while (view.hasRemaining()) {
            System.out.println("From view: " + view.get());
        }
    }
}
```
