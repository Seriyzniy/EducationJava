
> [!NOTE] Класс `Paths` — это утилитарный класс (utility class) из пакета `java.nio.file`
> По сути, это =="фабрика" для создания== `Path`.

## Методы
```java
//✅ Создание объекта Path из строкового пути
// Создаёт объект Path на основе переданных строк частей пути.
// @param first — первый элемент пути (обычно строка с директорией или файлом)
// 💡@param more — дополнительные элементы пути (дополняют путь)
// @return Path — объект, представляющий путь в файловой системе
public static Path get(String first, String... more);

//✅ Создание объекта Path из URI
// Создаёт Path на основе URI (например, file:///C:/example.txt).
// Поддерживает только URI со схемой "file".
// @param uri — объект URI, описывающий путь
// @return Path — объект пути, соответствующий URI
// @throws IllegalArgumentException — если схема URI не поддерживается
public static Path get(URI uri);
```

## 💡 Особенности и подводные камни 💡
- 💡Все методы статические и возвращают объект `Path`, который фактически создаётся реализацией `FileSystem`.
    
- При передаче `URI` с неподдерживаемой схемой (например, `http://`), метод `get(URI)` бросает `IllegalArgumentException`.
    
- Метод `get(String, String...)` автоматически подставляет корректные разделители в зависимости от ОС (например, `\` для Windows и `/` для Linux).
    
- ⭐⭐`Paths.get()` — это предпочтительный способ получить `Path` вместо устаревшего `new File(path).toPath()`.

## Пример
```java
import java.nio.file.*;

public class PathsExample {
    public static void main(String[] args) {
        // ✅ Создание Path из строк
        Path path1 = Paths.get("C:", "Users", "Seriy", "Documents", "notes.txt");
        System.out.println("Path1: " + path1);

        // ✅ Создание Path из URI
        Path path2 = Paths.get(URI.create("file:///C:/Users/Seriy/Documents/notes.txt"));
        System.out.println("Path2: " + path2);

        // ✅ Получение абсолютного пути
        Path relative = Paths.get("logs", "today.log");
        System.out.println("Absolute: " + relative.toAbsolutePath());
    }
}
```
