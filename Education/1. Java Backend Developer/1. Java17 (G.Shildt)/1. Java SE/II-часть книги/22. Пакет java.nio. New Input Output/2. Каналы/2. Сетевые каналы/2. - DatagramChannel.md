

> [!NOTE] `DatagramChannel` — это **абстрактный класс**, наследующий `SelectableChannel` и реализующий `ByteChannel`, `ScatteringByteChannel`, `GatheringByteChannel`, `MulticastChannel`.
> 
> Используется для работы с **UDP**: ==отправка и получение пакетов без установления соединения.==
> 
> 🧠В отличие от TCP (`SocketChannel`), здесь каждое сообщение (датаграмма) — отдельный пакет.

## Конструктор
У `DatagramChannel` нет публичного конструктора.  
Создание через фабрику:
```java
// Открытие нового UDP-канала
public static DatagramChannel open() throws IOException;

// Открытие канала для указанного протокола (обычно "udp")
public static DatagramChannel open(ProtocolFamily family) throws IOException;
```

## Методы
```java
// ✅ Привязка к локальному адресу
// Привязывает канал к указанному адресу (например, порт UDP)
public final DatagramChannel bind(SocketAddress local) throws IOException;

// Возвращает локальный адрес (порт, IP)
public final SocketAddress getLocalAddress() throws IOException;


// ✅ Подключение / отключение (опционально)
// "Подключает" канал к удалённому адресу — после этого read/write работают только с ним
public abstract DatagramChannel connect(SocketAddress remote) throws IOException;

// Проверяет, подключен ли канал
public abstract boolean isConnected();

// Разрывает подключение (возвращает в "безадресный" режим)
public abstract DatagramChannel disconnect() throws IOException;

// Возвращает удалённый адрес, если канал подключен
public abstract SocketAddress getRemoteAddress() throws IOException;


// ✅ Чтение (унаследовано из ReadableByteChannel)
// Чтение в ByteBuffer
public abstract int read(ByteBuffer dst) throws IOException;

// Чтение с scatter (в несколько буферов)
public abstract long read(ByteBuffer[] dsts, int offset, int length) throws IOException;

// Чтение в массив буферов (полная версия)
public final long read(ByteBuffer[] dsts) throws IOException;



// ✅ Запись (унаследовано из WritableByteChannel)
// Запись из ByteBuffer
public abstract int write(ByteBuffer src) throws IOException;

// Запись gather (из нескольких буферов)
public abstract long write(ByteBuffer[] srcs, int offset, int length) throws IOException;

// Запись из массива буферов (полная версия)
public final long write(ByteBuffer[] srcs) throws IOException;



// ✅ ОТПРАВКА / ПРИЁМ пакетов (UDP-специфика)
// Отправляет датаграмму на указанный адрес
public abstract int send(ByteBuffer src, SocketAddress target) throws IOException;

// Принимает датаграмму в буфер (возвращает адрес отправителя)
public abstract SocketAddress receive(ByteBuffer dst) throws IOException;



// ✅ Multicast (унаследовано из MulticastChannel)
// Присоединяется к мультикаст-группе
public abstract MembershipKey join(InetAddress group, NetworkInterface interf) throws IOException;

// Присоединяется к мультикаст-группе с источником
public abstract MembershipKey join(InetAddress group, NetworkInterface interf, InetAddress source) throws IOException;



// ✅ Настройка опций (унаследовано из NetworkChannel)
// Установка опции (например, SO_BROADCAST, SO_RCVBUF)
public abstract <T> DatagramChannel setOption(SocketOption<T> name, T value) throws IOException;

// Получение опции
public abstract <T> T getOption(SocketOption<T> name) throws IOException;

// Поддерживаемые опции
public abstract Set<SocketOption<?>> supportedOptions();



// ✅ Управление blocking/non-blocking режимом (унаследовано из SelectableChannel)
public final DatagramChannel configureBlocking(boolean block) throws IOException;

public final boolean isBlocking();

public final Object blockingLock();



// ✅ Регистрация с Selector
// Регистрирует канал в селекторе
public final SelectionKey register(Selector sel, int ops) throws ClosedChannelException;

public abstract SelectionKey register(Selector sel, int ops, Object att) throws ClosedChannelException;

// Возвращает маску поддерживаемых операций: OP_READ, OP_WRITE, OP_CONNECT
public final int validOps();

// Возвращает ключ регистрации для указанного селектора
public abstract SelectionKey keyFor(Selector sel);



// ✅ Доступ к java.net.DatagramSocket
// Возвращает связанный DatagramSocket (для настройки в старом API)
public abstract java.net.DatagramSocket socket();



// ✅ Закрытие (унаследовано из Channel)
// Закрывает канал
public abstract void close() throws IOException;

// Проверяет, открыт ли канал
public abstract boolean isOpen();
```

## 💡 Особенности и подводные камни 💡
1. 💡**UDP без соединения**
    - Можно просто отправлять/принимать пакеты через `send()` / `receive()`.
        
    - Можно "подключить" канал к адресу (`connect`) → тогда можно использовать `read()` / `write()` как в TCP.
        
2. 💡**Non-blocking режим**
    - В non-blocking режиме `receive()` может вернуть `null`, если данных нет.
        
    - `send()` может вернуть `0`, если пакет не был отправлен.
        
3. 💡**Размер буфера**
    - 💡Нужно заранее выделять `ByteBuffer` достаточного размера для пакета.
        
    - 💡💡Если буфер меньше пакета, лишние байты отбрасываются.
        
4. **Мультикаст**
    - Поддерживает `join()` для подписки на multicast-группы.
        
    - Работает только на поддерживающих ОС.
        
5. **Selector**
    - Поддерживает операции `OP_READ`, `OP_WRITE`, `OP_CONNECT`.
        
    - Отличие от TCP: здесь каждое событие соответствует отдельному пакету.
        

---

## Best Practices

- Используй `send()` / `receive()` для классического UDP.
- Для работы как с TCP — используй `connect()` и тогда можно применять `read()` / `write()`.
    
- Для multicast — всегда проверяй поддержку у ОС и сети.
    
- Для non-blocking всегда проверяй `null` при `receive()`.
    
- Настраивай буфер через `setOption(StandardSocketOptions.SO_RCVBUF, ...)` для больших пакетов.
    
- Для high-performance серверов комбинируй с `Selector`.

## Пример
UDP-клиент (send/receive)
```JAVA
import java.net.*;
import java.nio.*;
import java.nio.channels.*;

public class UDPClientExample {
    public static void main(String[] args) throws Exception {
        try (DatagramChannel channel = DatagramChannel.open()) {
            channel.bind(null); // ОС выберет порт

            String msg = "Hello UDP Server!";
            ByteBuffer buffer = ByteBuffer.wrap(msg.getBytes());

            SocketAddress serverAddr = new InetSocketAddress("localhost", 9000);
            channel.send(buffer, serverAddr);

            // Получаем ответ
            buffer.clear();
            
            SocketAddress addr = channel.receive(buffer);
            buffer.flip();
            System.out.println("Response from " + addr + ": " + new String(buffer.array(), 0, buffer.limit()));
        }
    }
}

```

UDP-сервер (receive → send)
```java
import java.net.*;
import java.nio.*;
import java.nio.channels.*;

public class UDPServerExample {
    public static void main(String[] args) throws Exception {
        try (DatagramChannel channel = DatagramChannel.open()) {
            channel.bind(new InetSocketAddress(9000));

            ByteBuffer buffer = ByteBuffer.allocate(1024);

            while (true) {
                buffer.clear();
                SocketAddress client = channel.receive(buffer);
                buffer.flip();

                System.out.println("Received from " + client + ": " + new String(buffer.array(), 0, buffer.limit()));

                // Отправляем ответ
                buffer.clear();
                
                buffer.put("Hello client!".getBytes());
                buffer.flip();
                channel.send(buffer, client);
            }
        }
    }
}
```
