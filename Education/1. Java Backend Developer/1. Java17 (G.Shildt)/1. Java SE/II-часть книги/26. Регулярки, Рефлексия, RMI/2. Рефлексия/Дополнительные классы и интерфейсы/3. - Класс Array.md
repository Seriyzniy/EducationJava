Является `final`-классом
> [!NOTE] `Array` — это служебный класс в пакете `java.lang.reflect`, 
> Предоставляет универсальные (`рефлексивные`) `методы` для: `создания`, `чтения` и `изменения` `массивов` во время выполнения.
> 
> 🧠Он используется, `когда тип массива неизвестен` `во время компиляции` (например, при работе с `Class<?>` или `Field`, содержащим массив).

❗❗❗ Экземпляры класса `Array` НЕ НУЖНЫ, так как весь функционал реализован через статические методы.

## Методы
### Создание массивов
```java
// Создает новый массив указанного типа и длины
public static Object newInstance(Class<?> componentType, int length);

// Создает многомерный массив указанного типа и размеров
public static Object newInstance(
	Class<?> componentType, 
	int... dimensions);
```
> ⚙️ `componentType` — тип элемента массива (например, `int.class`, `String.class`).  
> 
> ⚙️ `dimensions` — список размеров для каждого измерения (например, `{3, 4}` создаёт матрицу 3×4).
> 
> ⚙️ ⭐Возвращаемый объект ==всегда имеет тип== `Object`, поэтому для использования требуется приведение типа.

### Чтение элементов массива
```java
// Получает элемент массива как Object
public static Object get(Object array, int index);

// Для примитивных типов существуют специализированные версии:
public static boolean getBoolean(Object array, int index);
public static byte getByte(Object array, int index);
public static char getChar(Object array, int index);
public static double getDouble(Object array, int index);
public static float getFloat(Object array, int index);
public static int getInt(Object array, int index);
public static long getLong(Object array, int index);
public static short getShort(Object array, int index);
```

> ⚙️ Эти методы безопасно извлекают элемент массива **неизвестного во время компиляции типа**.  
> 
> ⚙️ Если тип не совпадает, выбрасывается `IllegalArgumentException`.

### Изменение элементов массива
```java
// Устанавливает элемент массива значением value
public static void set(Object array, int index, Object value);

// Для примитивных типов — отдельные методы:
public static void setBoolean(Object array, int index, boolean value);
public static void setByte(Object array, int index, byte value);
public static void setChar(Object array, int index, char value);
public static void setDouble(Object array, int index, double value);
public static void setFloat(Object array, int index, float value);
public static void setInt(Object array, int index, int value);
public static void setLong(Object array, int index, long value);
public static void setShort(Object array, int index, short value);
```

> ⚙️ Если тип значения не совместим с типом элементов массива, выбрасывается `IllegalArgumentException`.

### Получение длины массива
```java
// Возвращает длину массива
public static int getLength(Object array);
```

> ⚙️ Аналог `array.length`, но работает ==с массивами любого типа==, представленного как `Object`.

---
## 💡 Особенности и подводные камни 💡
1️⃣ `Array` работает ==**только с массивами**==; передача не-массива вызывает `IllegalArgumentException`.  

2️⃣ 💡Методы возвращают и принимают `Object`, поэтому всегда нужно приведение типа.  

3️⃣ Для примитивных массивов нужно использовать специальные `getXxx`/`setXxx` версии.  

4️⃣ ⭐⭐⭐`Array.newInstance()` — единственный способ создать массив при неизвестном заранее типе.  

5️⃣ Многомерные массивы создаются **как массив массивов** (например, `int[][]` — это `int[][ ]`, а не «настоящая» матрица).

---
## 💎 Best Practices 💎

✅ 💡Используй `Array.newInstance()` при работе с рефлексией, ==когда заранее НЕИЗВЕСТЕН ТИП МАССИВА==.  

✅ 💡Проверяй, что объект действительно является массивом через `obj.getClass().isArray()` перед вызовом методов.  

✅ При доступе к примитивным массивам применяй соответствующий `getXxx`/`setXxx` метод.  

✅ 🧠Избегай ненужных приведений типов — храни тип компонента (`componentType`) отдельно.  

✅ 🧠Используй `java.lang.reflect.Array` совместно с `Field.get()` для динамического извлечения массивов из объектов.

## Пример
```java
import java.lang.reflect.Array;

public class ArrayReflectionExample {
    public static void main(String[] args) throws Exception {
        // Создаем массив типа int[5]
        Object intArray = Array.newInstance(int.class, 5);

        // Устанавливаем значения
        for (int i = 0; i < 5; i++) {
            Array.setInt(intArray, i, i * 10);
        }

        // Получаем длину массива
        int length = Array.getLength(intArray);
        System.out.println("Array length = " + length);

        // Читаем элементы массива
        for (int i = 0; i < length; i++) {
            int value = Array.getInt(intArray, i);
            System.out.println("Element " + i + " = " + value);
        }

        // Создаем двумерный массив String[2][3]
        Object matrix = Array.newInstance(String.class, 2, 3);
        Array.set(Array.get(matrix, 0), 1, "Hello");
        System.out.println(((String[][]) matrix)[0][1]); // Hello
    }
}
```
