После того как создали два проекта, скомпилируем и запустим их.

Можно воспользоваться IDE, но там можно запустить лишь один проект. Для этого мы оба проекта запустим и скомпилируем через консоль.

> [!NOTE] ⭐ИМЯ-ПАПКИ-СКОМПИЛИРОВАНОГО-МОДУЛЯ = ФАКТИЧЕСКОЕ-ИМЯ-МОДУЛЯ-В-`MODULE-INFO.JAVA`⭐
> 

---
## 📕 1. Компиляция и запуск СЕРВЕРА
### Компиляция
```bash
javac -d MyRmiBuild/rmiserver MyRmiServer/src/rmiserver/server/*.java MyRmiServer/src/shared/*.java MyRmiServer/src/module-info.java
```

### Запуск
```bash
java --module-path MyRmiBuild -m rmiserver/rmiserver.server.BillingServiceImpl
```

---
## 📕 2. Компиляция и запуск КЛИЕНТА

### Компиляция
```bash
javac -d MyRmiBuild/rmiclient MyRmiClient/src/rmiclient/client/*.java MyRmiClient/src/shared/*.java MyRmiClient/src/module-info.java
```

### Запуск
```bash
java --module-path MyRmiBuild -m rmiclient/rmiclient.client.BillingClient
```
