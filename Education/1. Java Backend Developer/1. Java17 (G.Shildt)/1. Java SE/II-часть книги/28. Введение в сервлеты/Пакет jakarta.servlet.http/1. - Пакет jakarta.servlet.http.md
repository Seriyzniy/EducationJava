Содержит ==классы и интерфейсы==, которые ==описывают и определяют== ==контракты между сервлет-классом==, работающим ==по HTTP-протоколу==, И ==средой выполнения==, предоставляемой для экземпляра такого класса соответствующим сервлет-контейнером.

# 📕1 . Основные интефрейсы и класы
## Интерфейсы, определенные в пакете jakarta.servlet.http
| Интерфейс              | Описание                                           |
| ---------------------- | -------------------------------------------------- |
| `HttpServletRequest`✅  | Позволяет сервлетам читать данные из HTTP-запроса  |
| `HttpServletResponse`✅ | Позволяет сервлетам записывать данные в HTTP-ответ |
| `HttpSession`✅         | Позволяет читать и записывать данные сеанса        |

## Классы, определенные в пакете jakarta.servlet.http

| Класс          | Описание                                                        |
| -------------- | --------------------------------------------------------------- |
| `Cookie`✅      | Позволяет сохранять информацию о состоянии на клиентской машине |
| `HttpServlet`✅ | Предоставляет методы для обработки запросов и ответов HTTP      |

# 📕 2. Остальные интерфейсы
## **Интерфейсы (Interface Summary)**

| Интерфейс                       | Описание                                                                                                         |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `HttpServletMapping`✅           | Представляет способ сопоставления запроса с соответствующим сервлетом                                            |
| `HttpServletRequest`✅           | Расширяет ServletRequest для предоставления HTTP-специфичной информации запроса                                  |
| `HttpServletResponse`✅          | Расширяет ServletResponse для предоставления HTTP-специфичной функциональности отправки ответа                   |
| `HttpSession`✅                  | Предоставляет способ идентификации пользователя между несколькими запросами и хранения информации о пользователе |
| `HttpSessionActivationListener` | Объекты, привязанные к сессии, могут отслеживать события контейнера об активации/деактивации сессий              |
| `HttpSessionAttributeListener`  | Слушатель изменений атрибутов сессий в веб-приложении                                                            |
| `HttpSessionBindingListener`    | Уведомляет объект при привязке или отвязке от сессии                                                             |
| `HttpSessionContext`            | ❌**Устарело** (с Servlet API 2.1 по соображениям безопасности)                                                   |
| `HttpSessionIdListener`         | Уведомляет об изменении ID HttpSession                                                                           |
| `HttpSessionListener`           | Уведомляет об изменениях в списке активных сессий веб-приложения                                                 |
| `HttpUpgradeHandler`            | ❓Интерфейс между процессом HTTP-апгрейда и новым протоколом                                                      |
| `Part`✅                         | Представляет часть, загруженную на сервер как часть multipart/form-data запроса                                  |
| `PushBuilder`✅                  | Строит push-запрос на основе HttpServletRequest                                                                  |
| `WebConnection`                 | ❓Интерфейс для взаимодействия HttpUpgradeHandler с обновленным HTTP-соединением                                  |

## **Классы (Class Summary)**

| Класс                         | Описание                                                                                                                 |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| `Cookie`✅                     | Создает cookie - небольшой объем информации, отправляемый сервлетом браузеру и возвращаемый обратно на сервер            |
| `HttpFilter`✅                 | Базовый класс, реализующий Filter и гарантирующий, что Request/Response имеют тип HttpServletRequest/HttpServletResponse |
| `HttpServlet` ✅               | Абстрактный класс для создания HTTP-сервлетов                                                                            |
| `HttpServletRequestWrapper`✅  | Удобная реализация HttpServletRequest для адаптации запроса                                                              |
| `HttpServletResponseWrapper`✅ | Удобная реализация HttpServletResponse для адаптации ответа                                                              |
| `HttpSessionBindingEvent`     | События привязки/отвязки объектов от сессии                                                                              |
| `HttpSessionEvent`            | События изменений сессий в веб-приложении                                                                                |
| `HttpUtils`                   | ❌**Устарело** (с Servlet API 2.3)                                                                                        |
