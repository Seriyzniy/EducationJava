
> [!NOTE] `ServletRequest` — это интерфейс, `представляющий` `HTTP-запрос` `(или ИНОЙ ПРОТОКОЛ`) поступающий на сервер.
> он предоставляет `методы` `для доступа к параметрам запроса`, 
> - телу, 
> - атрибутам, 
> - кодировке, 
> - локали и другим данным, связанным с клиентом. 
> 
> Используется контейнером сервлетов (например, Tomcat) для передачи данных запроса в метод `service()` сервлета.  
> 
> ⭐**Реализуется классами** вроде `HttpServletRequest`, `ServletRequestWrapper` и их наследниками.

## Интерфейс
```JAVA
package jakarta.servlet;

import java.io.*;
import java.util.*;

public interface ServletRequest {

    // ==============================
    // 🟢 Работа с потоками ввода 🟢
    // ==============================
    /**
     * ⭐⭐⭐Возвращает бинарный поток для чтения тела запроса (например, при отправке файла).
     */
    ServletInputStream getInputStream() throws IOException;


    // ==============================
    // 🟢 Работа с символами 🟢
    // ==============================
    /**
     * 💡💡Возвращает Reader для чтения текста из тела запроса (учитывает кодировку).
     */
    BufferedReader getReader() throws IOException;


    // ==============================
    // 🟢 Параметры запроса (например, ?name=value) 🟢
    // ==============================
    /**
     * 💡Возвращает значение параметра по имени (String).
     */
    String getParameter(String name);

    /**
     * 🧠Возвращает массив значений параметра (если параметр встречается несколько раз).
     */
    String[] getParameterValues(String name);

    /**
     * 💡Возвращает карту всех параметров запроса.
     */
    Map<String, String[]> getParameterMap();

    /**
     * Возвращает перечисление имен всех параметров.
     */
    Enumeration<String> getParameterNames();



    // ==============================
    // 🟢 Атрибуты (передаются между сервлетами / фильтрами) 🟢
    // ==============================
    /**
     * 💡Устанавливает атрибут (любой объект).
     */
    void setAttribute(String name, Object o);

    /**
     * 💡Получает атрибут по имени.
     */
    Object getAttribute(String name);

    /**
     * Удаляет атрибут по имени.
     */
    void removeAttribute(String name);

    /**
     * Возвращает имена всех атрибутов.
     */
    Enumeration<String> getAttributeNames();



    // ==============================
    // 🟢 Кодировка и локаль 🟢
    // ==============================
    /**
     * Устанавливает кодировку тела запроса (например, UTF-8).
     * Должен быть вызван до getReader() или getParameter().
     */
    void setCharacterEncoding(String env) throws UnsupportedEncodingException;

    /**
     * Возвращает текущую кодировку тела запроса.
     */
    String getCharacterEncoding();

    /**
     * Возвращает длину тела запроса (в байтах), если известно.
     */
    int getContentLength();

    /**
     * Возвращает длину тела запроса (в виде long).
     */
    long getContentLengthLong();

    /**
     * Возвращает MIME-тип тела запроса (например, "application/json").
     */
    String getContentType();

    /**
     * Возвращает предпочитаемую локаль клиента.
     */
    Locale getLocale();

    /**
     * Возвращает все локали, которые клиент готов принять.
     */
    Enumeration<Locale> getLocales();



    // ==============================
    // 🟢 Информация о соединении 🟢
    // ==============================
    /**
     * ⭐⭐Возвращает IP-адрес клиента.
     */
    String getRemoteAddr();

    /**
     * ⭐⭐Возвращает имя хоста клиента.
     */
    String getRemoteHost();

    /**
     * ⭐⭐Возвращает порт клиента.
     */
    int getRemotePort();

    /**
     * Возвращает IP-адрес сервера, на который пришел запрос.
     */
    String getLocalAddr();

    /**
     * Возвращает имя хоста сервера.
     */
    String getLocalName();

    /**
     * Возвращает порт сервера, на который пришел запрос.
     */
    int getLocalPort();



    // ==============================
    // 🟢 Прочее 🟢
    // ==============================
    /**
     * ⭐⭐Возвращает протокол запроса (например, HTTP/1.1).
     */
    String getProtocol();

    /**
     * ⭐⭐Возвращает схему запроса (http или https).
     */
    String getScheme();

    /**
     * Возвращает URL, по которому пришёл запрос.
     */
    String getServerName();

    /**
     * Возвращает порт сервера, указанный в запросе.
     */
    int getServerPort();

    /**
     * Возвращает RequestDispatcher для пересылки запроса на другой ресурс.
     */
    RequestDispatcher getRequestDispatcher(String path);

    /**
     * Возвращает объект ServletContext, связанный с запросом.
     */
    ServletContext getServletContext();

    /**
     * Проверяет, обрабатывается ли запрос в рамках AsyncContext.
     */
    boolean isAsyncStarted();

    /**
     * Проверяет, поддерживает ли запрос асинхронную обработку.
     */
    boolean isAsyncSupported();

    /**
     * Возвращает объект AsyncContext (для асинхронной обработки запроса).
     */
    AsyncContext getAsyncContext();

    /**
     * ⭐⭐Получение АСИНХРОННОГО КОНТЕКСТА, при переводе запроса в асинхронный режим.⭐⭐
     */
    AsyncContext startAsync() throws IllegalStateException;

    /**
     * Начинает асинхронную обработку, связывая с другими объектами запроса/ответа.
     */
    AsyncContext startAsync(ServletRequest servletRequest, ServletResponse servletResponse)
        throws IllegalStateException;

    /**
     * Возвращает уникальный идентификатор запроса.
     */
    String getRequestId();

    /**
     * Возвращает момент времени начала обработки запроса (в наносекундах с эпохи).
     */
    long getRequestTime();
}
```

---
## 💡 Особенности и подводные камни 💡

1️⃣ ❗❗❗**Методы `getParameter()` и `getReader()` / `getInputStream()` нельзя комбинировать** — после одного из них тело запроса уже будет прочитано, и второй вызов вызовет исключение.  

2️⃣ **Кодировку (`setCharacterEncoding`) нужно устанавливать до чтения параметров**, иначе изменения не применятся.  

3️⃣ ❗❗❗**Атрибуты живут только в рамках одного запроса** — они не передаются между запросами.  

4️⃣ ⭐⭐⭐**`ServletRequest` — протокольно-независимый интерфейс.** 
Для HTTP-запросов используется наследник `HttpServletRequest`, в котором гораздо больше методов (например, `getHeader()`, `getCookies()`, `getMethod()`).  
5️⃣ **`getRequestDispatcher()`** позволяет передать запрос на другой ресурс без участия клиента (серверное перенаправление).

---
## 💎 Best Practices 💎

✅ 💡Устанавливай кодировку (`request.setCharacterEncoding("UTF-8")`) **в самом начале обработки**.  

✅ Используй `getParameterMap()` для массового доступа к параметрам — это надёжнее, чем вручную перебирать `getParameterNames()`.  

✅ Для обмена данными между фильтрами и сервлетом — используй `setAttribute()`.  

✅ 💡Не храни в атрибутах тяжёлые объекты (например, подключения к БД) — только данные, актуальные в рамках запроса.  

✅ Для асинхронных задач — корректно заверши `AsyncContext` вызовом `complete()`.

## Пример
```java
package servlets;

import java.io.BufferedReader;
import java.io.IOException;
import java.util.stream.Collectors;

import jakarta.servlet.ServletException;
import jakarta.servlet.ServletRequest;
import jakarta.servlet.ServletResponse;
import jakarta.servlet.http.HttpServlet;

public class HelloServlet extends HttpServlet {
	public void service(
	    ServletRequest request, 
	    ServletResponse response) throws ServletException, IOException
	{
        // Устанавливаем кодировку до чтения параметров
        request.setCharacterEncoding("UTF-8");

        // Получаем параметр запроса
        String name = request.getParameter("name");

        // Устанавливаем атрибут для передачи другому компоненту
        request.setAttribute("username", name);

        // Получаем IP клиента
        String clientIp = request.getRemoteAddr();

        // Читаем тело запроса
        try (BufferedReader reader = request.getReader()) {
            String body = reader
	            .lines()
	            .collect(Collectors.joining("\n"));
            
            System.out.println("Paramentr in Header: " + name);
            System.out.println("IP: " + clientIp);
            System.out.println("Body: " + body);
        }
    }
}
```
