Поскольку ==объекты== `String` ==являются неизменяемыми==, всякий раз, когда нужно модифицировать строку, ее понадобится: 
- ✅ либо скопировать в `StringBuffer` или `StringBuilder`, 
- ❓❗либо использовать метод `String`, который создает новую копию строки с готовыми изменениями

***
# 📕 1. Методы модификации
## 📙1. Метод substring()

==Извлечь подстроку== можно с применением метода `substring()`
```java
String substring(int startIndex)
String substring(int startIndex, int endIndex)
```

Пример
```java
String s = "This is a test";

String sub1 = s.substring(5); // "is a test"
String sub2 = s.substring(5, 7); // "is"
```

***
## 📙 2. Метод concat()

С применением метода concat() можно ==выполнять конкатенацию двух строк==: 
```java
String concat(String str)
```

> [!NOTE] Метод `concat()` ==создает новый объект==, содержащий вызывающую строку с добавленным в конец содержимым str.
> 

Пример
```java
String s1 = "one";
String s2 = s1.concat("two"); // "onetwo"

// ⭐Эквивалентно: s1 + "two"⭐
```

***
## 📙3. Метод replace() 
Метод `replace()` имеет две формы. 
1) Первая форма заменяет ВСЕ ВХОЖДЕНИЯ ==ОДНОГО СИМВОЛА== в вызывающей строке ==ДРУГИМ СИМВОЛОМ==:
```java
String replace(char original, char replacement)
```

2) Вторая форма заменяет одну ==ПОСЛЕДОВАТЕЛЬНОСТЬ символов другой:==
```java
String replace(CharSequence original, CharSequence replacement)
```

```java
String s1 = "Hello".replace('l', 'w'); // "Hewwo"
String s2 = "Hello".replace("ll", "ww"); // "Hewwo"
```

***
## 📙 4. Метод trim()
Удаляет ведущие и конечные пробелы (коды ≤ 32)

> [!NOTE] Применительно к этому методу `пробелами считаются символы с кодами 32 и меньше`
>

```java
String s = "  Hello World  ".trim(); // "Hello World"
```

## 📙5. Методы strip(), stripLeading(), stripTrailing() (JDK 11+)

> [!NOTE] К таким пробельным символам относятся `помимо прочих пробелы, символы табуляции, возврат каретки и перевод строки.`
> 

```java
String s1 = "  Hello  ".strip(); // "Hello"
String s2 = "  Hello  ".stripLeading(); // "Hello  "
String s3 = "  Hello  ".stripTrailing(); // "  Hello"
```

# 📕 2. Изменение регистра символов внутри строк

## 📙 1. Методы toLowerCase() и toUpperCase()
```java
String s = "This is a test";

String upper = s.toUpperCase(); // "THIS IS A TEST"
String lower = s.toLowerCase(); // "this is a test"
```

Еще один момент: предоставляются также ==перегруженные версии методов== `toLowerCase()` и `toUpperCase()` , которые ==позволяют указывать объект== `Locale` для управления преобразованием.

***
# 📕 3. Соединение строк (static метод join())
Метод `join()` ==используется ДЛЯ СОЕДИНЕНИЕ ДВУХ И БОЛЕЕ строк С ОТДЕЛЕНИЕМ строк друг от друга== с помощью разделителя, такого как пробел или за пятая.

```java
static String join(CharSequence delim, CharSequence . . . strs)
```
- В `delim` ==указывается разделитель==, который применяется для отделения символьных последовательностей, заданных в `strs`. 
(✅РАЗДЕЛИТЕЛЬ НЕОБЯЗАТЕЛЬНО ОДИН СИМВОЛ, может быть и последовательность символов✅)

- В `strs` можно передавать ==список строк==.

```java
String result = String.join(" ", "Alpha", "Beta", "Gamma");
// "Alpha Beta Gamma"

String result2 = String.join(", ", "John", "ID#: 569", "Email");
// "John, ID#: 569, Email"
```
