Операции включают: 
- автоматическое создание новых экземпляров String из строковых литералов, 
- конкатенацию нескольких объектов String с помощью операции + 
- преобразование других типов данных в строковое представление

***
# 📕 1. Строковые литералы
```java
String s1 = new String("abc"); // Явное создание

String s2 = "abc"; // ✅ Неявное создание через литерал✅

System.out.println("abc".length()); // 3
```
- Компилятор автоматически создает объекты String для литералов
    
- ==Литералы помещаются в **пул строк** (string pool)==

***
# 📕 2. Конкатенация строк
Обычно в Java **не разрешено применять операции** к объектам `String`. 

Единственным ==исключением== из этого правила ==является операция +==, которая выполняет конкатенацию двух строк, создавая в результате объект String
```java
String age = "9";
String s = "Ему " + age + " лет."; // "Ему 9 лет."
```

> [!NOTE] ⭐ Особенности конкатенации ⭐
> 🔶`Сложение литералов`
> Если у тебя **только строковые литералы**, то конкатенация выполняется **на этапе компиляции**.
> ```java
> String s = "Hello" + " " + "World";
> System.out.println(s);
> 
> //Компилятор сразу превратит это в:
> String s = "Hello World";
> ```
> 👉 Тут никакого `StringBuilder` нет — результат вычисляется заранее, и в пуле строк создаётся один объект `"Hello World"`.
> 
>🔶`Сложение переменных String`
> ```java
> String a = "Hello"; 
> String b = "World"; 
> String c = a + " " + b;
> 
> //Компилятор превратит это в:
> String c = new StringBuilder()
> 				.append(a)
> 				.append(" ")
> 				.append(b)
> 				.toString();
> ```
> 👉 То есть **одна операция конкатенации** = **один `StringBuilder`**.
>

# 📕 3. Конкатенация строк с другими типами данных
Конкатенацию строк можно выполнять с другими типами данных.
```java
int age = 9;
String s = "Ему " + age + " лет."; // Автоматическое преобразование
```

Причина в том, что внутри объекта `String` значение `int` в `age` ==автоматически преобразуется в свое строковое представление==, с кото рым выполняется конкатенация.

> [!NOTE] Конкатенация строк выполняется первой, а вычисления позже
> ```java
> String s = "Четыре: " + 2 + 2; // "Четыре: 22"
> String s = "Четыре: " + (2 + 2); // "Четыре: 4"
> ```

# 📕 3. Преобразование в строку и `toString()`
## 📙 1. Метод valueOf():
Одним из способов преобразования данных в строку, является использование метода `valueOf()`
```java
static String valueOf(double num)
static String valueOf(long num) 
static String valueOf(Object ob)
```
- Преобразует любой тип в строковое представление
    
- Для объектов вызывает метод `toString()`

## 📙2. Переопределение toString():
```java
class Box {
    double width, height, depth;
    
    public String toString() {
        return "Размеры: " + width + " × " + height + " × " + depth;
    }
}

Box b = new Box(10, 12, 14);
System.out.println(b); // Автоматически вызывается toString()
```
