Инкапсулирует ==данные о валюте== (код ISO 4217, символ, число дробных цифр и т.д.). Полезно при форматировании денежных значений в сочетании с `NumberFormat`/`Currency` API.

## Конструкторы
❗❗❗Конструкторов нет
```java
// Используются фабричные методы:

static Currency getInstance(String currencyCode)
// Возвращает объект по коду валюты ISO 4217 (например, "USD").

static Currency getInstance(Locale locale)
// Возвращает валюту для данной локали.
```

## Методы
```java
String getCurrencyCode()
// Трёхсимвольный код ISO ("USD", "RUB").

String getSymbol()
// Символ валюты по умолчанию для локали.

String getSymbol(Locale locale)
// Символ для конкретной локали.

int getDefaultFractionDigits()
// Количество дробных знаков (например, для USD — 2, для JPY — 0).

int getNumericCode()
// Числовой код ISO 4217.

String toString()
// То же, что getCurrencyCode().
```

## Пример
```java
import java.util.Currency;
import java.util.Locale;

public class CurrencyExample {
    public static void main(String[] args) {
        Currency usd = Currency.getInstance("USD");
        
        System.out.println("Код: " + usd.getCurrencyCode());
        System.out.println("Символ: " + usd.getSymbol());
        System.out.println("Дробные знаки: " + usd.getDefaultFractionDigits());

        Currency rub = Currency.getInstance(new Locale("ru", "RU"));
        System.out.println("Российский рубль: " + rub.getSymbol());
    }
}
```
