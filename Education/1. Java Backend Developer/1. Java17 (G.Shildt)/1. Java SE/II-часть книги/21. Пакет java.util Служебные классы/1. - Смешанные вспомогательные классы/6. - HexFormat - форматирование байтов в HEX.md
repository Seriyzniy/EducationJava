Позволяет форматировать байты в HEX-представление и обратно.
## Конструктор
```java
// Создается через 💡фабрику
public static HexFormat of() 
public static HexFormat ofDelimiter(String delimiter)
```

## Основные методы
```java
public static HexFormat of()
// HEX без разделителя

public static HexFormat ofDelimiter(String delimiter) 
// HEX с разделителем


public String formatHex(byte[] bytes) 
// Массив → HEX-строка

public String formatHex(byte[] bytes, int offset, int length) 
// Часть массива → HEX


public byte[] parseHex(CharSequence string) 
// HEX-строка → массив байт


public int fromHexDigit(char ch) 
// Символ HEX → число

public int fromHexDigits(CharSequence string) 
// HEX-строка → число

public int fromHexDigits(CharSequence string, int fromIndex, int length) 
// Часть строки → число



public char toHighHexDigit(int value) 
// Старший HEX-символ числа

public char toLowHexDigit(int value) 
// Младший HEX-символ числа

```

## Пример
```java
HexFormat hex = HexFormat.of();
String hexStr = hex.formatHex(new byte[]{1,2,3}); // "010203"
byte[] bytes = hex.parseHex("0A0B");
```
