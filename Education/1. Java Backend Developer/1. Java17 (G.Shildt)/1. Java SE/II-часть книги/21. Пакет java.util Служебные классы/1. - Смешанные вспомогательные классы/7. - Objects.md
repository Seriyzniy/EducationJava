Утилитарный класс для работы с объектами.

## Конструктор
Нет публичных конструкторов

## Основные методы
```java
public static boolean equals(Object a, Object b) 
// null-safe equals

public static boolean deepEquals(Object a, Object b) 
// Глубокое сравнение массивов

public static int hashCode(Object o) 
// hashCode c null-safe

public static int hash(Object... values) 
// hash нескольких значений

public static <T> T requireNonNull(T obj) 
// Проверка на null (NPE при null)

public static <T> T requireNonNull(T obj, String message) 
// Проверка на null с сообщением

public static boolean isNull(Object obj) 
// true если null

public static boolean nonNull(Object obj) 
// true если НЕ null

public static String toString(Object o) 
// null-safe toString

public static String toString(Object o, String nullDefault) 
// null-safe toString с дефолтом

public static <T> int compare(T a, T b, Comparator<? super T> c) 
// null-safe сравнение

```

## Пример
```java
String name = null;
String safe = Objects.toString(name, "default"); // "default"
```
