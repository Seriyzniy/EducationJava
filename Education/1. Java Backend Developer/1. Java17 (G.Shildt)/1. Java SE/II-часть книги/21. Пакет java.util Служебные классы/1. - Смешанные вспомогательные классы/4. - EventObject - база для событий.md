Базовый класс для всех событий в Java. Содержит источник события (`source`).

## Конструктор
```java
public EventObject(Object source) // source — объект-источник события
```

## Основные методы
```java
public Object getSource() // Возвращает источник
public String toString() // Представление в строке
```

## Особенности
- Используется в AWT, Swing, JavaBeans.
    
- Все события наследуются от него.

## Пример
```java
EventObject event = new EventObject("Button");
System.out.println(event.getSource()); // Button
```
