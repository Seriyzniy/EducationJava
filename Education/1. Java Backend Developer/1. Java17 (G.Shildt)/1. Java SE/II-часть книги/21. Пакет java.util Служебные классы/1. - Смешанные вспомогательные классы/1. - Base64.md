Класс для кодирования и декодирования данных в формате Base64. 

Используется для передачи двоичных данных в текстовом виде (например, в HTTP-заголовках, JSON, XML).

У `Base64` есть два **вложенных статических класса** -
- `Base64.Encoder` 
- `Base64.Decoder` 
Они используются для непосредственного кодирования/декодирования.
## Конструктор
У этого класса нет публичных конструкторов.
Используются статические фабрики `getEncoder()`, `getDecoder()` и т.д.

## Методы
```java
public static Base64.Encoder getEncoder() 
// Обычный энкодер

public static Base64.Decoder getDecoder() 
// Обычный декодер

public static Base64.Encoder getUrlEncoder() 
// URL-безопасный энкодер (без '+', '/' и '=')

public static Base64.Decoder getUrlDecoder() 
// URL-безопасный декодер

public static Base64.Encoder getMimeEncoder() 
// MIME-энкодер (с переносами строк)

public static Base64.Encoder getMimeEncoder(int lineLength, byte[] lineSeparator) 
// MIME-энкодер с настройкой длины строки

public static Base64.Decoder getMimeDecoder() 
// MIME-декодер
```

## Особенности

- Есть стандартный и URL-безопасный режим.
    
- Работает с массивами байт и строками.
    
- Потокобезопасен.

***
# 📙 1. Base64.Encoder
Класс для кодирования байтов в Base64. Объекты создаются через `Base64.getEncoder()`, `getUrlEncoder()`, `getMimeEncoder()`.

## Основные методы
```java
public byte[] encode(byte[] src) 
// Кодирует массив байт → новый массив байт Base64

public int encode(byte[] src, byte[] dst) 
// Кодирует в существующий массив dst, возвращает кол-во записанных байт

public String encodeToString(byte[] src) 
// Кодирует в строку Base64

public OutputStream wrap(OutputStream os) 
// Оборачивает OutputStream — все записываемые байты будут кодироваться

public ByteBuffer encode(ByteBuffer buffer) 
// Кодирует содержимое ByteBuffer в новый ByteBuffer

public Base64.Encoder withoutPadding() 
// Возвращает энкодер без '=' на конце
```
## Особенности

- Поддерживает разные режимы: обычный, URL-безопасный, MIME.
    
- По умолчанию добавляет `=` в конце для выравнивания, можно отключить (`withoutPadding`).
    
- Можно оборачивать потоки для on-the-fly кодирования.

***
# 📙 2. Base64.Decoder

Класс для декодирования из Base64 в байты. Объекты создаются через `Base64.getDecoder()`, `getUrlDecoder()`, `getMimeDecoder()`.

## Основные методы
```java
public byte[] decode(byte[] src) 
// Декодирует массив байт → исходные данные

public byte[] decode(String src) 
// Декодирует строку → массив байт

public int decode(byte[] src, byte[] dst) 
// Декодирует в массив назначения, возвращает кол-во байт

public ByteBuffer decode(ByteBuffer buffer) 
// Декодирует содержимое ByteBuffer в новый ByteBuffer

public InputStream wrap(InputStream is) 
// Оборачивает InputStream — байты будут декодироваться на лету
```
## Особенности

- Понимает стандартный, URL-безопасный и MIME-варианты.
    
- При некорректном вводе выбрасывает `IllegalArgumentException`.
    
- Может работать с потоками (обертка InputStream).

***
## Пример
```java
String text = "Hello";
String encoded = Base64.getEncoder().encodeToString(text.getBytes());
String decoded = new String(Base64.getDecoder().decode(encoded));
```
