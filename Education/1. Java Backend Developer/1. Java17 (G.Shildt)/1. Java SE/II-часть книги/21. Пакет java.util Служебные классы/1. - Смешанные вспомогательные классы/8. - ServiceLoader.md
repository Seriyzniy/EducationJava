Позволяет находить и загружать реализации сервисов (SPI).

## Конструктор
Нет публичных конструкторов, ==используется фабрика==
```java
public static <S> ServiceLoader<S> load(Class<S> service)
```

## Основные методы
```java
public static <S> ServiceLoader<S> load(Class<S> service)
// Загружает сервис через ClassLoader по умолчанию

public static <S> ServiceLoader<S> load(Class<S> service, ClassLoader loader) 
// С указанием ClassLoader

public static <S> ServiceLoader<S> loadInstalled(Class<S> service) 
// Загружает сервисы из "installed providers"

public void reload() 
// Перезагружает список реализаций

public Iterator<S> iterator() 
// Итератор по сервисам

public Spliterator<S> spliterator() 
// Spliterator для стримов

public Stream<ServiceLoader.Provider<S>> stream() 
// Stream с метаинфо о провайдерах

```

## Особенности
- Работает через `META-INF/services/...`.
- Основной механизм SPI.
