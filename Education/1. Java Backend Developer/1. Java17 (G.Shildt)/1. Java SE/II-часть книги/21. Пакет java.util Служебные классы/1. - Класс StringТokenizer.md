
> [!NOTE] Класс `StringTokenizer`
> **Назначение:** Лексический анализ (разбор) строки на отдельные части (лексемы) с использованием разделителей.
> 
> ⭐Это ==устаревший==, «наследуемый» инструмент (современная альтернатива — регулярные выражения и `String.split()` или `Scanner`)

## Конструкторы
```java
StringTokenizer(String str)
// Создаёт токенайзер для строки str. 
// 💡Разделители по умолчанию: пробел, табуляция, перевод строки, возврат каретки, символ новой страницы.

StringTokenizer(String str, String delim)
// Создаёт токенайзер для строки str 💡с заданным набором разделителей💡 delim. Символы delim считаются разделителями.

StringTokenizer(String str, String delim, boolean returnDelims)
// То же, что выше, но если returnDelims = true, разделители также возвращаются как токены.

```

## Методы
```java
boolean hasMoreTokens()
// Возвращает true, если в строке остались ещё токены.

String nextToken()
// Возвращает следующий токен. Если токенов больше нет – NoSuchElementException.

String nextToken(String delim)
// Устанавливает новый набор разделителей delim и возвращает следующий токен.

int countTokens()
// Возвращает количество токенов, которые ещё можно извлечь.

boolean hasMoreElements()
// То же, что hasMoreTokens(), но возвращает boolean для интерфейса Enumeration.

Object nextElement()
// То же, что nextToken(), но возвращает Object (для интерфейса Enumeration).
```

## Применение и замечания:
- Удобен для быстрого парсинга простых форматов (например, `key=value;key2=value2`), но не годится для сложных случаев (скобки, кавычки, экранирование).
    
- Для нового кода предпочтительны `Pattern`/`Matcher`, `Scanner`, `split()` — это гибче и понятнее

## Пример
```java
import java.util.StringTokenizer;

public class TokenizerExample {
    public static void main(String[] args) {
        String input = "Java,C++,Python,Go";
        
        StringTokenizer st = new StringTokenizer(input, ",");

        while (st.hasMoreTokens()) {
            String token = st.nextToken();
            System.out.println(token);
        }
    }
}
```
