
> [!NOTE] ⭐ Лучше используй `SheduledExecutorService` ⭐
> 

`Timer` управляет планированием выполнения задач (`TimerTask`) в будущем с возможностью повторения. 

Это упрощённый планировщик (фоновой поток)


> [!NOTE] ❌ Замечания и осторожности: ❌
> - `Timer` — удобен для простых задач, но у него есть ограничения (например, вся логика выполняется в одном/нескольких потоках Timer — при исключении в `TimerTask` можно нарушить работу; 
> - ⭐Для сложных/масштабируемых задач предпочтительнее `ScheduledExecutorService`)

***
# 📕 1. Timer
## Конструкторы
```java
Timer()
// Создаёт таймер с неименованным фоновым потоком (не daemon).

Timer(boolean isDaemon)
// Создаёт таймер, поток-демон если isDaemon = true.

Timer(String name)
// Создаёт таймер с именованным потоком.

Timer(String name, boolean isDaemon)
// Именованный таймер с возможностью выбора демона.
```

## Методы
```java
void schedule(TimerTask task, long delay)
// Планирует выполнение задачи через delay миллисекунд.

void schedule(TimerTask task, long delay, long period)
// Планирует выполнение задачи через delay и далее повторять каждые period.

void schedule(TimerTask task, Date time)
// Запускает задачу в указанное время.

void schedule(TimerTask task, Date firstTime, long period)
// Запускает в указанное время, далее повторяет каждые period.



void scheduleAtFixedRate(TimerTask task, long delay, long period)
// Запускает через delay и затем периодически с фиксированным интервалом.

void scheduleAtFixedRate(TimerTask task, Date firstTime, long period)
// Запускает в указанное время и затем с фиксированным интервалом.



void cancel()
// Останавливает таймер и все задачи.

int purge()
// Удаляет отменённые задачи из очереди.
```

# 📕 2. TimerTask
## Конструкторы
```java
// ⭐Абстрактный класс, нужно наследовать и ⭐переопределить run()⭐.
protected TimerTask()
// Конструктор вызывается в подклассе.
```

## Методы
```java
public abstract void run()
// Код задачи (обязателен к переопределению).

boolean cancel()
// Отменяет задачу.

long scheduledExecutionTime()
// Время последнего запуска.
```

## Пример
```java
import java.util.Timer;
import java.util.TimerTask;

public class TimerExample {
    public static void main(String[] args) {
        Timer timer = new Timer();

        TimerTask task = new TimerTask() {
            @Override
            public void run() {
                System.out.println("Выполнилось: " + System.currentTimeMillis());
            }
        };

        // Запустить через 1 секунду и повторять каждые 2 секунды
        timer.schedule(task, 1000, 2000);
    }
}
```
