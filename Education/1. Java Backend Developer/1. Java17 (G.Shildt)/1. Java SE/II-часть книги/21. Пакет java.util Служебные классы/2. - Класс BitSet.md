
> [!NOTE] Класс `BitSet` 
> Компактное хранилище булевых флагов (битовый вектор)
> 

Представляет ==массив битов==, который динамически расширяется. 

Полезен, когда нужны операции над множествами битов (AND/OR/XOR, сдвиги, поиск следующего установленного/очищенного бита).

## Применение и советы:

- Используется для компактного хранения множества индексов, булевых флагов, реализации матриц/масок.
    
- Операции выполняются быстро (побитовые операции над блоками `long`), удобно для булевых множеств и быстрого подсчёта пересечений/объединений.

## Конструкторы
```java
BitSet()
// Создаёт пустое множество битов с начальной длиной по умолчанию (64 бита).

BitSet(int nbits)
// Создаёт множество битов размером не меньше nbits (❗❗округляется до ближайшего числа кратного 64).
```

## Пример
```java
import java.util.BitSet;

public class BitSetExample {
    public static void main(String[] args) {
        BitSet bs1 = new BitSet();
        BitSet bs2 = new BitSet();

        // Устанавливаем биты
        bs1.set(0); bs1.set(2); bs1.set(4); // 0001 0101
        bs2.set(1); bs2.set(2); bs2.set(3);

        System.out.println("bs1: " + bs1); // {0, 2, 4}
        System.out.println("bs2: " + bs2); // {1, 2, 3}

        // Побитовые операции
        BitSet clone = (BitSet) bs1.clone();
        clone.and(bs2);
        System.out.println("bs1 AND bs2: " + clone); // {2}

        clone = (BitSet) bs1.clone();
        clone.or(bs2);
        System.out.println("bs1 OR bs2: " + clone); // {0,1,2,3,4}
    }
}
```
