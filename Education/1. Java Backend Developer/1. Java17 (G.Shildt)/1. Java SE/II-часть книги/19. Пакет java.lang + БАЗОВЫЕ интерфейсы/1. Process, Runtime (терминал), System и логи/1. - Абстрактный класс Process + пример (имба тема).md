
> [!NOTE] Process и ProcessBuilder
> 🔶Абстрактный класс `Process` ==представляет== исполняемый ==процесс== (программу).
> 
> 🔶`ProcessBuilder` предоставляет более гибкий ==способ запуска и управления процессами== (например, установка рабочего каталога, перенаправление ввода-вывода).

Он используется главным образом как суперкласс для типов объектов, созданных:
- методом `ехес()` в классе `Runtime` 
- методом `start()` в классе `ProcessBuilder`.


> [!NOTE] 🧠Дескриптор процесса 
> Дескриптор процесса можно получить в виде экземпляра `ProcessHandle` вместе с информацией о процессе, инкапсулирован ной в экземпляре `ProcessHandle.Info`

С **JDK 9**: появился `ProcessHandle` + вложенный `ProcessHandle.Info` для получения сведений о процессе:
- `isAlive()` — проверка, выполняется ли процесс.
	
- `totalCpuDuration()` — сколько процесс использовал CPU.

С **JDK 17**: добавлены удобные методы для работы с потоками:
- `inputReader()`,
	
- `errorReader()`,
	
- `outputWriter()`.
	

👉 Итог: `Process` — оболочка над внешней программой, `ProcessHandle` даёт расширенный контроль и мониторинг.

## Запуск блокнота и закрытие его
```java
try {
    // Запуск процесса
    Process process = Runtime.getRuntime().exec("notepad.exe");
    
    // Проверка активности
    if (process.isAlive()) {
        System.out.println("Process is running with PID: " + process.pid());
    }
    
    // Ожидание завершения с таймаутом
    boolean finished = process.waitFor(5, TimeUnit.SECONDS);
    if (!finished) {
        process.destroyForcibly(); // Принудительное завершение
    }
    
    // Получение кода завершения
    int exitCode = process.exitValue();
    System.out.println("Process exited with code: " + exitCode);
    
} catch (IOException | InterruptedException e) {
    e.printStackTrace();
}
```

# 📌 Таблица методов Process
| Метод                                                                           | Описание                                                                                                                                                                                                             |
| ------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Методы ==управления процессом==**                                             |                                                                                                                                                                                                                      |
| `void destroy()`                                                                | Прекращает работу процесса                                                                                                                                                                                           |
| `Process destroyForcibly()`                                                     | Принудительно прекращает работу вызывающего процесса. Возвращает ссылку на процесс                                                                                                                                   |
| `int waitFor() throws InterruptedException`                                     | Возвращает код завершения, возвращенный процессом. Метод не возвращает управление до тех пор, пока не прекратит работу процесс, для которого он был вызван                                                           |
| `boolean waitFor(long waitTime, TimeUnit timeUnit) throws InterruptedException` | Ожидает окончания вызывающего процесса. Время ожидания задается параметром `waitTime` в единицах, указанных с помощью `timeUnit`. Возвращает `true`, если процесс закончился, и `false`, если истекло время ожидания |
| `CompletableFuture<Process> onExit()`                                           | Возвращает объект `CompletableFuture` для вызывающего процесса, который может использоваться для выполнения задач при прекращении работы                                                                             |
|                                                                                 |                                                                                                                                                                                                                      |
| **Методы ==получения информации==**                                             |                                                                                                                                                                                                                      |
| `int exitValue()`                                                               | Возвращает код завершения, полученный из подчиненного процесса                                                                                                                                                       |
| `boolean isAlive()`                                                             | Возвращает `true`, если вызывающий процесс все еще активен, или `false` в противном случае                                                                                                                           |
| `long pid()`                                                                    | Возвращает идентификатор процесса, ассоциированный с вызывающим процессом                                                                                                                                            |
| `boolean supportsNormalTermination()`                                           | Определяет, к какому прекращению работы приведет вызов метода `destroy()` — нормальному или принудительному. Возвращает `true`, если прекращение работы будет нормальным, или `false` в противном случае             |
|                                                                                 |                                                                                                                                                                                                                      |
| **Методы ==работы с потоками ввода-вывода==**                                   |                                                                                                                                                                                                                      |
| `InputStream getErrorStream()`                                                  | Возвращает поток ввода, который читает данные из потока вывода `err` процесса                                                                                                                                        |
| `InputStream getInputStream()`                                                  | Возвращает поток ввода, который читает данные из потока вывода `out` процесса                                                                                                                                        |
| `OutputStream getOutputStream()`                                                | Возвращает поток вывода, который записывает данные в поток ввода `in` процесса                                                                                                                                       |
|                                                                                 |                                                                                                                                                                                                                      |
| **Методы ==работы с дочерними процессами==**                                    |                                                                                                                                                                                                                      |
| `Stream<ProcessHandle> children()`                                              | Возвращает поток, который содержит объекты `ProcessHandle`, представляющие непосредственные дочерние процессы вызывающего процесса                                                                                   |
| `Stream<ProcessHandle> descendants()`                                           | Возвращает поток, который содержит объекты `ProcessHandle`, представляющие непосредственные дочерние процессы вызывающего процесса, плюс все его потомки                                                             |
|                                                                                 |                                                                                                                                                                                                                      |
| **Методы ==получения информации о процессе==**                                  |                                                                                                                                                                                                                      |
| `ProcessHandle.Info info()`                                                     | Возвращает информацию о процессе в виде объекта `ProcessHandle.Info`                                                                                                                                                 |
| `ProcessHandle toHandle()`                                                      | Возвращает дескриптор вызывающего процесса в виде объекта `ProcessHandle`                                                                                                                                            |
## Дополнительные методы (появились в JDK 17+):

| Метод                   | Описание                                                                                          |
| ----------------------- | ------------------------------------------------------------------------------------------------- |
| `Reader inputReader()`  | Возвращает `Reader` для ==стандартного вывода процесса== (альтернатива `getInputStream()`)        |
| `Reader errorReader()`  | Возвращает `Reader` для стандартного ==потока ошибок процесса== (альтернатива `getErrorStream()`) |
| `Writer outputWriter()` | Возвращает `Writer` для ==стандартного ввода процесса== (альтернатива `getOutputStream()`)        |
