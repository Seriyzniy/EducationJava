В интерфейсе `CharSequence` определены методы, предоставляющие доступ только для чтения к последовательности символов. 

Это общий контракт для «строкоподобных» сущностей. Интерфейс говорит: _«Я — последовательность символов, которую можно прочитать»_.

## Классы, реализующие CharSequence:

- **`String`** - неизменяемая последовательность символов
    
- **`StringBuilder`** - изменяемая последовательность символов (несинхронизированная)
    
- **`StringBuffer`** - изменяемая последовательность символов (синхронизированная)
    
- **`CharBuffer`** - буфер символов из NIO

## Методы

| Метод                                                      | Описание                                                                                                                                                                                |
| ---------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Основные методы**                                        |                                                                                                                                                                                         |
| `char charAt(int idx)`                                     | Возвращает **символ** по индексу, указанному в `idx`                                                                                                                                    |
| `int length()`                                             | Возвращает **количество символов** в вызывающей последовательности                                                                                                                      |
| `CharSequence subSequence(int startIdx, int stopIdx)`      | Возвращает **подмножество** вызывающей последовательности, начиная с индекса `startIdx` и заканчивая индексом `stopIdx-1`                                                               |
| `String toString()`                                        | Возвращает **строковый эквивалент** вызывающей последовательности                                                                                                                       |
|                                                            |                                                                                                                                                                                         |
| **Методы сравнения**                                       |                                                                                                                                                                                         |
| `static int compare(CharSequence seqA, CharSequence seqB)` | **Сравнивает** `seqA` и `seqB`. Возвращает **ноль** если последовательности одинаковы, **отрицательное значение** если `seqA < seqB`, или **положительное значение** если `seqA > seqB` |
|                                                            |                                                                                                                                                                                         |
| **Методы проверки содержимого**                            |                                                                                                                                                                                         |
| `default boolean isEmpty()`                                | Возвращает `true`, если вызывающая последовательность **не содержит символов**, или `false` в противном случае                                                                          |
|                                                            |                                                                                                                                                                                         |
| **Методы работы с потоками**                               |                                                                                                                                                                                         |
| `default IntStream chars()`                                | Возвращает **поток данных** (в виде экземпляра `IntStream`) для **символов** в вызывающем объекте                                                                                       |
| `default IntStream codePoints()`                           | Возвращает **поток данных** (в виде экземпляра `IntStream`) для **кодовых точек** в вызывающем объекте                                                                                  |
## Пример 
```java
public class CharSequenceExample {
    public static void main(String[] args) {
        // Различные реализации CharSequence
        CharSequence str = "Hello World";
        CharSequence sb = new StringBuilder("Hello StringBuilder");
        CharSequence sbf = new StringBuffer("Hello StringBuffer");
        
        // Основные методы
        System.out.println("Длина: " + str.length());
        System.out.println("Символ по индексу 1: " + str.charAt(1)); // 'e'
        System.out.println("Подпоследовательность: " + str.subSequence(0, 5)); // "Hello"
        System.out.println("Строковое представление: " + str.toString());
        
        // Проверка на пустоту
        System.out.println("Пустая? " + str.isEmpty());
        System.out.println("Пустая? " + "".isEmpty());
        
        // Сравнение последовательностей
        int result = CharSequence.compare("abc", "def");
        System.out.println("Результат сравнения: " + result); // отрицательное значение
        
        // Работа с потоками символов
        System.out.println("Символы как поток:");
        str.chars().forEach(ch -> System.out.print((char) ch + " "));
        System.out.println();
        
        System.out.println("Кодовые точки как поток:");
        str.codePoints().forEach(cp -> System.out.print(cp + " "));
        System.out.println();
        
        // Универсальный метод для работы с любым CharSequence
        printCharSequenceInfo(str);
        printCharSequenceInfo(sb);
        printCharSequenceInfo(sbf);
    }
    
    public static void printCharSequenceInfo(CharSequence cs) {
        System.out.println("\nИнформация о CharSequence:");
        System.out.println("Тип: " + cs.getClass().getSimpleName());
        System.out.println("Содержимое: " + cs);
        System.out.println("Длина: " + cs.length());
        System.out.println("Пустая? " + cs.isEmpty());
        
        if (cs.length() > 0) {
            System.out.println("Первый символ: " + cs.charAt(0));
            System.out.println("Последний символ: " + cs.charAt(cs.length() - 1));
        }
    }
}
```

### Практическое применение (обработка любых последовательностей символов)
```java
// Универсальная обработка текста
public void processText(CharSequence text) {
    if (text.isEmpty()) {
        return;
    }
    
    // Работа с любым типом текстовых данных
    for (int i = 0; i < text.length(); i++) {
        char c = text.charAt(i);
        // обработка символа
    }
    
    // Или с использованием потоков
    text.codePoints()
        .filter(Character::isLetter)
        .forEach(cp -> System.out.print(Character.toChars(cp)));
}
```

## Собственная реализация реверс-строки
```java
class ReverseString implements CharSequence {
    private final String str;

    public ReverseString(String s) {
        this.str = s;
    }

    @Override
    public int length() {
        return str.length();
    }

    @Override
    public char charAt(int index) {
        return str.charAt(str.length() - 1 - index);
    }

    @Override
    public CharSequence subSequence(int start, int end) {
        return new ReverseString(str.substring(str.length() - end, str.length() - start));
    }

    @Override
    public String toString() {
        return new StringBuilder(str).reverse().toString();
    }
}

public class TestCharSeq {
    public static void main(String[] args) {
        CharSequence cs = new ReverseString("Java");
        System.out.println(cs.charAt(0));   // 'a'
        System.out.println(cs);             // avaJ
    }
}
```
