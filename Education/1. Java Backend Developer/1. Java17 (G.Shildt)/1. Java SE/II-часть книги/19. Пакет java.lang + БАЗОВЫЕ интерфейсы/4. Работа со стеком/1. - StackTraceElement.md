Класс `StackTraceElement` описывает ==одиночный стековый фрейм==, который является индивидуальным элементом трассировки стека, ==когда возникает исключение==

| Метод                                                 | Описание                                                                                                                                                                           |
| ----------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Методы ==получения информации о классе==**          |                                                                                                                                                                                    |
| `String getClassName()`                               | Возвращает **имя класса**, в котором оказалась точка выполнения, описанная вызываемым объектом `StackTraceElement`                                                                 |
| `String getClassLoaderName()`                         | Возвращает **имя загрузчика классов**, используемого для загрузки класса. Если объект не включает информацию о загрузчике классов, возвращается `null`                             |
| **Методы получения информации ==о методе==**          |                                                                                                                                                                                    |
| `String getMethodName()`                              | Возвращает **имя метода**, в котором оказалась точка выполнения, описанная вызывающим объектом `StackTraceElement`                                                                 |
| `boolean isNativeMethod()`                            | Возвращает `true`, если точка выполнения, описанная вызывающим объектом `StackTraceElement`, оказалась в **собственном (native) методе**, или `false` в противном случае           |
|                                                       |                                                                                                                                                                                    |
| **Методы получения информации ==об исходном коде==**  |                                                                                                                                                                                    |
| `String getFileName()`                                | Возвращает **имя файла**, в котором хранится исходный код точки выполнения, описанный вызываемым объектом `StackTraceElement`                                                      |
| `int getLineNumber()`                                 | Возвращает **номер строки** исходного кода, в которой оказалась точка выполнения. В ряде ситуаций номер строки не будет доступным, и тогда возвращается **отрицательное значение** |
|                                                       |                                                                                                                                                                                    |
| **Методы получения информации ==о модуле== (JDK 9+)** |                                                                                                                                                                                    |
| `String getModuleName()`                              | Возвращает **имя модуля**, в котором оказалась точка выполнения. Если объект не содержит информацию о модуле, возвращается `null`                                                  |
| `String getModuleVersion()`                           | Возвращает **версию модуля**, в котором оказалась точка выполнения. Если объект не содержит информацию о модуле, возвращается `null`                                               |
|                                                       |                                                                                                                                                                                    |
| **==Методы Object==**                                 |                                                                                                                                                                                    |
| `boolean equals(Object ob)`                           | Возвращает `true`, если вызывающий объект `StackTraceElement` является тем же, что и переданный в `ob`, или `false` в противном случае                                             |
| `int hashCode()`                                      | Возвращает **хеш-код** для вызывающего объекта `StackTraceElement`                                                                                                                 |
| `String toString()`                                   | Возвращает **строковый эквивалент** вызывающей последовательности                                                                                                                  |
