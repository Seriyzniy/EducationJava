
> [!NOTE] Класс **`java.net.HttpURLConnection`** ==расширяет== `URLConnection`
> Предоставляет **специализированные средства `для работы по протоколам` `HTTP` и `HTTPS`**.
> 
> ⭐Реализует интерфейс: `AutoCloseable`

❗❗Он позволяет:❗❗
- ❗❗управлять HTTP-запросами (`GET`, `POST`, `HEAD`, `PUT`, `DELETE`, `OPTIONS`);
    
- отправлять данные на сервер;
    
- ❗❗читать HTTP-заголовки и тело ответа;
    
- ❗❗получать код состояния (200, 404, 500 и т.п.);
    
- управлять кэшированием, редиректами и авторизацией.

> [!NOTE] ⭐⭐`HttpURLConnection` — это основной `низкоуровневый` API для HTTP в Java (до появления `HttpClient` в JDK 11).⭐⭐
> 

## Фабричный метод получения экземпляра
```java
//✅ Создает объект HttpURLConnection для указанного URL.
// Этот метод вызывается через URL.openConnection(), а не напрямую через конструктор.
HttpURLConnection (URL url)
```

🔹 Класс `HttpURLConnection` — **абстрактный**, поэтому ==экземпляр создаётся фабрично==:
```java
URL url = new URL("https://example.com");
HttpURLConnection conn = (HttpURLConnection) url.openConnection();
```

## Методы
Класс `HttpURLConnection` **==НАСЛЕДУЕТ ВСЕ МЕТОДЫ== `URLConnection`**,  
и добавляет HTTP-специфичные методы для работы с запросами и ответами.

### Управление запросом
```JAVA
//✅ Устанавливает HTTP-метод (по умолчанию "GET").
// Примеры: "GET", "POST", "PUT", "DELETE", "HEAD".
void setRequestMethod(String method)
    throws ProtocolException

//✅ Возвращает текущий метод запроса.
String getRequestMethod()

//✅ Устанавливает значение заголовка запроса.
void setRequestProperty(String key, String value)

//✅ Возвращает значение заголовка запроса по ключу.
String getRequestProperty(String key)

//✅ Устанавливает, будет ли соединение использовать кэш.
void setUseCaches(boolean useCaches)

//✅ Разрешает/запрещает автоматическое перенаправление (HTTP 3xx).
void setInstanceFollowRedirects(boolean followRedirects)

//✅ Проверяет, включены ли автоматические перенаправления.
boolean getInstanceFollowRedirects()
```

### Работа с ответом сервера
```java
//✅ Возвращает код состояния HTTP-ответа (например, 200, 404, 500).
int getResponseCode()
    throws IOException

//✅ Возвращает текстовое сообщение, связанное с кодом ответа (например, "OK", "Not Found").
String getResponseMessage()
    throws IOException

//✅ Возвращает входной поток для чтения тела ответа (при успешных кодах).
InputStream getInputStream()
    throws IOException

//✅ Возвращает поток для чтения тела ошибки (для кодов >= 400).
InputStream getErrorStream()

//✅ Возвращает длину тела ответа.
int getContentLength()

//✅ Возвращает MIME-тип содержимого ответа.
String getContentType()
```

### Соединение и завершение
```java
//✅ Явно открывает соединение (если ещё не открыто).
void connect()
    throws IOException

//✅ Завершает соединение, освобождая ресурсы.
// После вызова disconnect() соединение нельзя использовать повторно.
void disconnect()

//✅ Проверяет, разрешено ли повторное использование соединения.
boolean usingProxy()
```

## 💡 Особенности и подводные камни 💡

- ❗❗По умолчанию используется метод **GET**. Для отправки данных нужно явно указать **POST** и вызвать `setDoOutput(true)`.
    
- ❗❗ Нельзя изменять метод запроса **после открытия соединения** (`connect()` или `getInputStream()` уже вызваны).
    
- ⏱️ Без установки тайм-аутов программа может зависнуть при долгом ожидании.
    
- 🔒 При HTTPS соединении используются встроенные SSL-сертификаты Java (`cacerts`), но при работе с самоподписанными нужно настраивать TrustManager.
    
- 🧠Если `getResponseCode()` вызывается до `connect()`, соединение открывается автоматически.
    
- ⚠️ Поток из `getInputStream()` нужно закрывать обязательно, иначе соединение останется занятым в пуле.
    
- ❓❓Некоторые серверы требуют указания заголовков (`User-Agent`, `Accept`, `Content-Type`).

## 💎 Best Practices 💎

✅ Устанавливай тайм-ауты перед вызовом `connect()`:
```java
conn.setConnectTimeout(5000);
conn.setReadTimeout(5000);
```

✅ Для **POST** запросов:
```java
conn.setDoOutput(true);
conn.setRequestMethod("POST");

try (OutputStream out = conn.getOutputStream()) {
    out.write("data=test".getBytes(StandardCharsets.UTF_8));
}
```

✅ Проверяй **код ответа** перед чтением содержимого:
```java
int code = conn.getResponseCode();

if (code == HttpURLConnection.HTTP_OK) {
    try (BufferedReader in = new BufferedReader(
	    new InputStreamReader(conn.getInputStream()))
	) {
        ...
    }
}
```

✅ Всегда **закрывай соединение** после использования (`conn.disconnect()`).

✅ ❗❗Для HTTPS — при необходимости добавь собственное управление сертификатами через `HttpsURLConnection`.

## Пример
```java
import java.io.*;
import java.net.*;
import java.nio.charset.StandardCharsets;

public class HttpURLConnectionDemo {
    public static void main(String[] args) {
        try {
            URL url = new URL("https://httpbin.org/post");
            
            HttpURLConnection conn = 
	            (HttpURLConnection) url.openConnection();

            // Настройка соединения
            conn.setRequestMethod("POST");
            conn.setConnectTimeout(5000);
            conn.setReadTimeout(5000);
            conn.setDoOutput(true); // разрешаем отправку данных
            conn.setRequestProperty("Content-Type", "application/x-www-form-urlencoded");

            // Отправляем тело запроса
            String params = "name=Seriy&lang=Java";
            try (OutputStream os = conn.getOutputStream()) {
                os.write(params.getBytes(StandardCharsets.UTF_8));
            }

            // Читаем ответ
            int code = conn.getResponseCode();
            System.out.println("Код ответа: " + code + " " + conn.getResponseMessage());

            try (BufferedReader reader = new BufferedReader(
                    new InputStreamReader(
	                    conn.getInputStream(), 
	                    StandardCharsets.UTF_8))
            ) {
                System.out.println("\n--- Ответ сервера ---");
                reader.lines().limit(5).forEach(System.out::println);
            }

            conn.disconnect();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
