
> [!NOTE] Интерфейс **`java.net.CookiePolicy`** определяет **правила (ПОЛИТИКУ)**
> согласно которым `CookieManager` решает — **принимать или отклонять cookies**,  полученные от сервера.

## Пару слов о назначении
- `CookiePolicy` служит **фильтром** между сервером и хранилищем cookies.
    
- Он определяет, **допустимо ли сохранить cookie**, исходя из его параметров (`домен`, `путь`, `secure`, и т.д.).
    
- Политика применяется при каждом вызове `CookieManager.put(...)`.
    
- ❗❗❗Java предоставляет **три стандартные реализации**, а также позволяет задать собственную.

## Интерфейс
```java
public interface CookiePolicy {

    // ============ 🔹 Проверка допустимости cookie ============

    //✅ Определяет, можно ли принять cookie от указанного URI.
    // Параметры:
    //  uri — адрес ресурса, от которого пришёл cookie;
    //  cookie — объект HttpCookie, содержащий имя, значение, домен, путь и флаги.
    // Возвращает true, если cookie разрешён для сохранения, иначе false.
    boolean shouldAccept(
	    URI uri, 
	    HttpCookie cookie);
}
```


## 💡 Особенности и подводные камни 💡
1️⃣ ❗❗❗Вызывается при каждом получении cookie — решение принимается индивидуально для каждого cookie.

2️⃣ 💡⭐Если метод `shouldAccept()` возвращает false, cookie не будет сохранён в CookieStore.

3️⃣ 💡Политика ==ПРИМЕНЯЕТСЯ только ПРИ ПРИЁМЕ cookie== (Set-Cookie),
но не влияет на отправку уже сохранённых cookies.

4️⃣ В Java определены ==три стандартные политики==:
- ACCEPT_ALL — принимает все cookies.
- ACCEPT_NONE — отклоняет все cookies.
- ACCEPT_ORIGINAL_SERVER — принимает только те cookies, что принадлежат исходному серверу.

5️⃣ Можно создать собственную политику (например, принимающую cookies только по HTTPS или от конкретных доменов).

6️⃣ ⭐⭐⭐CookiePolicy не хранит состояния — это чистый ФУНКЦИОНАЛЬНЫЙ ИНТЕРФЕЙС,
легко реализуемый через лямбда-выражение.

7️⃣ При использовании сложных фильтров важно учитывать case-insensitive сравнение доменов (RFC 6265).

---
## 💎 Best Practices 💎

✅ **Используй встроенные политики** для простых сценариев:
```java
CookieManager manager = new CookieManager(
	null, 
	CookiePolicy.ACCEPT_ORIGINAL_SERVER);
```

✅ **Для REST-клиентов и тестов — принимай все cookies:**
```java
CookieManager manager = new CookieManager(
	null, 
	CookiePolicy.ACCEPT_ALL);

```

✅ **Для безопасности — отклоняй все cookies:**
```java
CookieManager manager = new CookieManager(
	null, 
	CookiePolicy.ACCEPT_NONE);
```

✅ **Создавай собственные политики через лямбда-выражения:**
```java
CookiePolicy secureOnly = (uri, cookie) ->
        "https".equalsIgnoreCase(uri.getScheme()) && cookie.getSecure();

```

✅ **Можно комбинировать условия (например, только для нужного домена):**
```java
CookiePolicy myPolicy = (uri, cookie) ->
        cookie.getDomain().endsWith(".mycompany.com") &&
        !cookie.hasExpired();
```

✅ **Назначай политику перед установкой CookieManager как глобального:**
```java
CookieHandler.setDefault(new CookieManager(null, myPolicy));
```

## Пример
```java
import java.net.*;
import java.io.*;

public class CookiePolicyDemo {
    public static void main(String[] args) throws Exception {
        // 1️⃣ Определяем собственную политику:
        // принимать только cookies от HTTPS и домена .httpbin.org
        CookiePolicy customPolicy = (uri, cookie) ->
                "https".equalsIgnoreCase(uri.getScheme()) &&
                cookie.getDomain().endsWith(".httpbin.org");

        // 2️⃣ Создаём CookieManager с нашей политикой
        CookieManager manager = new CookieManager(null, customPolicy);
        CookieHandler.setDefault(manager);

        // 3️⃣ Делаем запрос, сервер попытается установить cookie
        URL url = new URL("https://httpbin.org/cookies/set?token=abc123");
        try (InputStream in = url.openStream()) {
            in.readAllBytes();
        }

        // 4️⃣ Проверяем, какие cookies были приняты
        System.out.println("--- Сохранённые cookies ---");
        CookieStore store = manager.getCookieStore();
        for (HttpCookie cookie : store.getCookies())
            System.out.println(cookie);

        // 5️⃣ Проверяем, работает ли политика
        System.out.println("\nПолитика приняла cookie? " +
                (store.getCookies().isEmpty() ? "Нет ❌" : "Да ✅"));
    }
}

```
