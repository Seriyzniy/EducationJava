
> [!NOTE] ⭐Если `модифицировать коллекцию не нужно`, `тогда` лучше использовать цикл `for-each`⭐
> 

> [!NOTE] ⭐Для работы с циклом `for-each` НЕОБХОДИМО реализовать `Iterable`⭐
> Цикл for может цикли чески проходить по коллекции объектов, реализующей интерфейс `Iterable`. 
> 
> 🧠Поскольку интерфейс `Iterable` реализуют все классы коллекций, все они способны работать с циклом `for`.

```java
import java.util.*;

public class ForEachDemo {
    public static void main(String[] args) {
        List<String> langs = Arrays.asList("Java", "C++", "Python", "Go");

        // ✅Итерация через for-each
        for (String lang : langs) {
            System.out.println(lang);
        }
    }
}

```
