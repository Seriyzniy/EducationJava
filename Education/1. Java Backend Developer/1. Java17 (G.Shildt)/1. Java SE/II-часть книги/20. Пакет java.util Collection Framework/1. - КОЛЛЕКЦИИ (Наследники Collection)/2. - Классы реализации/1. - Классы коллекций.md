
> [!NOTE] 🧠Коллекции `могут работать только с классами`, (для примитивов используй обертки)🧠
> 

Теперь, когда вы знакомы с интерфейсами коллекций, можно приступать к изучению стандартных классов, которые их реализуют.

==Некоторые классы предоставляют полные реализации==, которые можно использовать в том виде как есть. 
==Другие являются абстрактными, обеспечивая базовые реализации==, которые применяются в качестве отправных точек для создания конкретных коллекций.

> [!NOTE] ⭐Классические коллекции `НЕ СИНХРОНИЗОВАННЫ`, но можно получить их синхронизованные версии⭐
> 

| Класс                    | Наследование и назначение                                                                                                             |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------- |
| `AbstractCollection`     | Реализует большую часть интерфейса ``Collection``                                                                                     |
| `AbstractList`           | Расширяет `AbstractCollection` и реализует большую часть интерфейса `List`                                                            |
| `AbstractQueue`          | Расширяет `AbstractCollection` и реализует часть интерфейса `Queue`                                                                   |
| `AbstractSequentialList` | Расширяет `AbstractList` для использования коллекцией, которая применяет последовательный, а не произвольный доступ к своим элементам |
| `AbstractSet`            | Расширяет `AbstractCollection` и реализует большую часть интерфейса `Set`                                                             |
|                          |                                                                                                                                       |
| `LinkedList`             | Реализует связный список, расширяя AbstractSequentialList                                                                             |
| `ArrayList`              | Реализует динамический массив, расширяя AbstractList                                                                                  |
| `ArrayDeque`             | Реализует динамическую двустороннюю очередь, расширяя AbstractCollection и реализуя интерфейс Deque                                   |
| `EnumSet`                | Расширяет AbstractSet для использования с элементами перечислений                                                                     |
| `HashSet`                | Расширяет AbstractSet для применения с хеш-таблицей                                                                                   |
| `LinkedHashSet`          | Расширяет HashSet, чтобы сделать возможной итерацию в порядке вставки                                                                 |
| `PriorityQueue`          | Расширяет AbstractQueue для поддержки очереди на основе приоритетов                                                                   |
| `TreeSet`                | Реализует набор, хранящийся в дереве. Расширяет AbstractSet                                                                           |
