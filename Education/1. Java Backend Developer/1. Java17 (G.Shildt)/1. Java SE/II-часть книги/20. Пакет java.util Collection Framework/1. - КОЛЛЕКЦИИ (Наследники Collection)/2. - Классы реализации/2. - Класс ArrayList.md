
> [!NOTE] Класс `ArrayList` - `Динамический МАССИВ`
> Класс `ArrayList` является обобщенным классом, который расширяет `AbstractList` и реализует интерфейс `List`. 

Вот его объявление
```java
class ArrayList<E>
```

## Ключевые характеристики:
- **==Порядок элементов сохраняется==** (индексированный доступ).
    
- **Допускает дубликаты**.
    
- **Не синхронизирован** (потокобезопасность нужно обеспечивать вручную).
    
- 💡Амортизированная сложность:
    - `get(index)` — **O(1)** (прямой доступ к массиву),
        
    - `add(element)` в конец — амортизированно **O(1)**, но при расширении массива — **O(n)** (создается новый массив, и в него копируется часть старого + вставляется новый элемент),
        
    - вставка/удаление в середину — **O(n)** (сдвиги элементов).

***
## Конструкторы
```java
// Создаёт пустой список с начальной ёмкостью = 10
public ArrayList()

// 💡Создаёт список с заданной начальной ёмкостью
// initialCapacity — положительное число, задающее размер внутреннего массива.
// Если 0 — массив создаётся при первом добавлении.
public ArrayList(int initialCapacity)

// ⭐Создаёт список, содержащий элементы из другой коллекции.
// Элементы копируются в новый массив в порядке итерации коллекции.
public ArrayList(Collection<? extends E> c)
```

## Методы
```java
// ✅ДОБАВЛЕНИЕ✅
// Добавляет элемент в конец списка.
// Всегда возвращает true (так как список динамический).
public boolean add(E e)

// Вставляет элемент по индексу.
// index — должен быть в диапазоне [0, size].
// Все элементы справа сдвигаются на 1.
public void add(int index, E element)

// Добавляет все элементы из коллекции c в конец списка.
// Возвращает true, если список изменился.
public boolean addAll(Collection<? extends E> c)

// Вставляет все элементы из коллекции c, начиная с позиции index.
// Возвращает true, если список изменился.
public boolean addAll(int index, Collection<? extends E> c)

// ✅УДАЛЕНИЕ✅
// Удаляет элемент по индексу, возвращает удалённый объект.
// Сдвигает элементы справа на 1.
// O(n).
public E remove(int index)

// Удаляет первый встреченный элемент равный o (equals).
// Возвращает true, если элемент был найден и удалён.
public boolean remove(Object o)

// Удаляет все элементы, содержащиеся в коллекции c.
// Возвращает true, если список изменился.
public boolean removeAll(Collection<?> c)

// Удаляет элементы, не содержащиеся в коллекции c.
// Возвращает true, если список изменился.
public boolean retainAll(Collection<?> c)

// ✅ОЧИСТКА✅
// Очищает список (размер = 0, но массив остаётся).
public void clear()

// ✅ПОЛУЧЕНИЕ И ЗАМЕНА✅
// Возвращает элемент по индексу.
// O(1).
public E get(int index)

// Заменяет элемент по индексу на указанный.
// Возвращает старый элемент.
public E set(int index, E element)

// ✅Информация о массиве✅
// Возвращает количество элементов.
public int size()

// Проверяет, пустой ли список (size == 0).
public boolean isEmpty()

// Проверяет, содержится ли элемент o в списке.
public boolean contains(Object o)

// Возвращает индекс первого вхождения элемента o или -1, если нет.
public int indexOf(Object o)

// Возвращает индекс последнего вхождения элемента o или -1, если нет.
public int lastIndexOf(Object o)

// Возвращает массив Object[] со всеми элементами списка.
public Object[] toArray()

// Возвращает массив T[] с элементами списка.
// Если переданный массив меньше списка — создаётся новый.
public <T> T[] toArray(T[] a)

// ✅ИТЕРАТОРЫ✅
// Возвращает итератор по элементам.
public Iterator<E> iterator()

// Возвращает ListIterator (можно перемещаться вперёд и назад).
public ListIterator<E> listIterator()

// Возвращает ListIterator, начиная с позиции index.
public ListIterator<E> listIterator(int index)

// Возвращает подсписок [fromIndex, toIndex).
// backed — изменения в subList влияют на основной список.
public List<E> subList(int fromIndex, int toIndex)

// Возвращает копию списка в виде "потока" Stream API.
public Spliterator<E> spliterator()

// ❗❗❗ Увеличивает внутренний массив до minCapacity. ❗❗❗
// Обычно вызывать вручную не нужно.
public void ensureCapacity(int minCapacity)

// ❗❗❗ Сжимает внутренний массив до фактического размера (size).❗❗❗
public void trimToSize()
```

## 💡Особенности и подводные камни💡

1. 🔶 **Увеличение размера массива**
    - При нехватке места создаётся **новый массив в 1.5 раза больше старого**.
        
    - Старый массив копируется → операция дорогая (**O(n)**).
        
    - Поэтому частые `add()` при маленькой начальной ёмкости могут быть неэффективны.
        
2. 🔶**Удаление и вставка в середину**
    - Нужно сдвигать элементы → **O(n)**.
        
    - Для частых вставок лучше использовать `LinkedList`.
        
3. 🔶**subList() — опасная ловушка**
    - Это **view**, а не копия.
        
    - Изменения в `subList` отражаются в исходном списке и наоборот.
    - Попытка структурно модифицировать исходный список напрямую после создания `subList` приведёт к `ConcurrentModificationException`.
        
4. 🔶**toArray(T[] a)**
    - Если переданный массив меньше списка → создаётся новый.
    - Если больше → "лишние" элементы массива обнуляются (`null`).
        
5. 🔶**Потокобезопасность**
    - `ArrayList` **НЕ синхронизирован**.
        
    - Для многопоточности нужно использовать:
        - `Collections.synchronizedList(new ArrayList<>())`
        - или `CopyOnWriteArrayList`.

***
## ✅ Best Practices ✅

- 💡Если заранее известен **размер списка** → задавай `initialCapacity`, чтобы избежать пересозданий массива.
    
- Для **частого удаления/вставки в середину** → используй `LinkedList`.
    
- Если список используется в **многопоточке** → оборачивай в `Collections.synchronizedList` или бери `CopyOnWriteArrayList`.
    
- 💡Для **постепенного роста** лучше `addAll()`, чем множество отдельных `add()`.
    
- 💡Для **работы с подмножеством** — копируй через `new ArrayList<>(list.subList(...))`, чтобы избежать проблем с view.

## Пример
```java
import java.util.*;

public class ArrayListExample {
    public static void main(String[] args) {
        // Создание с указанием начальной ёмкости
        ArrayList<String> list = new ArrayList<>(5);

        // Добавление элементов
        list.add("Java");
        list.add("C++");
        list.add("Python");

        // Вставка по индексу
        list.add(1, "Rust");

        // Замена
        list.set(2, "Go");

        // Получение
        System.out.println("Элемент [2]: " + list.get(2)); // Go

        // Проверка наличия
        System.out.println("Есть ли Rust? " + list.contains("Rust"));

        // Удаление
        list.remove("C++");
        list.remove(0); // удалит Java

        // Преобразование в массив
        String[] arr = list.toArray(new String[0]);

        // Итерация
        for (String lang : list) {
            System.out.println(lang);
        }

        // SubList
        List<String> sub = list.subList(0, 1);
        System.out.println("subList: " + sub);
    }
}
```
