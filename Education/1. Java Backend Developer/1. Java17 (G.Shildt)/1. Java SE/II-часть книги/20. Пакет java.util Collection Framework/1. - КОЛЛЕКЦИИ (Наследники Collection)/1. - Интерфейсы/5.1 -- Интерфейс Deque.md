
> [!NOTE] Интерфейс `Deque` - `двусторонняя очередь, либо FIFO, либо LIFO`
>  Расширяет `Queue` и обеспечивает поведение двусторонней очереди. 
>  
>  Двусторонние очереди могут функционировать как стандартные очереди "первым пришел - первым обслужен" или как стеки "последний пришел - первым обслужен"

`Deque` - обобщенный интерфейс со следующим объявлением: 
```java
interface Deque<E>
```

## Очередь может быть ограниченной
Реализация интерфейса `Deque` может быть ограниченной по емкости, т.е. в двустороннюю очередь разрешается добавлять лимитированное количество элементов. 

В таком случае попытка добавления элемента в очередь может потерпеть неудачу. 

Интерфейс Deque позволяет обрабатывать такой отказ двумя способами. 
1) Во-первых, методы вроде `addFirst()` и `addLast ()` ==генерируют исключение `IllegalStateException`==, если двусторонняя очередь с ограниченной емкостью заполнена. 
2) Во-вторых, методы наподобие `offerFirst ()` и `offerLast ()` ==возвращают false==, если элемент не может быть добавлен.

## Методы
Вдобавок к методам, унаследованным от `Queue`, в интерфейсе `Deque` ==объявлены методы==

```java
// ИНТЕРФЕЙС DEQUE<E>

// ✅ ДОБАВЛЕНИЕ ЭЛЕМЕНТОВ В НАЧАЛО ✅
/**
 * Добавляет obj в голову двусторонней очереди.
 * Генерирует исключение IllegalStateException, если в двусторонней очереди 
 * с ограниченной емкостью отсутствует свободное пространство.
 */
void addFirst(E obj);

/**
 * Предпринимает попытку добавления obj в голову двусторонней очереди.
 * Возвращает true, если obj был добавлен, или false в противном случае.
 * Таким образом, этот метод возвращает false при попытке добавить obj 
 * в переполненную очередь с ограниченной емкостью.
 */
boolean offerFirst(E obj);

/**
 * Добавляет obj в голову двусторонней очереди.
 * Генерирует исключение IllegalStateException, если в двусторонней очереди 
 * с ограниченной емкостью отсутствует свободное пространство.
 * (Эквивалентен addFirst(), предоставлен для совместимости со Stack)
 */
void push(E obj);



// ✅ ДОБАВЛЕНИЕ ЭЛЕМЕНТОВ В КОНЕЦ ✅
/**
 * Добавляет obj в хвост двусторонней очереди.
 * Генерирует исключение IllegalStateException, если в двусторонней очереди 
 * с ограниченной емкостью отсутствует свободное пространство.
 */
void addLast(E obj);

/**
 * Предпринимает попытку добавления obj в хвост двусторонней очереди.
 * Возвращает true, если obj был добавлен, или false в противном случае.
 */
boolean offerLast(E obj);



// ✅ ПРОСМОТР ЭЛЕМЕНТОВ ИЗ НАЧАЛА ✅
/**
 * Возвращает первый элемент из двусторонней очереди, не удаляя его.
 * Если двусторонняя очередь пуста, тогда генерируется исключение NoSuchElementException.
 */
E getFirst();

/**
 * Возвращает элемент из головы двусторонней очереди, не удаляя его.
 * Если двусторонняя очередь пуста, тогда возвращается null.
 */
E peekFirst();



// ✅ ПРОСМОТР ЭЛЕМЕНТОВ ИЗ КОНЦА ✅
/**
 * Возвращает последний элемент из двусторонней очереди, не удаляя его.
 * Если двусторонняя очередь пуста, тогда генерируется исключение NoSuchElementException.
 */
E getLast();

/**
 * Возвращает элемент из хвоста двусторонней очереди, не удаляя его.
 * Если двусторонняя очередь пуста, тогда возвращается null.
 */
E peekLast();



// ✅ ИЗВЛЕЧЕНИЕ ИЗ НАЧАЛА ✅
/**
 * Возвращает элемент из головы двусторонней очереди, в процессе удаляя его.
 * Если двусторонняя очередь пуста, тогда генерируется исключение NoSuchElementException.
 */
E removeFirst();

/**
 * Возвращает элемент из головы двусторонней очереди, в процессе удаляя его.
 * Если двусторонняя очередь пуста, тогда возвращается null.
 */
E pollFirst();

/**
 * Возвращает элемент из головы двусторонней очереди, в процессе удаляя его.
 * Если двусторонняя очередь пуста, тогда генерируется исключение NoSuchElementException.
 * (Эквивалентен removeFirst(), предоставлен для совместимости со Stack)
 */
E pop();



// ✅ ИЗВЛЕЧЕНИЕ ИЗ КОНЦА ✅
/**
 * Возвращает элемент из хвоста двусторонней очереди, в процессе удаляя его.
 * Если двусторонняя очередь пуста, тогда генерируется исключение NoSuchElementException.
 */
E removeLast();

/**
 * Возвращает элемент из хвоста двусторонней очереди, в процессе удаляя его.
 * Если двусторонняя очередь пуста, тогда возвращается null.
 */
E pollLast();



// ✅ УДАЛЕНИЕ ПО ЗНАЧЕНИЮ ✅
/**
 * Удаляет первое вхождение obj из двусторонней очереди.
 * Возвращает true в случае успеха и false, если двусторонняя очередь не содержит obj.
 */
boolean removeFirstOccurrence(Object obj);

/**
 * Удаляет последнее вхождение obj из двусторонней очереди.
 * Возвращает true в случае успеха и false, если двусторонняя очередь не содержит obj.
 */
boolean removeLastOccurrence(Object obj);



// ✅ ОБРАТНЫЙ ИТЕРАТОР ✅
/**
 * Возвращает итератор, который перемещается от хвоста к голове двусторонней очереди.
 * Другими словами, возвращается обратный итератор.
 */
Iterator<E> descendingIterator();
```
