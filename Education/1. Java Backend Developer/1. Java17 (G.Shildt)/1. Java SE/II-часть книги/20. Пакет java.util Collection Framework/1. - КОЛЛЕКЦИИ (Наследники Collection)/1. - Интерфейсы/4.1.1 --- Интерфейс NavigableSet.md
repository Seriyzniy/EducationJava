
> [!NOTE] Интерфейс `NavigableSet` - поддержка `извлечения элементов на основе совпадения` 
> Интерфейс `NavigableSet` расширяет `SortedSet` и обеспечивает поведение коллекции, которая поддерживает извлечение элементов на основе наиболее точного совпадения с заданным значением или значениями

NavigableSet обобщенный интерфейс с показанным ниже объявлением:
```java
interface NavigableSet<E>
```

Вдобавок к методам, унаследованным от `SortedSet`, в интерфейсе `NavigableSet` объявлены методы
## Методы

```java
// ИНТЕРФЕЙС NAVIGABLESET<E>

// ✅ ПОИСК ЭЛЕМЕНТОВ ✅
/**
 * Ищет внутри набора наименьший элемент e, который больше или равен obj.
 * Если такой элемент найден, то он возвращается. В противном случае возвращается null.
 */
E ceiling(E obj);

/**
 * Ищет внутри набора наибольший элемент e, который меньше или равен obj.
 * Если такой элемент найден, то он возвращается. В противном случае возвращается null.
 */
E floor(E obj);

/**
 * Ищет внутри набора наименьший элемент e, который больше obj.
 * Если такой элемент найден, то он возвращается. В противном случае возвращается null.
 */
E higher(E obj);

/**
 * Ищет внутри набора наибольший элемент e, который меньше obj.
 * Если такой элемент найден, то он возвращается. В противном случае возвращается null.
 */
E lower(E obj);



// ✅ ОБРАТНЫЙ ПОРЯДОК И ИТЕРАТОРЫ ✅
/**
 * Возвращает итератор, который перемещается от наибольшего элемента к наименьшему.
 * Другими словами, возвращается обратный итератор.
 */
Iterator<E> descendingIterator();

/**
 * Возвращает объект NavigableSet, противоположный вызывающему набору.
 * Результирующий набор поддерживается в актуальном состоянии вызывающим набором.
 */
NavigableSet<E> descendingSet();



// ✅ ИЗВЛЕЧЕНИЕ И УДАЛЕНИЕ ЭЛЕМЕНТОВ ✅
/**
 * Возвращает первый элемент, в процессе удаляя его.
 * Поскольку набор отсортирован, это будет элемент с наименьшим значением.
 * В случае пустого набора возвращается null.
 */
E pollFirst();

/**
 * Возвращает последний элемент, в процессе удаляя его.
 * Поскольку набор отсортирован, это будет элемент с наибольшим значением.
 * В случае пустого набора возвращается null.
 */
E pollLast();



// ✅ ПОЛУЧЕНИЕ ПОДМНОЖЕСТВ ✅
/**
 * Возвращает объект NavigableSet, содержащий все элементы из вызывающего набора, 
 * которые меньше upperBound. Если incl равно true, тогда включается элемент, равный upperBound.
 * Результирующий набор поддерживается в актуальном состоянии вызывающим набором.
 */
NavigableSet<E> headSet(E upperBound, boolean incl);

/**
 * Возвращает объект NavigableSet, содержащий все элементы из вызывающего набора,
 * которые больше lowerBound и меньше upperBound.
 * Если lowIncl равно true, тогда включается элемент, равный lowerBound.
 * Если highIncl равно true, тогда включается элемент, равный upperBound.
 * Результирующий набор поддерживается в актуальном состоянии вызывающим набором.
 */
NavigableSet<E> subSet(E lowerBound, boolean lowIncl, E upperBound, boolean highIncl);

/**
 * Возвращает объект NavigableSet, содержащий все элементы из вызывающего набора, которые больше lowerBound. Если incl равно true, тогда включается элемент, равный lowerBound.
 * Результирующий набор поддерживается в актуальном состоянии вызывающим набором.
 */
NavigableSet<E> tailSet(E lowerBound, boolean incl);
```
