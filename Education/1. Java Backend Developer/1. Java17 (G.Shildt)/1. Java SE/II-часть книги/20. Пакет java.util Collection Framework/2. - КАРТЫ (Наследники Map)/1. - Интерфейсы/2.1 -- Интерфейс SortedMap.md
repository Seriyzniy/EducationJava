
> [!NOTE] Интерфейс `SortedMap` расширяет `Мар`. 
> Он гарантирует, что э==лементы поддерживаются в возрастающем порядке== `на основе ключей`

В `К` указывается тип ключей, а в `V` - тип значений.

```java
public interface SortedMap<K, V> extends Map<K, V> {

    // ✅Основные особенности 
    Comparator<? super K> comparator();
    // Возвращает компаратор, по которому упорядочены ключи.
    // Если null → значит используется "естественный порядок" ключей (Comparable).


    // ✅Доступ к частям карты 
    SortedMap<K, V> subMap(K fromKey, K toKey);
    // Возвращает "подкарту" от fromKey (включительно) до toKey (исключительно).

    SortedMap<K, V> headMap(K toKey);
    // Возвращает все пары с ключами МЕНЬШЕ toKey.

    SortedMap<K, V> tailMap(K fromKey);
    // Возвращает все пары с ключами БОЛЬШЕ или РАВНО fromKey.


    // ✅Получение границ 
    K firstKey();
    // Возвращает минимальный ключ (первый по порядку).

    K lastKey();
    // Возвращает максимальный ключ (последний по порядку).


    // ✅Множество ключей/пар 
    Set<K> keySet();
    // Упорядоченное множество ключей.

    Collection<V> values();
    // Коллекция значений (в порядке ключей).

    Set<Map.Entry<K, V>> entrySet();
    // Упорядоченное множество пар (ключ-значение).
}
```

## 💡Особенности💡

1. 🔶`SortedMap` хранит ==пары в отсортированном порядке ключей==.
    - либо по **естественному порядку** (`Comparable` у ключей, напр. `String`, `Integer`),
    - либо по заданному **Comparator**.
        
2. 🔶Все "подкарты" (`subMap`, `headMap`, `tailMap`) — это **живые представления** карты.  
    💡Если изменить подкарту → изменится и оригинальная карта.
    
3. 🔶💡Реализации:
    - 💡 `TreeMap` — главная реализация `SortedMap`.
    - 💡 `ConcurrentSkipListMap` — потокобезопасная альтернатива (на skip-list).

***
## Пример
```java
SortedMap<Integer, String> map = new TreeMap<>();
map.put(3, "Three");
map.put(1, "One");
map.put(2, "Two");

System.out.println(map); 
// {1=One, 2=Two, 3=Three} — ключи автоматически отсортированы

System.out.println(map.firstKey()); // 1
System.out.println(map.lastKey());  // 3

System.out.println(map.headMap(3)); // {1=One, 2=Two}
System.out.println(map.tailMap(2)); // {2=Two, 3=Three}
System.out.println(map.subMap(1, 3)); // {1=One, 2=Two}
```
