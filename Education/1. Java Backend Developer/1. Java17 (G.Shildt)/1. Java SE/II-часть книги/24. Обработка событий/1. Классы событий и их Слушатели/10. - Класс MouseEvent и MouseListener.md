
> [!NOTE] `MouseEvent` — это событие, генерируемое `при действиях пользователя` с `мышью`:
> перемещение курсора, нажатие, отпускание, щелчки, вход/выход курсора из области компонента, прокрутка колеса и т. д.

## Иерархия
```java
java.lang.Object
   └── java.util.EventObject
        └── java.awt.AWTEvent
             └── java.awt.event.ComponentEvent
                  └── java.awt.event.InputEvent
                       └── java.awt.event.MouseEvent
```
🧠 `MouseEvent` наследует все поля и методы `InputEvent`  
(модификаторы клавиш, отметку времени, флаг `consumed`),  
и добавляет координаты, кнопки, клики и информацию о перемещении.

### 🧭 Назначение
`MouseEvent` сообщает компоненту о **любом взаимодействии с мышью**.  

Он используется с интерфейсами:
```java
public interface MouseListener {
    void mouseClicked(MouseEvent e);
    void mousePressed(MouseEvent e);
    void mouseReleased(MouseEvent e);
    void mouseEntered(MouseEvent e);
    void mouseExited(MouseEvent e);
}

public interface MouseMotionListener {
    void mouseDragged(MouseEvent e);
    void mouseMoved(MouseEvent e);
}

public interface MouseWheelListener {
    void mouseWheelMoved(MouseWheelEvent e);
}
```

> `MouseEvent` — универсальный контейнер данных для всех этих интерфейсов.

## Конструкторы
```java
// Создаёт событие мыши
// src — компонент, над которым произошло событие
// id — тип события (например, MOUSE_PRESSED, MOUSE_MOVED и т.д.)
// when — отметка времени в миллисекундах
// modifiers — битовые флаги активных клавиш/кнопок
// x, y — координаты курсора относительно компонента
// clickCount — количество кликов (1 для одиночного, 2 для двойного)
// popupTrigger — true, если это событие инициирует контекстное меню (обычно правая кнопка)
// button — номер кнопки (BUTTON1, BUTTON2, BUTTON3)
public MouseEvent(
	Component src, 
	int id, 
	long when, 
	int modifiers,
    int x, int y, 
    int clickCount, 
    boolean popupTrigger, 
    int button)

```

### Константы класса
|Константа|Значение|Описание|
|---|---|---|
|`MOUSE_FIRST`|500|Начальный ID событий мыши|
|`MOUSE_CLICKED`|500|Щелчок мыши (нажатие+отпускание)|
|`MOUSE_PRESSED`|501|Нажатие кнопки|
|`MOUSE_RELEASED`|502|Отпускание кнопки|
|`MOUSE_MOVED`|503|Перемещение мыши|
|`MOUSE_ENTERED`|504|Курсор вошёл в компонент|
|`MOUSE_EXITED`|505|Курсор покинул компонент|
|`MOUSE_DRAGGED`|506|Перетаскивание (мышь движется с зажатой кнопкой)|

###  Основные поля

|Поле|Тип|Описание|
|---|---|---|
|`x`, `y`|`int`|Координаты курсора относительно компонента|
|`clickCount`|`int`|Количество щелчков|
|`button`|`int`|Номер кнопки (`BUTTON1`, `BUTTON2`, `BUTTON3`)|
|`popupTrigger`|`boolean`|Является ли событие контекстным (правая кнопка)|

## Методы
Класс `MouseEvent` добавляет множество методов поверх `InputEvent`.

### Координаты курсора
```java
// Возвращает координату X курсора относительно компонента
public int getX()

// Возвращает координату Y курсора относительно компонента
public int getY()

// Возвращает координаты курсора как объект Point
public Point getPoint()

// Устанавливает координаты (редко используется, в основном внутренний метод)
public void translatePoint(int x, int y)
```

### Информация о кликах и кнопках
```java
// Возвращает количество последовательных кликов
public int getClickCount()

// Возвращает номер кнопки, вызвавшей событие
// BUTTON1 (левая), BUTTON2 (средняя), BUTTON3 (правая)
public int getButton()

// Возвращает true, если это событие инициировало всплывающее меню (например, правая кнопка)
public boolean isPopupTrigger()
```

### Отладочная информация
```java
// Возвращает строковое описание события
public String paramString()
```

### Работа с координатами
```java
// Возвращает X-координату на экране
public int getXOnScreen()

// Возвращает Y-координату на экране
public int getYOnScreen()
```
> Эти методы полезны, если нужно вычислить абсолютное положение курсора (например, для popup-меню или drag’n’drop).

## 💡 Особенности и подводные камни 💡

1️⃣ 💡 **`mouseClicked` вызывается только если были подряд `mousePressed` и `mouseReleased` на одной кнопке.**  
💡Если мышь сдвинулась между ними — `mouseClicked` может не сработать.

2️⃣ 💡**`clickCount` можно использовать для обнаружения двойного щелчка.**  
Например, `if (e.getClickCount() == 2)`.

3️⃣ 💡**`isPopupTrigger()` зависит от платформы.**  
На Windows — правая кнопка,  
на macOS — `Ctrl + левая`.  
Поэтому всегда используй `isPopupTrigger()`, а не `button == BUTTON3`.

4️⃣ ⭐**События `mouseMoved` и `mouseDragged` приходят часто (до сотен раз в секунду).**
Их нужно обрабатывать эффективно (например, без тяжёлых операций внутри цикла отрисовки).

5️⃣ **Координаты (`x`, `y`) локальны для компонента.**  
Если нужно глобальное положение курсора — используй `getXOnScreen()` / `getYOnScreen()`.

## 💎 Best Practices 💎

✅ 💡Для перетаскивания (drag) используйте `mousePressed()` → `mouseDragged()` → `mouseReleased()`.

✅ Проверяйте `getClickCount()` для двойных щелчков,  
а не пытайтесь вручную измерять время между кликами.

✅ Для контекстного меню (popup) используйте:
```java
if (e.isPopupTrigger()) 
	showPopup(e.getComponent(), e.getX(), e.getY());
```

✅ Всегда отделяйте `mousePressed` и `mouseReleased`,  
если нужно различать начало и конец действия.

✅ Используй `MouseAdapter`, если нужно переопределить не все методы:
```java
addMouseListener(new MouseAdapter() {
    public void mouseClicked(MouseEvent e) {
        ...
    }
});
```

## Пример
```java
import java.awt.*;
import java.awt.event.*;

public class MouseEventDemo extends Frame implements MouseListener, MouseMotionListener {
    private TextArea info;

    public MouseEventDemo() {
        setLayout(new BorderLayout());
        info = new TextArea();
        add(info, BorderLayout.CENTER);

        info.addMouseListener(this);
        info.addMouseMotionListener(this);

        setSize(400, 300);
        setVisible(true);

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
    }

    @Override
    public void mouseClicked(MouseEvent e) {
        info.append("Щелчок (" + e.getClickCount() + "): " +
                e.getX() + "," + e.getY() + "\n");
    }

    @Override
    public void mousePressed(MouseEvent e) {
        info.append("Нажата кнопка " + e.getButton() + "\n");
    }

    @Override
    public void mouseReleased(MouseEvent e) {
        info.append("Отпущена кнопка " + e.getButton() + "\n");
    }

    @Override
    public void mouseEntered(MouseEvent e) {
        info.append("Курсор вошёл в окно\n");
    }

    @Override
    public void mouseExited(MouseEvent e) {
        info.append("Курсор покинул окно\n");
    }

    @Override
    public void mouseDragged(MouseEvent e) {
        info.append("Перетаскивание: " + e.getX() + "," + e.getY() + "\n");
    }

    @Override
    public void mouseMoved(MouseEvent e) {
        info.append("Курсор: " + e.getX() + "," + e.getY() + "\n");
    }

    public static void main(String[] args) {
        new MouseEventDemo();
    }
}
```
🧠 **Как работает:**
- Движение мыши → `mouseMoved()`
    
- Перетаскивание (при зажатой кнопке) → `mouseDragged()`
    
- Нажатие / отпускание → `mousePressed()` / `mouseReleased()`
    
- Двойной щелчок → `mouseClicked()` с `clickCount == 2`.
