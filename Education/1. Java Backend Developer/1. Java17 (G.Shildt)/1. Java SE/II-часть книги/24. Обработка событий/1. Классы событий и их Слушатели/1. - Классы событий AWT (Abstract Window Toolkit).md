
> [!NOTE] **AWT (Abstract Window Toolkit)** — это `первая библиотека` `для создания графического` пользовательского `интерфейса` (GUI) в Java.  
> Она входит в стандартную библиотеку и предоставляет классы для создания **окон, кнопок, текстовых полей, меню, обработки событий** и т.д.
> 

В основе механизма обработки событий Java лежат классы, представляющие события.

## Иерархия классов событий Java

**==Корневой класс==: EventObject**
- Находится в пакете `java.util`
- Является ==суперклассом для всех событий==
- Конструктор: `EventObject(Object src)` - где `src` объект, генерирующий событие
    
- Методы:
    - `Object getSource()` - возвращает источник события
    - `toString()` - возвращает строковый эквивалент события

## Базовый класс для AWT: AWTEvent

Класс `AWTEvent`, определенный в пакете `java.awt`, представляет собой ==подкласс== `EventObject` и ==выступает в качестве суперкласса== (прямо или косвенно) ==для всех событий==, основанных на AWT

- 💡 Метод `int getID()` - используется для определения типа события


## Часто используемые классы событий в пакете java.awt.event
Каждому классу событий соответствует Lisntrer с таким же именем

Событие: ActionEvent
Прослушивателеь: ActionListner

| Класс события       | Обстоятельства генерации                                                                                                                                                                                      |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ActionEvent**     | Генерируется ==при нажатии кнопки==, двойном щелчке на элементе списка или выборе пункта меню                                                                                                                 |
| **AdjustmentEvent** | Генерируется при манипулировании ==полосой прокрутки==                                                                                                                                                        |
| **ComponentEvent**  | Генерируется, когда ==компонент== `скрывается`, `перемещается`, `изменяет размер` или становится видимым                                                                                                      |
| **ContainerEvent**  | Генерируется, когда ==компонент== `добавляется в контейнер` или `удаляется из него`                                                                                                                           |
| **FocusEvent**      | Генерируется, когда ==компонент== `получает или утрачивает` `фокус ввода с клавиатуры`                                                                                                                        |
| **InputEvent**      | Абстрактный==суперкласс для всех== классов событий, ==связанных с вводом в компонентах==                                                                                                                      |
| **ItemEvent**       | Генерируется, когда совершается ==щелчок на флажке или элементе списка==; <br>также возникает при выборе варианта либо выборе или отмене выбора отмечаемого пункта меню                                       |
| **KeyEvent**        | Генерируется при получении ==ввода с клавиатуры==                                                                                                                                                             |
| **MouseEvent**      | Генерируется ==при перетаскивании и перемещении указателя мыши==, <br><br>а также при щелчке, нажатии и отпускании кнопок мыши; также возникает, когда указатель мыши наводится на компонент или покидает его |
| **MouseWheelEvent** | Генерируется при ==прокручивании колесика мыши==                                                                                                                                                              |
| **TextEvent**       | Генерируется при ==изменении== значения ==в текстовой области== или текстовом поле                                                                                                                            |
| **WindowEvent**     | Генерируется, когда ==окно становится== `активным`, `закрывается`, перестает быть активным, `разворачивается`, `сворачивается`, `открывается` или `закрывается`                                               |
