`LineNumberReader` — это расширение `BufferedReader`, которое ==добавляет поддержку **номеров строк** при чтении== текста.  

> [!NOTE] Отслеживание строк
> Его основное назначение — **читать текст построчно и отслеживать номер текущей строки**.

💡Особенности:
- Наследуется от `BufferedReader`, поэтому всё, что умеет `BufferedReader`, умеет и он.
    
- ✅Автоматически отслеживает номер строки при вызове методов `read()` или `readLine()`.
    
- ✅Используется при обработке файлов, когда важно знать, в какой строке произошла ошибка.
    
- ✅Может работать с любым `Reader` (чаще всего с `FileReader` или `InputStreamReader`).

***
## Конструкторы
```java
// Создаёт LineNumberReader, оборачивая заданный Reader.
// readAheadLimit — размер буфера для буферизации символов.
public LineNumberReader(Reader in)

// Создаёт LineNumberReader с заданным размером буфера.
public LineNumberReader(Reader in, int sz)
```

## Методы
```java
// Читает один символ
public int read() throws IOException

// Читает массив символов
public int read(char[] cbuf, int off, int len) throws IOException

// Читает строку без символа перевода строки
public String readLine() throws IOException

// ✅ РАБОТА С НОМЕРАМИ СТРОК ✅
// Возвращает текущий номер строки (начинается с 0 по умолчанию)
public int getLineNumber()

// Устанавливает номер строки
public void setLineNumber(int lineNumber)


// Пропускает символы
public long skip(long n) throws IOException

// Проверяет, готов ли поток к чтению
public boolean ready() throws IOException

// Закрывает поток
public void close() throws IOException

// Устанавливает маркер позиции в потоке
public void mark(int readAheadLimit) throws IOException

// Проверяет поддержку mark/reset
public boolean markSupported()

// Возвращает в маркер
public void reset() throws IOException
```

***
## 💡 Особенности💡 

- 💡**Нумерация строк** начинается с `0` по умолчанию (можно изменить через `setLineNumber(int)`).
    
- 💡Полезен при парсинге логов, конфигов или любых текстовых файлов, где нужно знать строку ошибки.
    
- Всё, что делает `BufferedReader`, доступно и здесь (включая `readLine()`).
    
- 💡 При чтении через `readLine()` номер строки увеличивается автоматически.
    
- Поддержка `mark`/`reset` наследуется от `BufferedReader`.

***
## Пример
```java
import java.io.*;

public class LineNumberReaderExample {
    public static void main(String[] args) {
        try (LineNumberReader lnr = new LineNumberReader(
                new FileReader("example.txt"))) {

            String line;
            while ((line = lnr.readLine()) != null) {
                System.out.printf("Line %d: %s%n", lnr.getLineNumber(), line);
            }

        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
