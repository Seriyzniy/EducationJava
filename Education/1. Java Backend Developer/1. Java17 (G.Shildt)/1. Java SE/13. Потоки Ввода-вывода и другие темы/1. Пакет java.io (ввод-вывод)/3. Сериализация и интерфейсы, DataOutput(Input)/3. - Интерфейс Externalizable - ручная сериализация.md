
> [!NOTE] `java.io.Externalizable` — это интерфейс, дающий ==**полный контроль над сериализацией**==.
>  В отличие от `Serializable`, где JVM автоматически пишет/читает поля, ==здесь ты обязан сам реализовать методы записи и чтения==.

## Интерфейс
```java
public interface Externalizable extends Serializable {
    /**
     * Вызывается при сериализации
     * @param out поток для записи
     */
    void writeExternal(ObjectOutput out) throws IOException;

    /**
     * Вызывается при десериализации
     * @param in поток для чтения
     */
    void readExternal(ObjectInput in) throws IOException, ClassNotFoundException;
}
```

## 💡 Особенности и подводные камни 💡

1. 💡**Обязательный ==публичный конструктор БЕЗ АРГУМЕНТОВ==.**  
    Иначе при десериализации будет `InvalidClassException`.
    
2. **Ты сам отвечаешь за совместимость.**  
    Если добавил поле — должен учесть это в `writeExternal`/`readExternal`.
    
3. **Производительность.**  
    Externalizable может быть быстрее и компактнее, если грамотно писать/читать только нужные данные.
    
4. **Совместимость.**  
    `serialVersionUID` тоже имеет смысл задавать, иначе можно получить проблемы при изменении класса.


# Best Practices 

- используйте только если реально нужен полный контроль (например, кастомный бинарный формат),
	
- всегда реализуйте пустой конструктор,
	
- внимательно поддерживайте обратную совместимость.

## Пример
```java
import java.io.*;

class Config implements Externalizable {
    private static final long serialVersionUID = 2L;

    private String key;
    private int value;

    // ✅✅обязателен конструктор без аргументов✅✅
    public Config() {}

    public Config(String key, int value) {
        this.key = key;
        this.value = value;
    }

    @Override
    public void writeExternal(ObjectOutput out) throws IOException {
        out.writeUTF(key);
        out.writeInt(value);
    }

    @Override
    public void readExternal(ObjectInput in) throws IOException, ClassNotFoundException {
        key = in.readUTF();
        value = in.readInt();
    }

    @Override
    public String toString() {
        return key + " = " + value;
    }
}

public class ExternalizableExample {
    public static void main(String[] args) throws Exception {
        Config cfg = new Config("threads", 8);

        try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("cfg.ser"))) {
            oos.writeObject(cfg);
        }

        try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream("cfg.ser"))) {
            Config restored = (Config) ois.readObject();
            System.out.println(restored); // threads = 8
        }
    }
}

```
