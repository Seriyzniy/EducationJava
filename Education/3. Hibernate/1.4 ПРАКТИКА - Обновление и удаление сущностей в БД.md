# 1. Обновление записей

Чтобы обновить запись в БД, сначала ее нужно получить при помощи `Hibernate`, и затем использовать метод для обновления записи в БД по `id`

Изменим запись под `id` = 4
![[Pasted image 20241201105426.png]]


> [!NOTE] ОБНОВЛЕНИЕ данных при помощи `Hibernate`
> Для начала получи объект из БД и потом при помощи `SET-метода` изменим имя объекта. И затем выполним транзакцию. ==Hibernate поймет что нужно обновить запись==, ==благодаря связывания объекта и записи БД==
> ```java
> try {
> mySession.beginTransaction();
>
> Person updatePers = mySession.get(Person.class, 4);
> updatePers.setName("Larisa");
>
>mySession.getTransaction().commit();
>}finally {
> 	mySessionFact.close();
>}
> ```

# 2. Удаление записи

Действия похожи что и в предыдущем разделе. Необходимо получить объект из БД, и затем вызвать метод сессии `delete()`

```java
. . .
try {
	mySession.beginTransaction();
	
	Person delPers = mySession.get(Person.class, 4);
	mySession.delete(delPers);
	
	mySession.getTransaction().commit();
}finally {
	mySessionFact.close();
}
. . .
```

# 5. СВЯЗЫВАНИЕ ОБЪЕКТА и ЗАПИСИ в БД

> [!NOTE] Связывание ОБЪЕКТА и ЗАПИСИ после выполнения транзакции
> После того как `Hibernate` выполнит транзакцию `mySession.getTransaction().commit()`,  ЗАПИСЬ и ОБЪЕКТ в Java синхронизируются, и будут иметь одинаковые значения
> 
> При помощи этой особенности, можно после того как выполнится транзакция сразу же получить `id` добавленной записи
