
> [!NOTE] `java.io.Serializable` — это `**маркерный интерфейс**, без методов`
> Его задача: пометить класс как сериализуемый. 
> 
> Тогда `ObjectOutputStream`/`ObjectInputStream` сможет автоматически преобразовать объект в байтовый поток и обратно.

## Интерфейс
```java
public interface Serializable {
    // пусто, это marker interface
}
```

## 💡 Особенности и подводные камни💡
1. **Все нестатические поля сериализуются.**  
    А `static` (классовые) и `transient` (помеченные явно) — НЕТ.
    
2. **`serialVersionUID`.**
    - Это уникальный идентификатор версии класса.
        
    - Если не указать — JVM сгенерирует автоматически (на основе структуры класса). Но при малейшем изменении класса (например, сменили тип поля) десериализация старого объекта сломается с `InvalidClassException`.
        
    - Best practice: **всегда явно указывать**:
```java
private static final long serialVersionUID = 1L;
```

3. ⭐**Конструкторы не вызываются при десериализации.**  
    ==Объект восстанавливается напрямую из байтов==. Это может сломать инварианты. Если нужно, используйте методы `readObject` и `writeObject` для кастомного контроля.
    
4. ⭐**Наследование.**  
    Если родитель не `Serializable`, его поля не сериализуются (но конструктор родителя вызовется при десериализации).
    
5. **Безопасность.**  
    Сериализация/десериализация из недоверенных источников = дыра. Можно получить RCE (remote code execution). В современных системах — всегда фильтруйте классы при десериализации (`ObjectInputFilter`).

***

# Best Practices
- всегда задавайте `serialVersionUID`,
	
- помечайте секретные поля как `transient`,
	
- используйте `readObject`/`writeObject` для контроля,
	
- не доверяйте входящим данным (ObjectInputFilter).

## Пример
```java
import java.io.*;

class User implements Serializable {
    private static final long serialVersionUID = 1L;
    
    private String name;
    private transient String password; // не сериализуем
    
    public User(String name, String password) {
        this.name = name;
        this.password = password;
    }

    @Override
    public String toString() {
        return name + " / " + password;
    }
}

public class SerializableExample {
    public static void main(String[] args) throws Exception {
        User u = new User("Alice", "secret123");

        try (ObjectOutputStream oos = new ObjectOutputStream(
	        new FileOutputStream("user.ser"))
	    ){
		    // Объект записывается в виде потока байт
            oos.writeObject(u);
        }

        try (ObjectInputStream ois = new ObjectInputStream(
	        new FileInputStream("user.ser"))
	    ){
            User restored = (User) ois.readObject();
            System.out.println(restored); // password будет null
        }
    }
}

```
