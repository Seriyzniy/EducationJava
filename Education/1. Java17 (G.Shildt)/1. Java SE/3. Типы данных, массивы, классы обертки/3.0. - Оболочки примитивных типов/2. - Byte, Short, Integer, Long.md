Эти классы предоставляют ==оболочки для целочисленных типов==.

Содержат ==методы для преобразования между строками и числами==, а также для работы в **различных системах счисления.**

Определены следующие константы:
- BYTES - Ширина в байтах значения целочисленного типа
- MIN VALUE - Минимальное значение
- MAX VALUE - Максимальное значение
- SIZE - Ширина в битах значения внутри оболочки
- TYPE - Объект Class для значения типа byte, short, int или long

# 📕 1. Преобразование чисел в строки и обратно

Одной из наиболее распространенных задач программирования является ==преобразование СТРОКОВОГО ПРЕДСТАВЛЕНИЯ числа В его внутренний ДВОИЧНЫЙ ФОРМАТ==

Классы `Byte`, `Short`, `Integer` и `Long` предоставляют СТАТИЧЕСКИЕ методы:
- parseByte(), 
- parseShort(), 
- parseInt() 
- parseLong(). 

Указанные методы ==возвращают эквивалент типа== `byte`, `short`, `int` или `long` числовой строки, с которой они вызываются. 

⭐(Аналогичные методы также существуют для классов Float и DouЫe.)
