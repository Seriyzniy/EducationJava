Класс `Boolean` — оболочка для `boolean`. 

## Константы класса Boolean:

- **`TRUE`** - объект `Boolean`, представляющий значение `true`
    
- **`FALSE`** - объект `Boolean`, представляющий значение `false`
    
- **`TYPE`** - объект `Class` для примитивного типа `boolean`

# Таблица методов
| Метод                                                 | Описание                                                                                                                                                                                                           |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Статические методы**                                |                                                                                                                                                                                                                    |
| `static int compare(boolean b1, boolean b2)`          | Возвращает ноль, если `b1` и `b2` содержат одно и то же значение. Возвращает положительное значение, если `b1` равно `true`, а `b2` - `false`, и отрицательное значение в противном случае                         |
| ⭐`static Boolean getBoolean(String propertyName)`     | Возвращает `true`, если ==системное свойство,== указанное в `propertyName`, равно `true`, или `false` в противном случае                                                                                           |
| `static int hashCode(boolean boolVal)`                | Возвращает хеш-код для `boolVal`                                                                                                                                                                                   |
| `static boolean logicalAnd(boolean op1, boolean op2)` | Выполняет операцию логического И над `op1` и `op2` и возвращает результат                                                                                                                                          |
| `static boolean logicalOr(boolean op1, boolean op2)`  | Выполняет операцию логического ИЛИ над `op1` и `op2` и возвращает результат                                                                                                                                        |
| `static boolean logicalXor(boolean op1, boolean op2)` | Выполняет операцию логического исключающего ИЛИ над `op1` и `op2` и возвращает результат                                                                                                                           |
| `static boolean parseBoolean(String str)`             | Возвращает `true`, если `str` содержит строку `"true"` (регистр символов роли не играет), или `false` в противном случае                                                                                           |
| `static String toString(boolean boolVal)`             | Возвращает строковый эквивалент `boolVal`                                                                                                                                                                          |
| ⭐`static Boolean valueOf(boolean boolVal)`            | Возвращает булевский эквивалент `boolVal`                                                                                                                                                                          |
| ⭐`static Boolean valueOf(String boolString)`          | Возвращает `true`, если `boolString` содержит строку `"true"` (регистр символов роли не играет), или `false` в противном случае                                                                                    |
|                                                       |                                                                                                                                                                                                                    |
| **==Методы экземпляра==**                             |                                                                                                                                                                                                                    |
| `boolean booleanValue()`                              | Возвращает эквивалент `boolean`                                                                                                                                                                                    |
| `int compareTo(Boolean b)`                            | Возвращает ноль, если вызывающий объект и `b` содержат одно и то же значение. Возвращает положительное значение, если вызывающий объект равен `true`, а `b` - `false`, и отрицательное значение в противном случае |
| `boolean equals(Object boolObj)`                      | Возвращает `true`, если вызывающий объект эквивалентен `boolObj`, или `false` в противном случае                                                                                                                   |
| `int hashCode()`                                      | Возвращает хеш-код для вызывающего объекта                                                                                                                                                                         |
| `String toString()`                                   | Возвращает строковый эквивалент вызывающего объекта                                                                                                                                                                |
