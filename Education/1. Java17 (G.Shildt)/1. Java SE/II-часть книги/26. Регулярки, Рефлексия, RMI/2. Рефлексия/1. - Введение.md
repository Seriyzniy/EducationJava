
> [!NOTE] Рефлексия 
> `Способность` программного обеспечения `анализировать себя`, которая обеспечивается пакетом `java.lang.reflect` и элементами в классе `Class`.

Она позволяет ==анализировать программный компонент== и динамически описывать его возможности ==во время выполнения==, а не на этапе компиляции

Скажем, с применением рефлексии можно выяснить, какие методы, конструкторы и поля поддерживает класс.

Рефлексия - важное средство, особенно при работе с компонентами Java Beans. 

---
## Интерфейсы `java.lang.reflect`

| №   | Интерфейс              | Назначение                                                                                                        |
| --- | ---------------------- | ----------------------------------------------------------------------------------------------------------------- |
| 1   | **AnnotatedElement**   | Определяет методы для получения аннотаций, присутствующих в элементе программы (класс, метод, поле и т.д.).       |
| 2   | **GenericArrayType**   | Представляет тип массива, компонентный тип которого — параметризованный тип или переменная типа.                  |
| 3   | **GenericDeclaration** | Описывает элемент, который может объявлять типовые параметры (например, класс, интерфейс, метод или конструктор). |
| 4   | **InvocationHandler**  | Определяет обработчик вызовов методов для динамических прокси-объектов.                                           |
| 5⭐  | **Member**             | Общий интерфейс для всех элементов класса, таких как методы, поля, конструкторы.                                  |
| 6   | **ParameterizedType**  | Представляет параметризованный тип, например `List<String>`.                                                      |
| 7   | **Type**               | Базовый интерфейс для всех типов, включая классы, параметризованные типы, массивы и т.д.                          |
| 8   | **TypeVariable**       | Представляет переменную типа (например, `<T>` в `class Box<T>`).                                                  |
| 9   | **WildcardType**       | Представляет тип с подстановочным знаком (`? extends Number`, `? super String`).                                  |

## Классы `java.lang.reflect`
| №   | Класс                | Назначение                                                                                                                         |
| --- | -------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| 1   | **AccessibleObject** | Базовый класс для `Field`, `Method`, `Constructor`; управляет доступом к членам класса (через `setAccessible`).                    |
| 2✅  | **Array**            | Предоставляет статические методы для создания и работы с массивами через рефлексию.                                                |
| 3✅  | **Constructor**      | Представляет конструктор класса и позволяет создавать экземпляры через `newInstance()`.                                            |
| 4   | **Executable**       | Абстрактный суперкласс для `Method` и `Constructor`, описывает общие свойства исполняемых элементов (параметры, аннотации и т.п.). |
| 5✅  | **Field**            | Представляет поле класса и позволяет читать и изменять его значение через рефлексию.                                               |
| 6✅  | **Method**           | Представляет метод класса, позволяет вызывать методы объектов (`invoke()`) и получать метаданные.                                  |
| 7   | **Modifier**         | Содержит статические методы для анализа и интерпретации модификаторов (`public`, `static`, `final` и т.п.).                        |
| 8   | **Parameter**        | Представляет параметр метода или конструктора, включая имя, тип и аннотации.                                                       |
| 9✅  | **Proxy**            | Предоставляет механизмы создания динамических прокси-классов во время выполнения.                                                  |
| 10  | **RecordComponent**  | Представляет компонент записи (`record`), включая его имя, тип и аннотации.                                                        |
| 11  | ReflectPermission    | Делает возможной рефлексию закрытых или защищенных членов класса                                                                   |
