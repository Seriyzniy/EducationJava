## 💡 Подсказки

1️⃣ `Instant` ↔ `ZonedDateTime` — **основная ось преобразований** (через неё идут почти все переходы).  
2️⃣ `LocalDateTime` без зоны нельзя напрямую превратить в `Instant`.
3️⃣ Все классы `java.time` **иммутабельны и потокобезопасны**.

## Таблица: Instant
| Преобразование     | Метод / Способ                              | Пример кода                                                 | Комментарий                                    |
| ------------------ | ------------------------------------------- | ----------------------------------------------------------- | ---------------------------------------------- |
| → `LocalDateTime`  | `LocalDateTime.ofInstant(Instant, ZoneId)`  | `LocalDateTime.ofInstant(inst, ZoneId.systemDefault())`     | Требуется зона, т.к. `Instant` — всегда UTC.   |
| → `ZonedDateTime`  | `ZonedDateTime.ofInstant(Instant, ZoneId)`  | `ZonedDateTime.ofInstant(inst, ZoneId.of("Europe/Moscow"))` | Преобразует в дату/время указанной зоны.       |
| → `OffsetDateTime` | `OffsetDateTime.ofInstant(Instant, ZoneId)` | `OffsetDateTime.ofInstant(inst, ZoneId.of("UTC"))`          | Добавляет смещение к UTC.                      |
| → `LocalDate`      | Через `LocalDateTime`                       | `LocalDateTime.ofInstant(inst, zone).toLocalDate()`         | Вытаскивает только дату.                       |
| → `LocalTime`      | Через `LocalDateTime`                       | `LocalDateTime.ofInstant(inst, zone).toLocalTime()`         | Вытаскивает только время.                      |
| → `long` (millis)  | `toEpochMilli()`                            | `inst.toEpochMilli()`                                       | Возвращает миллисекунды с эпохи.               |
| ← `LocalDateTime`  | `ldt.atZone(zone).toInstant()`              | `ldt.atZone(ZoneId.systemDefault()).toInstant()`            | Получение `Instant` из локальной даты-времени. |
| ← `ZonedDateTime`  | `zdt.toInstant()`                           | `zdt.toInstant()`                                           | Потеря информации о зоне.                      |
| ← `OffsetDateTime` | `odt.toInstant()`                           | `odt.toInstant()`                                           | Преобразует к UTC.                             |
