
> [!NOTE] `FormatStyle` — это `enum`, `определяющий уровень детализации`
> при форматировании даты/времени (аналог констант `DateFormat.FULL`, `LONG`, `MEDIUM`, `SHORT`).

## Значения enum
```java
public enum FormatStyle {
    FULL,    // Полная форма (включает день недели, название месяца и т.д.)
    LONG,    // Длинная форма (только месяц и год словами)
    MEDIUM,  // Средняя форма (сокращенные названия)
    SHORT    // Короткая форма (цифровая)
}
```

### 💬 Использование
Передаётся в фабричные методы `DateTimeFormatter`:
```java
DateTimeFormatter fmt1 = DateTimeFormatter
	.ofLocalizedDate(FormatStyle.FULL);

DateTimeFormatter fmt2 = DateTimeFormatter
	.ofLocalizedDateTime(
		FormatStyle.LONG, 
		FormatStyle.SHORT);
```

## Пример: сравнение стилей
```java
import java.time.*;
import java.time.format.*;
import java.util.*;

public class StylesExample {
    public static void main(String[] args) {
        ZonedDateTime now = ZonedDateTime.now(ZoneId.of("Europe/Moscow"));
        
    	Locale locale = Locale.US;

        System.out.println("FULL:   " + 
        now
        .format(DateTimeFormatter
	        .ofLocalizedDateTime(FormatStyle.FULL).withLocale(locale)));
        
        
        System.out.println("LONG:   " + 
        now
        .format(DateTimeFormatter
	        .ofLocalizedDateTime(FormatStyle.LONG).withLocale(locale)));
        
        
        System.out.println("MEDIUM: " + 
        now
	    .format(DateTimeFormatter
	       .ofLocalizedDateTime(FormatStyle.MEDIUM).withLocale(locale)));
        
        
        System.out.println("SHORT:  " + 
        now
        .format(DateTimeFormatter
	        .ofLocalizedDateTime(FormatStyle.SHORT).withLocale(locale)));
    }
}

LONG:   October 15, 2025, 9:06:07 AM MSK
MEDIUM: Oct 15, 2025, 9:06:07 AM
SHORT:  10/15/25, 9:06 AM

```
