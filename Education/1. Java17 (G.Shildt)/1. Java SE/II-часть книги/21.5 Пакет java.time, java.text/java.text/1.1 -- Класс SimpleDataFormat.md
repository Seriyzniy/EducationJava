
> [!NOTE] `SimpleDateFormat` — конкретная реализация `DateFormat`
> позволяющая `устанавливать` `собственные шаблоны форматирования` (например, `"dd.MM.yyyy HH:mm:ss"`).  
> 
> Он поддерживает гибкий синтаксис шаблонов с символами (`y`, `M`, `d`, `H`, `m`, `s` и др.).

## Конструктор
```java
// Создает объект с шаблоном по умолчанию и текущей локалью.
public SimpleDateFormat()

// Создает форматтер с заданным шаблоном и локалью по умолчанию.
// pattern — строка-шаблон формата (например, "dd.MM.yyyy HH:mm:ss")
public SimpleDateFormat(String pattern)

// Создает форматтер с заданным шаблоном и локалью.
public SimpleDateFormat(String pattern, Locale locale)
```

## Методы
(Наследует все методы `DateFormat`, а также добавляет собственные:)

### Шаблон и локаль
```java
// Возвращает текущий шаблон форматирования.
public String toPattern()

// Задает новый шаблон форматирования.
public void applyPattern(String pattern)

// Возвращает шаблон в "локализованном" виде (учитывает особенности языка).
public String toLocalizedPattern()

// Применяет локализованный шаблон.
public void applyLocalizedPattern(String pattern)
```

### Форматирование и парсинг
```java
// Форматирует дату в строку по текущему шаблону.
@Override
public StringBuffer format(
	Date date, 
	StringBuffer toAppendTo, 
	FieldPosition pos)

// Разбирает строку в объект Date.
@Override
public Date parse(String text, ParsePosition pos)
```

## Символы формата (основные)

|Символ|Описание|
|---|---|
|**a**|AM (до полудня) или PM (после полудня)|
|**d**|День месяца (1-31)|
|**h**|Часы в формате AM/PM (1-12)|
|**k**|Часы в сутках (1-24)|
|**m**|Минуты в часе (0-59)|
|**s**|Секунды в минуте (0-59)|
|**u**|День недели (с понедельником, соответствующим 1)|
|**w**|Неделя в году (1-52)|
|**y**|Год|
|**z**|Часовой пояс|
|**D**|День в году (1-366)|
|**E**|День недели (например, Thursday - четверг)|
|**F**|День недели в месяце|
|**G**|Эра (например, AD - н.э. или BC - до н.э.)|
|**H**|Часы в сутках (0-23)|
|**K**|Часы в формате AM/PM (0-11)|
|**L**|Месяц|
|**M**|Месяц|
|**S**|Миллисекунды в секунде|
|**W**|Неделя в месяце (1-5)|
|**X**|Часовой пояс в формате ISO 8601|
|**Y**|Неделя в году|
|**Z**|Часовой пояс в формате RFC 822|

## 💡 Особенности и подводные камни 💡

1️⃣ `SimpleDateFormat` **не потокобезопасен** — создавайте отдельный экземпляр для каждого потока.  

2️⃣ Символы шаблона чувствительны к регистру (`M` ≠ `m`).  

3️⃣ Если шаблон не совпадает с датой — выбрасывается `ParseException`.  

4️⃣ При `applyLocalizedPattern()` символы могут отличаться в зависимости от локали.  

5️⃣ Форматтер не проверяет существование даты при `lenient=true` (может «исправить» ошибочную дату).

## Пример
```java
import java.text.SimpleDateFormat;
import java.util.Date;

public class Example2 {
    public static void main(String[] args) {
        SimpleDateFormat sdf = new SimpleDateFormat("dd.MM.yyyy HH:mm:ss");

        String formatted = sdf.format(new Date());
        System.out.println("Formatted: " + formatted);

        try {
            Date parsed = sdf.parse("15.10.2025 08:30:45");
            System.out.println("Parsed: " + parsed);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

```
