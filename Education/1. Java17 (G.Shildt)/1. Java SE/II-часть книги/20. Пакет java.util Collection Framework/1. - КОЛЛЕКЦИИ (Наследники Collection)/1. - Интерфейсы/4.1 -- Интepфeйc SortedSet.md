
> [!NOTE] Интерфейс `SortedSet` расширяет `Set` и обеспечивает поведение `набора, ОТСОРТИРОВАННОГО в возрастающем порядке`.
> 

`SortedSet` - обобщенный интерфейс со следующим объявлением:
```JAVA
interface SortedSet<E>
```

В дополнение к методам, предоставляемым `Set`, в интерфейсе `SortedSet` ==объявлены методы==

## Методы

```java
// ИНТЕРФЕЙС SORTEDSET<E>

// ✅ ПОЛУЧЕНИЕ КОМПАРАТОРА ✅
/**
 * Возвращает компаратор вызывающего отсортированного набора.
 * Если для этого набора используется естественное упорядочение, тогда возвращается null.
 */
Comparator<? super E> comparator();



// ✅ ПОЛУЧЕНИЕ КРАЙНИХ ЭЛЕМЕНТОВ ✅
/**
 * Возвращает первый элемент в вызывающем отсортированном наборе.
 */
E first();

/**
 * Возвращает последний элемент в вызывающем отсортированном наборе.
 */
E last();



// ✅ ПОЛУЧЕНИЕ ПОДМНОЖЕСТВ ✅
/**
 * Возвращает объект SortedSet с элементы, предшествующими end, которые содержатся 
 * в вызывающем отсортированном наборе. На элементы в возвращаемом отсортированном 
 * наборе также ссылается и вызывающий отсортированный набор.
 */
SortedSet<E> headSet(E end);

/**
 * Возвращает объект SortedSet, который содержит элементы между start и end-1.
 * На элементы в возвращаемом отсортированном наборе также ссылается и вызывающий 
 * отсортированный набор.
 */
SortedSet<E> subSet(E start, E end);

/**
 * Возвращает объект SortedSet с элементами, начиная с позиции start и до конца, 
 * которые содержатся в вызывающем отсортированном наборе. На элементы в возвращаемом 
 * отсортированном наборе также ссылается и вызывающий отсортированный набор.
 */
SortedSet<E> tailSet(E start);
```
