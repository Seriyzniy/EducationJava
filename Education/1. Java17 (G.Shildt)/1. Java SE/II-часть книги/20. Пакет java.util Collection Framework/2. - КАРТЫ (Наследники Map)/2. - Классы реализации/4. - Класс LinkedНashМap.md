
> [!NOTE] Класс `LinkedHashMap` расширяет `HashМap` 
> `Поддерживает связный список элементов` карты `в` том `порядке`, в котором они были вставлены

Под капотом:
- **хэш-таблица** (как у `HashMap`),
	 - **двусвязный список** для сохранения порядка элементов.

***
## Конструкторы
```java
public LinkedHashMap();  
// Конструктор по умолчанию (capacity = 16, loadFactor = 0.75, порядок вставки)

public LinkedHashMap(int initialCapacity);  
// Указание начальной ёмкости (количество бакетов)

public LinkedHashMap(int initialCapacity, float loadFactor);  
// Указание capacity и loadFactor

public LinkedHashMap(int initialCapacity, float loadFactor, boolean accessOrder);  
// accessOrder = false → порядок вставки
// accessOrder = true  → порядок доступа (LRU-кэш)

public LinkedHashMap(Map<? extends K, ? extends V> m);  
// Создание на основе другой Map (копирование всех элементов)

```

## Методы
`LinkedHashMap` унаследовал все методы `HashMap`, поэтому набор идентичен:

***
## ✅ Best Practices ✅ 
- Использовать `LinkedHashMap`, если важен **порядок элементов** (например, JSON-парсеры, кэш, логирование). 
	
- Для многопоточности → использовать 
	- `Collections.synchronizedMap(new LinkedHashMap<>())`
	- `ConcurrentLinkedHashMap` из сторонних библиотек.

***
## Пример
```java
import java.util.*;

public class LinkedHashMapDemo {
    public static void main(String[] args) {
        // По умолчанию — порядок вставки
        Map<String, Integer> map = new LinkedHashMap<>();
        map.put("Alice", 30);
        map.put("Bob", 25);
        map.put("Charlie", 28);

        System.out.println("Порядок вставки: " + map);

        // LinkedHashMap с порядком доступа (LRU)
        Map<String, Integer> accessOrderMap = new LinkedHashMap<>(16, 0.75f, true);
        accessOrderMap.put("A", 1);
        accessOrderMap.put("B", 2);
        accessOrderMap.put("C", 3);

        // Доступ к элементу "A" перемещает его в конец
        accessOrderMap.get("A");

        System.out.println("Порядок доступа: " + accessOrderMap);

        // Реализация простого LRU-кэша
        LinkedHashMap<Integer, String> lruCache = new LinkedHashMap<>(3, 0.75f, true) {
            @Override
            protected boolean removeEldestEntry(Map.Entry<Integer, String> eldest) {
                return size() > 3; // автоматически удаляем самый старый элемент
            }
        };

        lruCache.put(1, "one");
        lruCache.put(2, "two");
        lruCache.put(3, "three");
        lruCache.get(1);  // "1" становится свежим
        lruCache.put(4, "four"); // удалится "2"

        System.out.println("LRU Cache: " + lruCache);
    }
}

```
