| Критерий                 | `reduce()`                                                            | `collect()`                                                                       |
| ------------------------ | --------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| 💡 Основная идея         | Сведение элементов ==в одно значение== (число, строка, объект и т.п.) | Сбор элементов в ==**мутабельную** структуру== (список, карту, статистику и т.п.) |
| Возвращает               | Одно значение (`T` или `Optional<T>`)                                 | Произвольный результат (`R`), обычно коллекцию                                    |
| Тип операции             | Иммутабельное сведение (без изменения контейнера)                     | Мутабельное накопление (изменение контейнера)                                     |
| Основа                   | Интерфейс `BinaryOperator<T>`                                         | Интерфейс `Collector<T, A, R>`                                                    |
| Поддержка параллельности | Неэффективен для сложных структур                                     | Оптимизирован под `parallelStream()`                                              |
| 🔒 Состояние             | Stateless — не хранит промежуточное состояние                         | Stateful — через аккумулятор (`A`)                                                |
