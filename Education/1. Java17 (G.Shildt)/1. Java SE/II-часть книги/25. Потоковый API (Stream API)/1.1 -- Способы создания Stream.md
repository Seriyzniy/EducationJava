## Категория 1: Потоки из коллекций (Collection)

Самый распространённый способ — через методы интерфейса `Collection`.
```java
Stream<T> stream()
Stream<T> parallelStream()
```
- `stream()` — создаёт **последовательный** поток.
- `parallelStream()` — создаёт **параллельный** поток.

💬 Пример:
```java
List<String> names = List.of("Alice", "Bob", "Charlie");

Stream<String> s1 = names.stream();
Stream<String> s2 = names.parallelStream();
```

## Категория 2: Потоки из массивов (Arrays)

Используется класс `Arrays`:
```java
public static <T> Stream<T> stream(T[] array)
public static <T> Stream<T> stream(
	T[] array, 
	int startInclusive, 
	int endExclusive)
```

Также есть примитивные аналоги:
```java
IntStream stream(int[] array)
LongStream stream(long[] array)
DoubleStream stream(double[] array)
```

💬 Пример:
```java
int[] nums = {1, 2, 3, 4, 5};
IntStream intStream = Arrays.stream(nums);
```

Класс `Stream` (и его примитивные подтипы) 
предоставляют ==фабричные методы== `of(...)`.

```java
Stream<T> Stream.of(T... values)

```

💬 Пример:
```java
Stream<String> stream = Stream.of("A", "B", "C");

IntStream numbers = IntStream.of(10, 20, 30);
```

## Категория 4: Потоки из диапазонов (только для примитивных потоков)

Позволяют создавать диапазоны чисел
```java
IntStream range(int startInclusive, int endExclusive)
IntStream rangeClosed(int startInclusive, int endInclusive)

LongStream range(long startInclusive, long endExclusive)
LongStream rangeClosed(long startInclusive, long endInclusive)
```

rangeClosed - включает последний элемент диапазона

💬 Пример:
```java
IntStream s1 = IntStream.range(0, 5);      // 0,1,2,3,4
IntStream s2 = IntStream.rangeClosed(0, 5); // 0,1,2,3,4,5
```

## Категория 5: Потоки из генераторов
Создают **потенциально бесконечные** потоки
(❗❗Нужно ограничивать через limit())
```java
Stream<T> generate(Supplier<T> supplier)
Stream<T> iterate(T seed, UnaryOperator<T> f)

Stream<T> iterate(
	T seed, 
	Predicate<? super T> hasNext, 
	UnaryOperator<T> next) // Java 9+
```

💬 Примеры:
```java
// Случайные числа
Stream<Double> randoms = Stream.generate(Math::random).limit(5);

// Последовательность чисел
Stream<Integer> seq = Stream.iterate(0, n -> n + 1).limit(10);

// Java 9+: с условием
Stream<Integer> untilTen = Stream
	.iterate(
		0, 
		n -> n < 10, 
		n -> n + 1);
```

## ⭐⭐Категория 6: Потоки из файлов и строк

### 📁 Файлы (`java.nio.file.Files`)
```java
Stream<String> lines(Path path)
Stream<String> lines(Path path, Charset cs)
```
→ поток строк из файла (по строкам).

💬 Пример:
```java
Path path = Path.of("data.txt");

Stream<String> lines = Files.lines(path);
```

### 📄 Строки (`java.util.regex.Pattern`)
```java
Stream<String> lines()            // из многострочного текста

Stream<String> splitAsStream(CharSequence input) // из строки по regex
```

💬 Примеры:
```java
String text = "one\ntwo\nthree";
Stream<String> lines = text.lines(); // "one", "two", "three"

Stream<String> words = Pattern.compile("\\s+").splitAsStream("a b c d");
```

## ⭐⭐Категория 7: Потоки из других API

### 1 `Stream.ofNullable()` — Java 9+
```java
Stream<T> ofNullable(T t)
// Возвращает Stream из одного элемента или пустой Stream, если t == null
```

💬 Пример:
```java
Stream<String> s = Stream.ofNullable(null); // пустой поток
```

### 2 `Files.walk()` и `Files.find()` — обход файловой системы
```java
Stream<Path> walk(Path start, int maxDepth)
Stream<Path> find(
	Path start, 
	int maxDepth, 
	BiPredicate<Path, BasicFileAttributes> matcher)
```

💬 Пример:
```java
Files.walk(Path.of("src"))
     .filter(Files::isRegularFile)
     .forEach(System.out::println);
```

### 3 `BufferedReader.lines()`
```java
Stream<String> lines()
// Возвращает поток строк, считанных из файла или ввода
```

## Категория 8: Преобразование примитивных потоков
```java
IntStream chars()                 // из строки — поток символов

DoubleStream mapToDouble(...)     // преобразование типов

LongStream mapToLong(...)

Stream<T> boxed()                 // примитив → объект
```

💬 Пример:
```java
IntStream chars = "Hello".chars();

Stream<Character> boxed = chars.mapToObj(c -> (char) c);
```
