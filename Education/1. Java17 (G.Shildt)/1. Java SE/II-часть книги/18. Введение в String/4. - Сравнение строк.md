В состав класса `String` входит несколько методов, предназначенных для сравнения строк или подстрок внутри строк

***
# 📕 1. equals() и equalsIgnoreCase()
Для сравнения двух строк  С УЧЕТОМ РЕГИСТРА используйте метод `equals()` 
```java
boolean equals(Object str)
```
- `true`, если строки содержат одни и те же символы в одном и том же порядке, 
- `false` в противном случае. 

Для сравнения двух строк  БЕЗ УЧЕТОМ РЕГИСТРА используйте
```java
boolean equalsIgnoreCase(String str)
```

Пример
```java
String s1 = "Hello";
String s2 = "HELLO";

s1.equals(s2); // false
s1.equalsIgnoreCase(s2); // true
```

# 📕 🧠 2. Метод regionMatches()
Метод `regionMatches()` ==сравнивает определенную область внутри одной строки== с определенной областью ==в другой строке==

```java
boolean regionMatches(int startIndex, //Индекс подстроки вызывающего объекта
					 String str2,     // ВТОРАЯ строка
                     int str2StartIndex, //Индекс подстроки ВТОРОЙ строки
                     int numChars) //Количество сравниваемых символов

// ✅ Перегрузка с флагом, учитывать регистр или нет
boolean regionMatches(boolean ignoreCase, 
					 int startIndex, 
                     String str2, 
                     int str2StartIndex, 
                     int numChars)
```

# 📕 3. Методы startsWith() и endsWith() 
- Метод `startWith()` определяет, ==начинается ли объект== `String` с указанной строки
- Метод `endWith()` выясняет, ==заканчивается ли объект== `String` заданной строкой.
```java
"Foobar".startsWith("Foo"); // true
"Foobar".endsWith("bar"); // true

//Указываем индекс с которой начинается поиск
"Foobar".startsWith("bar", 3); // true
```

***
# 📕 ⭐ 4. Важное различие: equals() vs ==
```java
String s1 = "Hello";
String s2 = new String(s1);

s1.equals(s2); // true ✅(сравнивает содержимое)
s1 == s2; // false ❌❗(сравнивает ссылки)
```

# 📕 5. Метод compareTo()

> [!NOTE] ⭐Больше/Меньше строка другой строки? ⭐(`По таблице ASCII`)
> Одна строка `меньше` другой, если она `расположена раньше` в словарном порядке. 
> Одна строка `больше` другой, если она `расположена позже` в словарном порядке.


```java
int compareTo(String str)
```
- Возвращает:
    - `< 0` - вызывающая строка меньше
        
    - `= 0` - строки равны
        
    - `> 0` - вызывающая строка больше

```java
"abc".compareTo("abd"); // отрицательное число
"abc".compareTo("abc"); // 0
"abd".compareTo("abc"); // положительное число
```
