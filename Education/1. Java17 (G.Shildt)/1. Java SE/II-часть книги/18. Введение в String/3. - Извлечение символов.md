Класс `String` поддерживает несколько способов извлечения символов из объекта `String`

==Символы в объекте String НЕ МОГУТ БЫТЬ ПРОИНДЕКСИРОВАНЫ==, как в обычном массиве, однако многие методы в String используют индексы

***
# 📕 1. Метод charAt()
```java
char ch = "abc".charAt(1); // 'b'
```
- ==Возвращает символ по указанному индексу==
- Индексация с 0

***
# 📕 2. Метод getChars()
```java
void getChars(int sourceStart, int sourceEnd, char[] target, int targetStart)
```
- `sourceStart` - начальный индекс
    
- `sourceEnd` - индекс после последнего символа
    
- `target` - целевой массив
    
- `targetStart` - начальная позиция в целевом массиве


```java
//          01234567891011
String s = "Строка для демонстрации";
char[] buf = new char[4];

s.getChars(11, 15, buf, 0); // "демо"
```

***
# 📕 3. Метод getBytes()
```java
byte[] bytes = "Hello".getBytes();
```
- Преобразует ==строку в массив БАЙТОВ== по кодировке платформы

***
# 📕 4. Метод toCharArray()
```java
char[] chars = "Hello".toCharArray();
```
- Преобразует всю ==строку в массив СИМВОЛОВ==
