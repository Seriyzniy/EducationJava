Класс `ProcessBuilder` ==предоставляет еще один способ запуска процессов== (т.е. программ) ==и управления ими.==

Класс `ProcessBuilder` предлагает ==более высокую степень контроля== над процессами. Например, он разрешает установку текущего рабочего каталога.

В классе ProcessBuilder определены следующие конструкторы: 
```java
ProcessBuilder(List<String> args) 
ProcessBuilder(String . . . args)
```

> [!NOTE] 🧠Запуск ИЗ КОМАНДНОЙ СТРОКИ
> Здесь args - `список аргументов`, которые `задают имя` подлежащей запуску `программы` вместе `со всеми необходимыми аргументами командной строки`

| `ProcessBuilder command(List<String> args)`                                          | Устанавливает имя программы и ее аргументы так, как указано в `args`. Изменения в этом списке влияют на вызывающий объект. Возвращает ссылку на вызывающий объект                                                                                             |
| `ProcessBuilder command(String... args)`                                             | Устанавливает имя программы и ее аргументы так, как указано в `args`. Возвращает ссылку на вызывающий объект                                                                                                                                                  |
|                                                                                      |                                                                                                                                                                                                                                                               |
| **Методы ==работы с каталогом==**                                                    |                                                                                                                                                                                                                                                               |
| `File directory()`                                                                   | Возвращает текущий рабочий каталог вызывающего объекта. Возвращаемое значение будет равно `null`, если каталог совпадает с каталогом программы на Java, которая запустила процесс                                                                             |
| `ProcessBuilder directory(File dir)`                                                 | Устанавливает текущий рабочий каталог вызывающего объекта. Возвращает ссылку на вызывающий объект                                                                                                                                                             |
|                                                                                      |                                                                                                                                                                                                                                                               |
| **Методы ==работы с окружением==**                                                   |                                                                                                                                                                                                                                                               |
| `Map<String, String> environment()`                                                  | Возвращает переменные среды, ассоциированные с вызывающим объектом, в виде пар "ключ-значение"                                                                                                                                                                |
|                                                                                      |                                                                                                                                                                                                                                                               |

| `ProcessBuilder redirectOutput(ProcessBuilder.Redirect target)`                      | Устанавливает цель для стандартного потока вывода как указано в `target`. Возвращает ссылку на вызывающий объект                                                                                                                                              |
|                                                                                      |                                                                                                                                                                                                                                                               |

Если значение `merge` равно `false`, то потоки являются отдельными — состояние, принятое по умолчанию. Возвращает ссылку на вызывающий объект |

| `static List<Process> startPipeline(List<ProcessBuilder> pbList) throws IOException` | Направляет процессы по конвейеру в `pbList`                                                                                                                                                                                                                   |
