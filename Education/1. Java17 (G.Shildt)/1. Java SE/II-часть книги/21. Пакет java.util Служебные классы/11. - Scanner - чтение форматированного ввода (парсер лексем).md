==Читает и разбирает текстовый ввод== (из `InputStream`, `File`, `Path`, `Readable`, `String`), выделяет лексемы по разделителям и может автоматически парсить числа, булевы, строки, BigInteger/BigDecimal и т.д. 

Очень удобен для CLI и простого парсинга.


> [!NOTE] ❗❗Подводные камни: ❗❗
> Всегда проверяйте `hasNextX()` перед `nextX()` во избежание `InputMismatchException` или `NoSuchElementException`.
> 

## Порядок работы
Как правило, для использования `Scanner` необходимо придерживаться описанной ниже процедуры. 
1. Выяснить, доступен ли конкретный вид входных данных, вызвав один из методов `hasNextX ()` класса `Scanner`, rде `Х` - желаемый тип данных. 
2. Если входные данные доступны, тогда прочитать их, вызвав один из методов `nextX()` класса `Scanner`. 
3. Повторять процесс до тех пор, пока входные данные не будут исчерпаны. 
4. Закрьrrь объект `Scanner`, вызвав метод `close()`

## Конструкторы
```java
Scanner(File source)
// Чтение из файла.

Scanner(InputStream source)
// Чтение из потока (например, System.in).

Scanner(Path source) throws IOException
// Чтение из файла по пути.

Scanner(Readable source)
// Чтение из любого Readable (например, StringReader).

Scanner(ReadableByteChannel source)
// Чтение из канала.

Scanner(String source)
// Чтение из строки.
```

## Методы
```java
boolean hasNext()
// Проверяет, есть ли ещё токены.


boolean hasNextInt()
// Проверяет, есть ли следующий токен типа int.
int nextInt()
// Возвращает следующий токен как int.


boolean hasNextDouble()
// Проверяет, есть ли следующий токен типа double
double nextDouble()
// Аналогично для double.


String next()
// Возвращает следующий токен как String.
String nextLine()
// Возвращает всю строку.



String findInLine(String pattern)
// Ищет шаблон в текущей строке.

String findWithinHorizon(String pattern, int horizon)
// Ищет шаблон в пределах horizon символов.



Scanner useDelimiter(Pattern pattern)
// Устанавливает разделитель (по умолчанию — пробелы).



Scanner useLocale(Locale locale)
// Устанавливает локаль для числового формата.

Scanner useRadix(int radix)
// Устанавливает систему счисления для чисел.

void close()
// Закрывает Scanner.
```

## Пример
```java
import java.util.Scanner;

public class ScannerExample {
    public static void main(String[] args) {
        String data = "10 true hello";
        Scanner sc = new Scanner(data);

        int a = sc.nextInt();
        
        boolean c = sc.nextBoolean();
        String d = sc.next();

        System.out.println("int=" + a  + ", boolean=" + c + ", string=" + d);
        sc.close();
    }
}
```
