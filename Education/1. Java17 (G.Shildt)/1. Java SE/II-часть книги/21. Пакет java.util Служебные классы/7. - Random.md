`Random` — генератор псевдослучайных чисел

Класс ==для генерации псевдослучайных значений== (булевских/byte/int/long/float/double/гауссовских). 

Начиная с JDK 17, `Random` реализует интерфейс `RandomGenerator`. 
Также существуют поточные методы для интеграции с Stream API (JDK 8+).


> [!NOTE] ⭐ Применение ⭐
> - Используйте с предсказуемым `seed` для воспроизводимости; 
> - Для криптографических задач используйте `SecureRandom`.

## Конструкторы
```java
Random()
// Создаёт генератор с начальными параметрами (seed зависит от текущего времени).

Random(long seed)
// ⭐Создаёт генератор с фиксированным seed - начальное значение (для воспроизводимых последовательностей).
```

## Методы
```java
void setSeed(long seed)
// Устанавливает новое значение seed.


int nextInt()
// Возвращает случайное int в полном диапазоне.

int nextInt(int bound)
// Возвращает случайное int от 0 (включительно) до bound (исключительно).



long nextLong()
// Случайное long.

boolean nextBoolean()
// Случайное true/false.

float nextFloat()
// Случайное float от 0.0 до 1.0.

double nextDouble()
// Случайное double от 0.0 до 1.0.



double nextGaussian()
// Случайное double из нормального распределения (среднее 0.0, σ = 1.0).



void nextBytes(byte[] bytes)
// Заполняет массив случайными байтами.



IntStream ints()
// Возвращает бесконечный поток случайных int.

IntStream ints(long streamSize)
// Поток фиксированной длины случайных int.

IntStream ints(long streamSize, int randomNumberOrigin, int randomNumberBound)
// Поток случайных int в заданном диапазоне.

LongStream longs(...), DoubleStream doubles(...)
// Аналогично для long и double.
```

## Пример
```java
import java.util.Random;

public class RandomExample {
    public static void main(String[] args) {
        Random rand = new Random();

        System.out.println("Случайный int: " + rand.nextInt());
        System.out.println("От 0 до 9: " + rand.nextInt(10));
        System.out.println("Случайное double: " + rand.nextDouble());
        System.out.println("Случайный boolean: " + rand.nextBoolean());
    }
}
```
