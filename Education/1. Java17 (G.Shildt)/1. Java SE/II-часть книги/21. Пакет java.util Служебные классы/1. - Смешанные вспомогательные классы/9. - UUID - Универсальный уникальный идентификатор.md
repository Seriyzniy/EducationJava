Универсальный уникальный идентификатор (128 бит).
## Конструктор
Приватный, ==используется фабрика==
```java
public static UUID randomUUID()
public static UUID fromString(String name)
public static UUID nameUUIDFromBytes(byte[] name)
```

## Основные методы
```java
public static UUID randomUUID() 
// Случайный UUID

public static UUID fromString(String name) 
// Создать из строки (⭐строковое представление UUID⭐)

public static UUID nameUUIDFromBytes(byte[] name) 
// UUID из массива байт (MD5)

public long getMostSignificantBits() 
// Старшие 64 бита

public long getLeastSignificantBits() 
// Младшие 64 бита

public int version() 
// Версия UUID (1, 4, и т.д.)

public int variant() 
// Вариант (RFC 4122 и др.)

public String toString() 
// Строка в формате XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX

public int hashCode() 
public boolean equals(Object obj) 
// equals + hashCode

```

## Особенности
- Гарантированная уникальность (в пределах вероятности).
- ⭐⭐Используется для идентификаторов в БД, системах.

## Пример
```java
UUID id = UUID.randomUUID();
System.out.println(id.toString());
```
