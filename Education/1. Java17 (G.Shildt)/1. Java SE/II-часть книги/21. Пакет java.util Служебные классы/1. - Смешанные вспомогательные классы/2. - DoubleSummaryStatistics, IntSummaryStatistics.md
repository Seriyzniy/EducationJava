### DoubleSummaryStatistics
Собирает статистику по потоку `double` значений: количество, сумму, минимум, максимум, среднее.

### IntSummaryStatistics
Аналог `DoubleSummaryStatistics`, но для `int`.

## Конструктор
```java
public DoubleSummaryStatistics() // Создает пустую статистику
```

## Основные методы
```java
public void accept(double value) 
// Добавить значение

public void combine(DoubleSummaryStatistics other) 
// Объединить с другой статистикой

public final long getCount() 
// Кол-во элементов

public final double getSum() 
// Сумма

public final double getMin() 
// Минимум

public final double getMax() 
// Максимум

public final double getAverage() 
// Среднее

public String toString() 
// Представление в строке

```

## Особенности

- Используется в стримах для агрегирования данных.
    
- Реализует интерфейс `Consumer<Double>`.

## Пример
```java
DoubleSummaryStatistics stats = Stream.of(1.2, 3.4, 5.6)
        .mapToDouble(Double::doubleValue)
        .summaryStatistics();

System.out.println(stats.getAverage()); // 3.4
```
