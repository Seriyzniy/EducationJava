| Класс                       | Назначение                                           | Ключевые возможности                                                                                                                                             |
| --------------------------- | ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 💡**Base64**                | Поддержка кодировки Base64                           | • Кодирование/декодирование данных в формат Base64  <br>• Вложенные классы Encoder и Decoder  <br>• Поддержка различных вариантов Base64 (Basic, URL-safe, MIME) |
| **DoubleSummaryStatistics** | Сбор статистики для значений double                  | • Среднее значение  <br>• Минимальное значение  <br>• Максимальное значение  <br>• Количество элементов  <br>• Сумма значений                                    |

| 💡**EventObject**           | 💡Суперкласс для всех классов событий💡              | • Базовый класс для всех событий в Java  <br>• Содержит источник события  <br>• Используется в модели событий AWT и Swing                                        |
| **FormattableFlags**        | Флаги форматирования для интерфейса Formattable      | • LEFT_JUSTIFY - выравнивание по левому краю  <br>• UPPERCASE - преобразование в верхний регистр  <br>• ALTERNATE - альтернативный формат вывода                 |
| **HexFormat**               | Преобразования в/из шестнадцатеричного формата       | • Конвертация между hex-строками и байтами  <br>• Форматирование и парсинг hex-данных  <br>• Класс, основанный на значениях (value-based)                        |
| **IntSummaryStatistics**    | Сбор статистики для значений int                     | • Среднее значение  <br>• Минимальное значение  <br>• Максимальное значение  <br>• Количество элементов  <br>• Сумма значений                                    |
| **Objects**                 | Утилитарные методы для работы с объектами            | • Проверка на null  <br>• Сравнение объектов  <br>• Вычисление хеш-кода  <br>• Генерация строкового представления                                                |
| **PropertyPermission**      | Управление разрешениями свойств                      | • Контроль доступа к системным свойствам  <br>• Управление правами чтения/записи  <br>• Используется в security manager                                          |
| **ServiceLoader**           | Поиск поставщиков служб (service providers)          | • Обнаружение реализаций интерфейсов  <br>• Загрузка служб через META-INF/services  <br>• Поддержка модульной системы                                            |
| **StringJoiner**            | Конкатенация CharSequence объектов                   | • Объединение строк с разделителем  <br>• Поддержка префикса и суффикса  <br>• Используется в String.join()                                                      |
| 💡**UUID**                  | Работа с универсальными уникальными идентификаторами | • Генерация UUID (random, name-based)  <br>• Парсинг из строки  <br>• Сравнение и сортировка UUID                                                                |
