**String Pool** (пул строк) — это специальная область в памяти Java Heap (часть памяти Java Virtual Machine), где хранятся строковые литералы. Эта область используется для оптимизации работы с объектами типа `String`, так как строки в Java являются **неизменяемыми (immutable)**, и повторное создание одинаковых строковых объектов неэффективно.


Такие строки помещаются в String pool
```java
String str = "Hello";
```

А такие ==НЕ ПОМЕЩАЮТСЯ в String Pool==
```java
String str = new String("Hello world");
```
